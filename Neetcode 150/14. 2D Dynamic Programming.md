## 1️⃣ Unique Paths

### 📖 Description 

`m x n` grid mein robot top-left `(0,0)` se bottom-right `(m-1, n-1)` tak jaana chahta hai. Sirf **right** ya **down** move kar sakta hai. **Kitne unique paths** hain?

**Example:** `m = 3, n = 7` → Output: `28`

### 💡 Concept

```
paths(i, j) = paths(i-1, j) + paths(i, j-1)
Base: First row or column → 1 way
```

###  Recursion + Memoization Solution

```java
import java.util.Arrays;

class Solution {
    private int[][] memo;
    
    public int uniquePaths(int m, int n) {
        memo = new int[m][n];
        for (int i = 0; i < m; i++) {
            Arrays.fill(memo[i], -1);
        }
        return solve(m - 1, n - 1);
    }
    
    private int solve(int i, int j) {
        // Base case: first row ya first column
        if (i == 0 || j == 0) return 1;
        
        if (memo[i][j] != -1) return memo[i][j];
        
        // Right se aaya ya down se aaya
        memo[i][j] = solve(i - 1, j) + solve(i, j - 1);
        return memo[i][j];
    }
}
```

### ✅ Time: O(m × n) | Space: O(m × n)

## 2️⃣ Longest Common Subsequence (LCS)

### 📖 Description 

Do strings `text1` aur `text2` diye hain. **Longest common subsequence** ki length find karni hai. Subsequence means order maintain karna hai, contiguous hona zaroori nahi.

**Example:** `text1 = "abcde", text2 = "ace"` → Output: `3` ("ace")

### 💡 Concept

```
LCS(i, j) = 1 + LCS(i-1, j-1) if chars match
          = max(LCS(i-1, j), LCS(i, j-1)) if chars don't match
Base: i == 0 or j == 0 → 0
```

###  Recursion + Memoization Solution

```java
import java.util.Arrays;

class Solution {
    private String s1, s2;
    private int[][] memo;
    
    public int longestCommonSubsequence(String text1, String text2) {
        this.s1 = text1;
        this.s2 = text2;
        int n1 = s1.length(), n2 = s2.length();
        memo = new int[n1 + 1][n2 + 1];
        for (int i = 0; i <= n1; i++) {
            Arrays.fill(memo[i], -1);
        }
        return solve(n1, n2);
    }
    
    private int solve(int i, int j) {
        if (i == 0 || j == 0) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        if (s1.charAt(i - 1) == s2.charAt(j - 1)) {
            memo[i][j] = 1 + solve(i - 1, j - 1);
        } else {
            memo[i][j] = Math.max(solve(i - 1, j), solve(i, j - 1));
        }
        return memo[i][j];
    }
}
```

### ✅ Time: O(m × n) | Space: O(m × n)

## 3️⃣ Best Time to Buy and Sell Stock With Cooldown

### 📖 Description 

Stock prices ka array hai. Tum buy/sell kar sakte ho. **Cooldown**: Sell karne ke baad agle din buy nahi kar sakte. Maximum profit?

**Example:** `prices = [1,2,3,0,2]` → Output: `3` (Buy 1, Sell 2, Cooldown, Buy 0, Sell 2)

### 💡 Concept

```
Teen states:
- s0: No stock, not on cooldown (free to buy)
- s1: Holding stock
- s2: Just sold (on cooldown)

s0[i] = max(s0[i-1], s2[i-1])
s1[i] = max(s1[i-1], s0[i-1] - prices[i])
s2[i] = s1[i-1] + prices[i]
```

###  Recursion + Memoization Solution

```java
import java.util.Arrays;

class Solution {
    private int[] prices;
    private int[][] memo; // memo[i][state] where state: 0=s0, 1=s1, 2=s2
    private int n;
    
    public int maxProfit(int[] prices) {
        this.prices = prices;
        this.n = prices.length;
        memo = new int[n][3];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        return solve(0, 0); // Start with s0 state (no stock, not on cooldown)
    }
    
    private int solve(int day, int state) {
        if (day >= n) return 0;
        if (memo[day][state] != -1) return memo[day][state];
        
        int result = 0;
        
        if (state == 0) { // s0: No stock, not on cooldown
            // Option 1: Do nothing, stay in s0
            // Option 2: Buy stock, go to s1
            result = Math.max(solve(day + 1, 0), 
                              -prices[day] + solve(day + 1, 1));
        } 
        else if (state == 1) { // s1: Holding stock
            // Option 1: Hold stock, stay in s1
            // Option 2: Sell stock, go to s2 (cooldown)
            result = Math.max(solve(day + 1, 1), 
                              prices[day] + solve(day + 1, 2));
        } 
        else { // s2: On cooldown (just sold)
            // Must rest for one day, go to s0
            result = solve(day + 1, 0);
        }
        
        memo[day][state] = result;
        return result;
    }
}
```

### ✅ Time: O(n × 3) = O(n) | Space: O(n)

## 4️⃣ Coin Change II (Number of Combinations)

### 📖 Description 

`coins` denominations diye hain, infinite supply hai. `amount` banane ke **kitne unique combinations** hain? Order matter nahi karta.

**Example:** `amount = 5, coins = [1,2,5]` → Output: `4` (1+1+1+1+1, 1+1+1+2, 1+2+2, 5)

### 💡 Concept

```
combinations(amount, coinIndex) = combinations(amount, coinIndex+1) + 
                                  combinations(amount - coins[coinIndex], coinIndex)

Base: amount == 0 → 1 (ek tarika: koi coin nahi)
      amount < 0 → 0
```

###  Recursion + Memoization Solution

```java
import java.util.Arrays;

class Solution {
    private int[] coins;
    private int[][] memo;
    
    public int change(int amount, int[] coins) {
        this.coins = coins;
        memo = new int[coins.length][amount + 1];
        for (int i = 0; i < coins.length; i++) {
            Arrays.fill(memo[i], -1);
        }
        return solve(0, amount);
    }
    
    private int solve(int idx, int amount) {
        if (amount == 0) return 1;
        if (amount < 0 || idx >= coins.length) return 0;
        if (memo[idx][amount] != -1) return memo[idx][amount];
        
        // Option 1: Skip current coin
        // Option 2: Use current coin (stay at same index because infinite supply)
        int ways = solve(idx + 1, amount) + solve(idx, amount - coins[idx]);
        
        memo[idx][amount] = ways;
        return ways;
    }
}
```

### ✅ Time: O(n × amount) | Space: O(n × amount)

## 5️⃣ Target Sum

### 📖 Description 

Array `nums` hai. Har element ke aage `+` ya `-` laga sakte ho. Final sum `target` ke equal aaye. **Kitne ways** hain?

**Example:** `nums = [1,1,1,1,1], target = 3` → Output: `5`

### 💡 Concept

Is problem ko **Subset Sum** mein transform kar sakte hain:

```
P - N = target
P + N = total
∴ 2P = target + total
∴ P = (target + total) / 2

Ab find subsets with sum = P
```

###  Recursion + Memoization Solution

```java
import java.util.Arrays;

class Solution {
    private int[] nums;
    private int[][] memo;
    
    public int findTargetSumWays(int[] nums, int target) {
        int total = 0;
        for (int num : nums) total += num;
        
        // Check feasibility
        if (total < Math.abs(target) || (total + target) % 2 != 0) return 0;
        
        int subsetSum = (total + target) / 2;
        this.nums = nums;
        memo = new int[nums.length][subsetSum + 1];
        for (int i = 0; i < nums.length; i++) {
            Arrays.fill(memo[i], -1);
        }
        return countSubsets(0, subsetSum);
    }
    
    private int countSubsets(int idx, int sum) {
        if (sum == 0) return 1;
        if (idx >= nums.length || sum < 0) return 0;
        if (memo[idx][sum] != -1) return memo[idx][sum];
        
        // Include or exclude current element
        int ways = countSubsets(idx + 1, sum - nums[idx]) + 
                   countSubsets(idx + 1, sum);
        
        memo[idx][sum] = ways;
        return ways;
    }
}
```

### ✅ Time: O(n × sum) | Space: O(n × sum)

## 6️⃣ Interleaving String

### 📖 Description 

Check karna hai ki `s3` `s1` aur `s2` ka interleaving hai ya nahi. Order maintain hona chahiye dono strings ka.

**Example:** `s1 = "aabcc", s2 = "dbbca", s3 = "aadbbcbcac"` → Output: `true`

### 💡 Concept

```
isInterleave(i, j) = (s1[i] == s3[i+j] && isInterleave(i+1, j)) OR
                     (s2[j] == s3[i+j] && isInterleave(i, j+1))

Base: i == s1.length && j == s2.length → true
```

###  Recursion + Memoization Solution

```java
import java.util.Arrays;

class Solution {
    private String s1, s2, s3;
    private int[][] memo; // -1: unknown, 0: false, 1: true
    private int n1, n2;
    
    public boolean isInterleave(String s1, String s2, String s3) {
        this.s1 = s1;
        this.s2 = s2;
        this.s3 = s3;
        this.n1 = s1.length();
        this.n2 = s2.length();
        
        if (n1 + n2 != s3.length()) return false;
        
        memo = new int[n1 + 1][n2 + 1];
        for (int i = 0; i <= n1; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        return solve(0, 0);
    }
    
    private boolean solve(int i, int j) {
        if (i == n1 && j == n2) return true;
        if (memo[i][j] != -1) return memo[i][j] == 1;
        
        boolean result = false;
        int k = i + j;
        
        // Try taking from s1
        if (i < n1 && s1.charAt(i) == s3.charAt(k)) {
            result = result || solve(i + 1, j);
        }
        
        // Try taking from s2
        if (j < n2 && s2.charAt(j) == s3.charAt(k)) {
            result = result || solve(i, j + 1);
        }
        
        memo[i][j] = result ? 1 : 0;
        return result;
    }
}
```

### ✅ Time: O(m × n) | Space: O(m × n)

## 7️⃣ Longest Increasing Path in a Matrix

### 📖 Description 

Matrix mein longest **strictly increasing** path find karni hai. Move up, down, left, right kar sakte ho.

**Example:** `matrix = [[9,9,4],[6,6,8],[2,1,1]]` → Output: `4` (1→2→6→9)

### 💡 Concept

```
LIP(i, j) = 1 + max(LIP(neighbors with greater value))

Memoization: Har cell se start karke DFS karo, result store karo.
```

###  Recursion + Memoization Solution

```java
import java.util.Arrays;

class Solution {
    private int[][] matrix;
    private int[][] memo;
    private int rows, cols;
    private int[][] dirs = {{0, 1}, {0, -1}, {1, 0}, {-1, 0}};
    
    public int longestIncreasingPath(int[][] matrix) {
        this.matrix = matrix;
        this.rows = matrix.length;
        this.cols = matrix[0].length;
        memo = new int[rows][cols];
        
        int maxPath = 0;
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                maxPath = Math.max(maxPath, dfs(i, j));
            }
        }
        return maxPath;
    }
    
    private int dfs(int r, int c) {
        if (memo[r][c] != 0) return memo[r][c];
        
        int maxLen = 1; // Current cell itself
        
        for (int[] dir : dirs) {
            int nr = r + dir[0];
            int nc = c + dir[1];
            
            if (nr >= 0 && nr < rows && nc >= 0 && nc < cols && 
                matrix[nr][nc] > matrix[r][c]) {
                maxLen = Math.max(maxLen, 1 + dfs(nr, nc));
            }
        }
        
        memo[r][c] = maxLen;
        return maxLen;
    }
}
```

### ✅ Time: O(m × n) | Space: O(m × n)

## 8️⃣ Distinct Subsequences

### 📖 Description 

String `s` mein kitni baar string `t` distinct subsequence ke roop mein aati hai? (Order maintain karna hai)

**Example:** `s = "rabbbit", t = "rabbit"` → Output: `3` (Delete different 'b's)

### 💡 Concept

```
numDistinct(i, j) = numDistinct(i-1, j-1) + numDistinct(i-1, j) if s[i] == t[j]
                  = numDistinct(i-1, j) if s[i] != t[j]

Base: j == 0 → 1 (empty string ek tarika)
      i == 0 and j > 0 → 0
```

###  Recursion + Memoization Solution

```java
import java.util.Arrays;

class Solution {
    private String s, t;
    private int[][] memo;
    private int n1, n2;
    
    public int numDistinct(String s, String t) {
        this.s = s;
        this.t = t;
        this.n1 = s.length();
        this.n2 = t.length();
        memo = new int[n1 + 1][n2 + 1];
        for (int i = 0; i <= n1; i++) {
            Arrays.fill(memo[i], -1);
        }
        return solve(n1, n2);
    }
    
    private int solve(int i, int j) {
        if (j == 0) return 1;  // t is empty → 1 way
        if (i == 0) return 0;  // s is empty but t is not → 0 ways
        if (memo[i][j] != -1) return memo[i][j];
        
        int result;
        if (s.charAt(i - 1) == t.charAt(j - 1)) {
            // Match: use this char OR skip this char
            result = solve(i - 1, j - 1) + solve(i - 1, j);
        } else {
            // Don't match: must skip
            result = solve(i - 1, j);
        }
        
        memo[i][j] = result;
        return result;
    }
}
```

### ✅ Time: O(m × n) | Space: O(m × n)

## 9️⃣ Edit Distance

### 📖 Description 

`word1` ko `word2` mein convert karne ke **minimum operations** (Insert, Delete, Replace).

**Example:** `word1 = "horse", word2 = "ros"` → Output: `3`

### 💡 Concept

```
minDistance(i, j) = minDistance(i-1, j-1) if chars match
                  = 1 + min( delete, insert, replace ) if chars don't match

delete: minDistance(i-1, j)
insert: minDistance(i, j-1)
replace: minDistance(i-1, j-1)
```

###  Recursion + Memoization Solution

```java
import java.util.Arrays;

class Solution {
    private String word1, word2;
    private int[][] memo;
    private int n1, n2;
    
    public int minDistance(String word1, String word2) {
        this.word1 = word1;
        this.word2 = word2;
        this.n1 = word1.length();
        this.n2 = word2.length();
        memo = new int[n1 + 1][n2 + 1];
        for (int i = 0; i <= n1; i++) {
            Arrays.fill(memo[i], -1);
        }
        return solve(n1, n2);
    }
    
    private int solve(int i, int j) {
        if (i == 0) return j; // Insert all characters of word2
        if (j == 0) return i; // Delete all characters of word1
        if (memo[i][j] != -1) return memo[i][j];
        
        int result;
        if (word1.charAt(i - 1) == word2.charAt(j - 1)) {
            result = solve(i - 1, j - 1);
        } else {
            int deleteOp = solve(i - 1, j);
            int insertOp = solve(i, j - 1);
            int replaceOp = solve(i - 1, j - 1);
            result = 1 + Math.min(deleteOp, Math.min(insertOp, replaceOp));
        }
        
        memo[i][j] = result;
        return result;
    }
}
```

### ✅ Time: O(m × n) | Space: O(m × n)

## 🔟 Burst Balloons

### 📖 Description 

Balloons burst karo. Jab balloon `i` burst karte ho toh `nums[left] × nums[i] × nums[right]` coins milte hain. Maximum coins collect karo.

**Example:** `nums = [3,1,5,8]` → Output: `167`

### 💡 Concept

```
Key insight: Socho ki kaun sa balloon LAST mein burst hoga
dp[i][j] = max coins from bursting balloons i to j

dp[i][j] = max(dp[i][k-1] + dp[k+1][j] + nums[i-1] * nums[k] * nums[j+1])
```

###  Recursion + Memoization Solution

```java
import java.util.Arrays;

class Solution {
    private int[] balloons;
    private int[][] memo;
    private int n;
    
    public int maxCoins(int[] nums) {
        // Add virtual balloons with value 1 at boundaries
        n = nums.length;
        balloons = new int[n + 2];
        balloons[0] = 1;
        balloons[n + 1] = 1;
        for (int i = 0; i < n; i++) {
            balloons[i + 1] = nums[i];
        }
        
        memo = new int[n + 2][n + 2];
        for (int i = 0; i <= n + 1; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        return solve(1, n);
    }
    
    private int solve(int left, int right) {
        if (left > right) return 0;
        if (memo[left][right] != -1) return memo[left][right];
        
        int maxCoins = 0;
        // Try each balloon as the LAST to burst in range [left, right]
        for (int k = left; k <= right; k++) {
            int coins = solve(left, k - 1) + solve(k + 1, right) + 
                        balloons[left - 1] * balloons[k] * balloons[right + 1];
            maxCoins = Math.max(maxCoins, coins);
        }
        
        memo[left][right] = maxCoins;
        return maxCoins;
    }
}
```

### ✅ Time: O(n³) | Space: O(n²)

## 1️⃣1️⃣ Regular Expression Matching

### 📖 Description 

Pattern matching with:
- `.` → matches any single character
- `*` → matches zero or more of preceding element

**Example:** `s = "aab", p = "c*a*b"` → Output: `true`

### 💡 Concept

```
isMatch(i, j) = isMatch(i-1, j-1) if p[j] == s[i] or p[j] == '.'
              = isMatch(i, j-2) OR (match && isMatch(i-1, j)) if p[j] == '*'
```

###  Recursion + Memoization Solution

```java
import java.util.Arrays;

class Solution {
    private String s, p;
    private int[][] memo; // -1: unknown, 0: false, 1: true
    private int n1, n2;
    
    public boolean isMatch(String s, String p) {
        this.s = s;
        this.p = p;
        this.n1 = s.length();
        this.n2 = p.length();
        memo = new int[n1 + 1][n2 + 1];
        for (int i = 0; i <= n1; i++) {
            Arrays.fill(memo[i], -1);
        }
        return solve(0, 0);
    }
    
    private boolean solve(int i, int j) {
        if (j == n2) return i == n1; // Pattern exhausted
        if (memo[i][j] != -1) return memo[i][j] == 1;
        
        boolean firstMatch = (i < n1 && (s.charAt(i) == p.charAt(j) || p.charAt(j) == '.'));
        boolean result;
        
        if (j + 1 < n2 && p.charAt(j + 1) == '*') {
            // '*' case: zero occurrences OR one/more occurrences
            result = solve(i, j + 2) || (firstMatch && solve(i + 1, j));
        } else {
            // Normal case: single character match
            result = firstMatch && solve(i + 1, j + 1);
        }
        
        memo[i][j] = result ? 1 : 0;
        return result;
    }
}
```

### ✅ Time: O(m × n) | Space: O(m × n)

## 1️⃣2️⃣ Longest Palindromic Subsequence (Bonus)

### 📖 Description 

String ka longest palindromic subsequence find karo (contiguous hona zaroori nahi).

**Example:** `s = "bbbab"` → Output: `4` ("bbbb")

### 💡 Concept

```
LPS(i, j) = 2 + LPS(i+1, j-1) if s[i] == s[j]
          = max(LPS(i+1, j), LPS(i, j-1)) if s[i] != s[j]
```

###  Recursion + Memoization Solution

```java
import java.util.Arrays;

class Solution {
    private String s;
    private int[][] memo;
    
    public int longestPalindromeSubseq(String s) {
        this.s = s;
        int n = s.length();
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        return solve(0, n - 1);
    }
    
    private int solve(int i, int j) {
        if (i > j) return 0;
        if (i == j) return 1; // Single character
        if (memo[i][j] != -1) return memo[i][j];
        
        int result;
        if (s.charAt(i) == s.charAt(j)) {
            result = 2 + solve(i + 1, j - 1);
        } else {
            result = Math.max(solve(i + 1, j), solve(i, j - 1));
        }
        
        memo[i][j] = result;
        return result;
    }
}
```

### ✅ Time: O(n²) | Space: O(n²)

##  Complete Summary Table

| Problem | Recursive Relation | Base Cases | Time |
|---------|-------------------|------------|------|
| Unique Paths | `f(i,j) = f(i-1,j) + f(i,j-1)` | i=0 or j=0 → 1 | O(m×n) |
| LCS | match: `1+f(i-1,j-1)`, else: `max(f(i-1,j), f(i,j-1))` | i=0 or j=0 → 0 | O(m×n) |
| Stock Cooldown | 3 states with transitions | day≥n → 0 | O(n) |
| Coin Change II | `skip + use` | amount=0 → 1 | O(n×amount) |
| Target Sum | `include + exclude` | sum=0 → 1 | O(n×sum) |
| Interleaving | `s1 match OR s2 match` | both empty → true | O(m×n) |
| LIP Matrix | `1 + max(neighbor)` | base cell → 1 | O(m×n) |
| Distinct Subseq | match: `use+skip`, else: `skip` | j=0 → 1 | O(m×n) |
| Edit Distance | `1 + min(delete,insert,replace)` | i=0 → j, j=0 → i | O(m×n) |
| Burst Balloons | `left + right + coins` | left>right → 0 | O(n³) |
| Regex Matching | `*` case and normal case | pattern exhausted → check | O(m×n) |
| LPS | match: `2 + f(i+1,j-1)`, else: `max` | i=j → 1 | O(n²) |

##  Key Takeaways for Interviews

1. **State Definition is Key**: Socho ki tumhe kya track karna hai (index, sum, etc.)

2. **Draw Recursion Tree**: Paper par example bana ke dekho

3. **Identify Patterns**:
   - 2 Strings → Usually 2D DP (LCS, Edit Distance)
   - 1 String with range → Interval DP (LPS, Burst Balloons)
   - Grid → 2D DP (Unique Paths)
   - Subsets → 1D DP with iteration order matter (Coin Change II)

4. **Memoization vs Tabulation**:
   - Memoization: Easy to code from recursion
   - Tabulation: Better for space optimization

5. **Common Optimizations**:
   - Space: O(n²) → O(n) where possible
   - Rolling arrays for 2D to 1D

# Bitmask DP Pattern

## 1. **Traveling Salesman Problem (TSP) - Shortest Path Visiting All Nodes**

**Instinct:**  
n cities, distances given. Shortest cycle that visits each city exactly once. State: mask of visited cities, last visited city. Transition: try all unvisited cities.

**Approach:**  
1. Precompute distance matrix (Floyd-Warshall if needed)
2. DP[mask][last] = min distance to visit cities in mask ending at last
3. Start from each city as starting point
4. Answer = min(DP[all][last] + dist[last][start])

```java
class Solution {
    int n;
    int[][] dist;
    int[][] memo;
    int INF = Integer.MAX_VALUE / 2;
    
    public int tsp(int[][] graph) {
        n = graph.length;
        dist = graph;
        memo = new int[1 << n][n];
        for (int i = 0; i < (1 << n); i++) {
            Arrays.fill(memo[i], -1);
        }
        
        int minCost = INF;
        for (int start = 0; start < n; start++) {
            minCost = Math.min(minCost, dfs(1 << start, start) + dist[start][0]);
        }
        return minCost;
    }
    
    private int dfs(int mask, int last) {
        if (mask == (1 << n) - 1) return 0;
        if (memo[mask][last] != -1) return memo[mask][last];
        
        int minCost = INF;
        for (int next = 0; next < n; next++) {
            if ((mask & (1 << next)) == 0) {
                int cost = dist[last][next] + dfs(mask | (1 << next), next);
                minCost = Math.min(minCost, cost);
            }
        }
        
        memo[mask][last] = minCost;
        return minCost;
    }
}
```

**Key Insight:** Bitmask tracks visited cities. State = (mask, last). O(2ⁿ × n²) time.

## 2. **Hamiltonian Path (Leetcode 980 - Unique Paths III variant)**

**Instinct:**  
Grid me path jo har empty cell exactly ek baar visit kare. State: mask of visited cells, current position.

**Approach:**  
1. Number all empty cells (0 to k-1)
2. DP[mask][pos] = number of Hamiltonian paths
3. Try all unvisited adjacent cells

```java
class Solution {
    int m, n;
    int[][] grid;
    int emptyCount = 0;
    int startX, startY;
    int[][] memo;
    int[][] dirs = {{0,1},{1,0},{0,-1},{-1,0}};
    
    public int uniquePathsIII(int[][] grid) {
        this.grid = grid;
        m = grid.length;
        n = grid[0].length;
        
        int cellId = 0;
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (grid[i][j] != -1) {
                    if (grid[i][j] == 1) {
                        startX = i;
                        startY = j;
                    }
                    grid[i][j] = cellId++;
                }
            }
        }
        emptyCount = cellId;
        
        memo = new int[1 << emptyCount][emptyCount];
        for (int i = 0; i < (1 << emptyCount); i++) {
            Arrays.fill(memo[i], -1);
        }
        
        int startId = grid[startX][startY];
        return dfs(1 << startId, startId);
    }
    
    private int dfs(int mask, int pos) {
        if (mask == (1 << emptyCount) - 1) {
            // Check if last cell is the ending cell (grid value 2)
            // For simplicity, assume last cell is valid
            return 1;
        }
        if (memo[mask][pos] != -1) return memo[mask][pos];
        
        int total = 0;
        // Find position from pos id
        int x = -1, y = -1;
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (grid[i][j] == pos) {
                    x = i; y = j;
                    break;
                }
            }
        }
        
        for (int[] dir : dirs) {
            int nx = x + dir[0];
            int ny = y + dir[1];
            if (nx >= 0 && nx < m && ny >= 0 && ny < n && grid[nx][ny] != -1) {
                int nextId = grid[nx][ny];
                if ((mask & (1 << nextId)) == 0) {
                    total += dfs(mask | (1 << nextId), nextId);
                }
            }
        }
        
        memo[mask][pos] = total;
        return total;
    }
}
```

**Key Insight:** Hamiltonian path = visiting all cells exactly once. Bitmask over cell IDs.

## 3. **Minimum Cost to Assign Jobs (Assignment Problem)**

**Instinct:**  
n workers, n jobs. cost[i][j] = cost to assign job j to worker i. Min total cost. State: mask of assigned jobs.

**Approach:**  
1. DP[mask] = min cost to assign jobs in mask
2. Next worker = number of bits set in mask
3. Try all unassigned jobs for this worker

```java
class Solution {
    int n;
    int[][] cost;
    int[] memo;
    int INF = Integer.MAX_VALUE / 2;
    
    public int assignJobs(int[][] cost) {
        this.cost = cost;
        n = cost.length;
        memo = new int[1 << n];
        Arrays.fill(memo, -1);
        return dfs(0);
    }
    
    private int dfs(int mask) {
        int worker = Integer.bitCount(mask);
        if (worker == n) return 0;
        if (memo[mask] != -1) return memo[mask];
        
        int minCost = INF;
        for (int job = 0; job < n; job++) {
            if ((mask & (1 << job)) == 0) {
                minCost = Math.min(minCost, cost[worker][job] + dfs(mask | (1 << job)));
            }
        }
        
        memo[mask] = minCost;
        return minCost;
    }
}
```

**Key Insight:** Worker = number of assigned jobs. Bitmask tracks assigned jobs.

## 4. **Partition to K Equal Sum Subsets (Leetcode 698)**

**Instinct:**  
Array ko k subsets me divide karna hai with equal sum. State: mask of used elements, current subset sum.

**Approach:**  
1. target = total / k
2. DP[mask][currSum] or DP[mask] with sum tracking
3. If currSum == target, start new subset

```java
class Solution {
    int[] nums;
    int target;
    Boolean[] memo;
    
    public boolean canPartitionKSubsets(int[] nums, int k) {
        this.nums = nums;
        int total = 0;
        for (int num : nums) total += num;
        if (total % k != 0) return false;
        
        target = total / k;
        Arrays.sort(nums);
        if (nums[nums.length - 1] > target) return false;
        
        memo = new Boolean[1 << nums.length];
        return dfs(0, 0, k);
    }
    
    private boolean dfs(int mask, int currSum, int remaining) {
        if (remaining == 0) return true;
        if (currSum == target) return dfs(mask, 0, remaining - 1);
        if (memo[mask] != null) return memo[mask];
        
        for (int i = 0; i < nums.length; i++) {
            if ((mask & (1 << i)) != 0) continue;
            if (currSum + nums[i] > target) continue;
            if (dfs(mask | (1 << i), currSum + nums[i], remaining)) {
                memo[mask] = true;
                return true;
            }
        }
        
        memo[mask] = false;
        return false;
    }
}
```

**Key Insight:** Bitmask tracks used elements. When subset sum reaches target, start new subset.

## 5. **Shortest Path Visiting All Nodes (Leetcode 847)**

**Instinct:**  
Graph me shortest path jo saare nodes visit kare (can revisit nodes). BFS on (mask, node).

**Approach:**  
1. BFS from each node as starting point
2. State = (mask, node)
3. When mask = all nodes, return steps

```java
class Solution {
    public int shortestPathLength(int[][] graph) {
        int n = graph.length;
        int[][] dist = new int[1 << n][n];
        for (int i = 0; i < (1 << n); i++) {
            Arrays.fill(dist[i], -1);
        }
        
        Queue<int[]> q = new LinkedList<>();
        for (int i = 0; i < n; i++) {
            dist[1 << i][i] = 0;
            q.offer(new int[]{1 << i, i});
        }
        
        while (!q.isEmpty()) {
            int[] curr = q.poll();
            int mask = curr[0], node = curr[1];
            
            if (mask == (1 << n) - 1) return dist[mask][node];
            
            for (int next : graph[node]) {
                int newMask = mask | (1 << next);
                if (dist[newMask][next] == -1) {
                    dist[newMask][next] = dist[mask][node] + 1;
                    q.offer(new int[]{newMask, next});
                }
            }
        }
        return -1;
    }
}
```

**Key Insight:** BFS on (mask, node) for unweighted graph. Can revisit nodes.

## 6. **Maximum Profit from Jobs with Deadlines (Job Sequencing)**

**Instinct:**  
n jobs with profit and deadlines. Max profit. State: mask of completed jobs.

**Approach:**  
1. Sort jobs by profit descending
2. DP[mask] = max profit
3. Try assigning job at earliest available slot

```java
class Solution {
    int n;
    int[] profit;
    int[] deadline;
    int[] memo;
    
    public int jobSequencing(int[] profit, int[] deadline) {
        this.profit = profit;
        this.deadline = deadline;
        n = profit.length;
        memo = new int[1 << n];
        Arrays.fill(memo, -1);
        return dfs(0);
    }
    
    private int dfs(int mask) {
        int time = Integer.bitCount(mask);
        if (time == n) return 0;
        if (memo[mask] != -1) return memo[mask];
        
        int maxProfit = 0;
        for (int job = 0; job < n; job++) {
            if ((mask & (1 << job)) == 0 && deadline[job] > time) {
                maxProfit = Math.max(maxProfit, profit[job] + dfs(mask | (1 << job)));
            }
        }
        
        memo[mask] = maxProfit;
        return maxProfit;
    }
}
```

**Key Insight:** Time = number of assigned jobs. Job can be assigned if deadline > time.

## 7. **Minimum Number of Operations to Move All Balls (Leetcode 1769 variant)**

**Instinct:**  
Binary string me balls hain. Minimum operations to move all balls to each position. Bitmask not needed, but pattern similar.

**Approach:**  
1. Prefix sums for left and right
2. Or bitmask DP for small n

```java
class Solution {
    public int[] minOperations(String boxes) {
        int n = boxes.length();
        int[] ans = new int[n];
        
        int balls = 0, ops = 0;
        for (int i = 0; i < n; i++) {
            ans[i] += ops;
            balls += boxes.charAt(i) - '0';
            ops += balls;
        }
        
        balls = 0;
        ops = 0;
        for (int i = n - 1; i >= 0; i--) {
            ans[i] += ops;
            balls += boxes.charAt(i) - '0';
            ops += balls;
        }
        
        return ans;
    }
}
```

**Key Insight:** Prefix sum approach O(n). Bitmask not needed for this problem.

## 8. **Maximum Students Taking Exam (Leetcode 1349)**

**Instinct:**  
Grid me students seat karne hain such that no two adjacent (4 directions). Maximum students. DP[row][mask].

**Approach:**  
1. For each row, generate all valid masks (no adjacent seats)
2. Check conflict with previous row
3. DP[row][mask] = max students

```java
class Solution {
    int m, n;
    char[][] seats;
    List<Integer> validMasks;
    int[][] memo;
    
    public int maxStudents(char[][] seats) {
        this.seats = seats;
        m = seats.length;
        n = seats[0].length;
        
        validMasks = new ArrayList<>();
        for (int mask = 0; mask < (1 << n); mask++) {
            if ((mask & (mask << 1)) == 0) {
                validMasks.add(mask);
            }
        }
        
        memo = new int[m][1 << n];
        for (int i = 0; i < m; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        return dfs(0, 0);
    }
    
    private int dfs(int row, int prevMask) {
        if (row == m) return 0;
        if (memo[row][prevMask] != -1) return memo[row][prevMask];
        
        int max = 0;
        for (int mask : validMasks) {
            // Check if mask is valid for current row (no broken seats)
            boolean valid = true;
            for (int j = 0; j < n; j++) {
                if ((mask & (1 << j)) != 0 && seats[row][j] == '#') {
                    valid = false;
                    break;
                }
            }
            if (!valid) continue;
            
            // Check conflict with previous row
            if ((mask & (prevMask << 1)) != 0 || (mask & (prevMask >> 1)) != 0) {
                continue;
            }
            
            int count = Integer.bitCount(mask);
            max = Math.max(max, count + dfs(row + 1, mask));
        }
        
        memo[row][prevMask] = max;
        return max;
    }
}
```

**Key Insight:** Row by row DP. Valid masks = no adjacent bits. Check diagonal conflicts.

## 9. **Number of Ways to Assign Tasks (Task Assignment)**

**Instinct:**  
n tasks, m workers (m >= n). Each worker can do at most one task. Count assignments. DP[mask] = ways.

**Approach:**  
1. DP[mask] = ways to assign tasks in mask
2. Next task = number of bits set in mask
3. Try all workers for this task

```java
class Solution {
    int n, m;
    boolean[][] canDo;
    int[] memo;
    int MOD = 1_000_000_007;
    
    public int assignTasks(int n, int m, boolean[][] canDo) {
        this.n = n;
        this.m = m;
        this.canDo = canDo;
        memo = new int[1 << n];
        Arrays.fill(memo, -1);
        return dfs(0);
    }
    
    private int dfs(int mask) {
        int task = Integer.bitCount(mask);
        if (task == n) return 1;
        if (memo[mask] != -1) return memo[mask];
        
        long ways = 0;
        for (int worker = 0; worker < m; worker++) {
            if (canDo[worker][task]) {
                ways += dfs(mask | (1 << worker));
            }
        }
        
        memo[mask] = (int)(ways % MOD);
        return memo[mask];
    }
}
```

**Key Insight:** Task index = number of assigned workers. Bitmask tracks assigned workers.

## 10. **Minimum Cost to Connect All Nodes (Bitmask MST)**

**Instinct:**  
Complete graph me min cost to connect all nodes. DP[mask] = min cost to connect nodes in mask.

**Approach:**  
1. DP[mask] = min cost to connect all nodes in mask
2. Try adding a new node connected to any node in mask

```java
class Solution {
    int n;
    int[][] dist;
    int[] memo;
    int INF = Integer.MAX_VALUE / 2;
    
    public int minCostToConnect(int n, int[][] edges) {
        this.n = n;
        dist = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(dist[i], INF);
            dist[i][i] = 0;
        }
        for (int[] e : edges) {
            dist[e[0]][e[1]] = Math.min(dist[e[0]][e[1]], e[2]);
            dist[e[1]][e[0]] = Math.min(dist[e[1]][e[0]], e[2]);
        }
        
        // Floyd-Warshall
        for (int k = 0; k < n; k++) {
            for (int i = 0; i < n; i++) {
                for (int j = 0; j < n; j++) {
                    dist[i][j] = Math.min(dist[i][j], dist[i][k] + dist[k][j]);
                }
            }
        }
        
        memo = new int[1 << n];
        Arrays.fill(memo, -1);
        memo[0] = 0;
        
        return dfs(1 << 0);
    }
    
    private int dfs(int mask) {
        if (mask == (1 << n) - 1) return 0;
        if (memo[mask] != -1) return memo[mask];
        
        int minCost = INF;
        for (int i = 0; i < n; i++) {
            if ((mask & (1 << i)) != 0) {
                for (int j = 0; j < n; j++) {
                    if ((mask & (1 << j)) == 0) {
                        minCost = Math.min(minCost, dist[i][j] + dfs(mask | (1 << j)));
                    }
                }
            }
        }
        
        memo[mask] = minCost;
        return minCost;
    }
}
```

**Key Insight:** Add one node at a time to connected component. Use precomputed shortest paths.

## 11. **Maximum XOR Subset (Leetcode 1863)**

**Instinct:**  
Array me subset ka maximum XOR. Bitmask over elements.

**Approach:**  
1. Try all subsets using bitmask
2. Compute XOR and track max

```java
class Solution {
    public int subsetXORMax(int[] nums) {
        int n = nums.length;
        int maxXor = 0;
        
        for (int mask = 0; mask < (1 << n); mask++) {
            int xor = 0;
            for (int i = 0; i < n; i++) {
                if ((mask & (1 << i)) != 0) {
                    xor ^= nums[i];
                }
            }
            maxXor = Math.max(maxXor, xor);
        }
        return maxXor;
    }
}
```

**Key Insight:** Simple brute force with bitmask for small n (n ≤ 20).

## 12. **Maximum Number of Achievable Transfer Requests (Leetcode 1601)**

**Instinct:**  
Building me transfer requests. Maximum requests such that net balance = 0. Bitmask over requests.

**Approach:**  
1. Try all subsets of requests using bitmask
2. Check if balance zero
3. Track max size

```java
class Solution {
    public int maximumRequests(int n, int[][] requests) {
        int m = requests.length;
        int max = 0;
        
        for (int mask = 0; mask < (1 << m); mask++) {
            int[] balance = new int[n];
            int count = 0;
            
            for (int i = 0; i < m; i++) {
                if ((mask & (1 << i)) != 0) {
                    balance[requests[i][0]]--;
                    balance[requests[i][1]]++;
                    count++;
                }
            }
            
            boolean valid = true;
            for (int b : balance) {
                if (b != 0) {
                    valid = false;
                    break;
                }
            }
            
            if (valid) {
                max = Math.max(max, count);
            }
        }
        return max;
    }
}
```

**Key Insight:** Brute force over 2ᵐ subsets. Valid if all balances zero.

## 13. **Smallest Sufficient Team (Leetcode 1125)**

**Instinct:**  
People with skills. Choose smallest team covering all required skills. Bitmask DP.

**Approach:**  
1. Encode skills as bits
2. DP[mask] = smallest team covering skills in mask
3. Try adding each person

```java
class Solution {
    public int[] smallestSufficientTeam(String[] req_skills, List<List<String>> people) {
        int n = req_skills.length;
        Map<String, Integer> skillIndex = new HashMap<>();
        for (int i = 0; i < n; i++) {
            skillIndex.put(req_skills[i], i);
        }
        
        int[] personMask = new int[people.size()];
        for (int i = 0; i < people.size(); i++) {
            int mask = 0;
            for (String skill : people.get(i)) {
                mask |= 1 << skillIndex.get(skill);
            }
            personMask[i] = mask;
        }
        
        long[] dp = new long[1 << n];
        Arrays.fill(dp, (1L << people.size()) - 1);
        dp[0] = 0;
        
        for (int mask = 0; mask < (1 << n); mask++) {
            for (int i = 0; i < people.size(); i++) {
                int newMask = mask | personMask[i];
                long newTeam = dp[mask] | (1L << i);
                if (Long.bitCount(newTeam) < Long.bitCount(dp[newMask])) {
                    dp[newMask] = newTeam;
                }
            }
        }
        
        long team = dp[(1 << n) - 1];
        int[] result = new int[Long.bitCount(team)];
        int idx = 0;
        for (int i = 0; i < people.size(); i++) {
            if ((team & (1L << i)) != 0) {
                result[idx++] = i;
            }
        }
        return result;
    }
}
```

**Key Insight:** Skills as bits. DP[mask] stores team as bitmask of people.

## 14. **Maximum Profit from Building (Leetcode 2025)**

**Instinct:**  
n buildings, m workers. Each worker can work on at most one building. Max profit. DP[mask].

**Approach:**  
1. DP[mask] = max profit using workers in mask
2. Try assigning building to subset of workers

```java
class Solution {
    int n, m;
    int[][] profit;
    int[] memo;
    
    public int maxProfit(int n, int m, int[][] profit) {
        this.n = n;
        this.m = m;
        this.profit = profit;
        memo = new int[1 << m];
        Arrays.fill(memo, -1);
        return dfs(0, 0);
    }
    
    private int dfs(int building, int workerMask) {
        if (building == n) return 0;
        if (memo[workerMask] != -1) return memo[workerMask];
        
        int max = 0;
        // Try all subsets of available workers for this building
        int available = ((1 << m) - 1) & (~workerMask);
        for (int subset = available; subset > 0; subset = (subset - 1) & available) {
            int workersCount = Integer.bitCount(subset);
            if (workersCount <= profit[building].length) {
                max = Math.max(max, profit[building][workersCount - 1] + 
                              dfs(building + 1, workerMask | subset));
            }
        }
        // Also try using no workers for this building
        max = Math.max(max, dfs(building + 1, workerMask));
        
        memo[workerMask] = max;
        return max;
    }
}
```

**Key Insight:** Subset enumeration over available workers. `for (subset = mask; subset > 0; subset = (subset-1) & mask)`

## 15. **Number of Ways to Wear Different Hats (Leetcode 1434)**

**Instinct:**  
n people, 40 hats. Each person likes some hats. Ways to assign distinct hats. DP[mask] = ways to assign hats to people in mask.

**Approach:**  
1. For each hat, which people like it
2. DP over hats, track assigned people mask

```java
class Solution {
    int MOD = 1_000_000_007;
    int n;
    List<Integer>[] hatToPeople;
    int[][] memo;
    
    public int numberWays(List<List<Integer>> hats) {
        n = hats.size();
        hatToPeople = new List[41];
        for (int i = 1; i <= 40; i++) hatToPeople[i] = new ArrayList<>();
        
        for (int i = 0; i < n; i++) {
            for (int hat : hats.get(i)) {
                hatToPeople[hat].add(i);
            }
        }
        
        memo = new int[41][1 << n];
        for (int i = 0; i <= 40; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        return dfs(1, 0);
    }
    
    private int dfs(int hat, int mask) {
        if (mask == (1 << n) - 1) return 1;
        if (hat > 40) return 0;
        if (memo[hat][mask] != -1) return memo[hat][mask];
        
        long ways = dfs(hat + 1, mask); // skip this hat
        
        for (int person : hatToPeople[hat]) {
            if ((mask & (1 << person)) == 0) {
                ways += dfs(hat + 1, mask | (1 << person));
            }
        }
        
        memo[hat][mask] = (int)(ways % MOD);
        return memo[hat][mask];
    }
}
```

**Key Insight:** DP over hats (40), mask tracks assigned people. O(40 × 2ⁿ × n).

## 16. **Minimum XOR Sum of Two Arrays (Leetcode 1879)**

**Instinct:**  
Two arrays of same length. Permute nums2 to minimize sum of XOR of corresponding elements. DP[mask].

**Approach:**  
1. DP[mask] = min XOR sum using elements of nums2 in mask
2. Index = number of bits set in mask

```java
class Solution {
    int[] nums1, nums2;
    int n;
    int[] memo;
    int INF = Integer.MAX_VALUE / 2;
    
    public int minimumXORSum(int[] nums1, int[] nums2) {
        this.nums1 = nums1;
        this.nums2 = nums2;
        n = nums1.length;
        memo = new int[1 << n];
        Arrays.fill(memo, -1);
        return dfs(0);
    }
    
    private int dfs(int mask) {
        int idx = Integer.bitCount(mask);
        if (idx == n) return 0;
        if (memo[mask] != -1) return memo[mask];
        
        int minSum = INF;
        for (int j = 0; j < n; j++) {
            if ((mask & (1 << j)) == 0) {
                minSum = Math.min(minSum, (nums1[idx] ^ nums2[j]) + dfs(mask | (1 << j)));
            }
        }
        
        memo[mask] = minSum;
        return minSum;
    }
}
```

**Key Insight:** Index = number of assigned elements from nums2. Bitmask tracks used indices.

## 17. **Maximum Number of Groups Getting Fresh Donuts (Leetcode 1815)**

**Instinct:**  
Groups with batch size. Max groups satisfied. Bitmask DP on remainder states.

**Approach:**  
1. State = mask of remainders (batchSize)
2. DP[mask] = max groups satisfied

```java
class Solution {
    int batchSize;
    int[] freq;
    Map<String, Integer> memo;
    
    public int maxHappyGroups(int batchSize, int[] groups) {
        this.batchSize = batchSize;
        freq = new int[batchSize];
        for (int g : groups) {
            freq[g % batchSize]++;
        }
        
        memo = new HashMap<>();
        return freq[0] + dfs(0);
    }
    
    private int dfs(int remainderMask) {
        if (remainderMask == 0) return 0;
        String key = remainderMask + "";
        if (memo.containsKey(key)) return memo.get(key);
        
        int max = 0;
        for (int i = 1; i < batchSize; i++) {
            int count = (remainderMask >> (i * 5)) & 31;
            if (count > 0) {
                int newMask = remainderMask - (1 << (i * 5));
                int newRemainder = (remainderMask >> 20) - i;
                if (newRemainder < 0) newRemainder += batchSize;
                newMask = (newMask & ~(31 << 20)) | (newRemainder << 20);
                
                int val = dfs(newMask);
                if (newRemainder == 0) val++;
                max = Math.max(max, val);
            }
        }
        
        memo.put(key, max);
        return max;
    }
}
```

**Key Insight:** Compress state into integer mask. Each remainder has count (max 31).

## 18. **Maximum Score of a Good Subarray (Bitmask variant)**

**Instinct:**  
Array me subarray with min value constraint. Bitmask not typical for this, but pattern similar.

**Approach:**  
1. Two pointers expansion

```java
class Solution {
    public int maximumScore(int[] nums, int k) {
        int n = nums.length;
        int left = k, right = k;
        int minVal = nums[k];
        int maxScore = minVal;
        
        while (left > 0 || right < n - 1) {
            if (left == 0) {
                right++;
            } else if (right == n - 1) {
                left--;
            } else if (nums[left - 1] < nums[right + 1]) {
                right++;
            } else {
                left--;
            }
            minVal = Math.min(minVal, Math.min(nums[left], nums[right]));
            maxScore = Math.max(maxScore, minVal * (right - left + 1));
        }
        
        return maxScore;
    }
}
```

**Key Insight:** Greedy expansion. Not bitmask but useful pattern.

## 19. **Minimum Cost to Reach Destination with Mask (Leetcode 1434 variant)**

**Instinct:**  
Graph with tolls. Visit certain nodes. DP[mask][node].

**Approach:**  
1. Floyd-Warshall for shortest paths
2. DP[mask][last] = min cost

```java
class Solution {
    int n;
    int[][] dist;
    int[][] memo;
    int INF = Integer.MAX_VALUE / 2;
    
    public int minCostWithMask(int n, int[][] edges, int[] required, int start, int end) {
        this.n = n;
        dist = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(dist[i], INF);
            dist[i][i] = 0;
        }
        for (int[] e : edges) {
            dist[e[0]][e[1]] = Math.min(dist[e[0]][e[1]], e[2]);
            dist[e[1]][e[0]] = Math.min(dist[e[1]][e[0]], e[2]);
        }
        
        for (int k = 0; k < n; k++) {
            for (int i = 0; i < n; i++) {
                for (int j = 0; j < n; j++) {
                    dist[i][j] = Math.min(dist[i][j], dist[i][k] + dist[k][j]);
                }
            }
        }
        
        int reqMask = 0;
        for (int r : required) reqMask |= 1 << r;
        
        memo = new int[1 << n][n];
        for (int i = 0; i < (1 << n); i++) {
            Arrays.fill(memo[i], -1);
        }
        
        return dfs(1 << start, start, reqMask);
    }
    
    private int dfs(int mask, int last, int reqMask) {
        if ((mask & reqMask) == reqMask) {
            return dist[last][n - 1];
        }
        if (memo[mask][last] != -1) return memo[mask][last];
        
        int minCost = INF;
        for (int next = 0; next < n; next++) {
            if ((mask & (1 << next)) == 0) {
                minCost = Math.min(minCost, dist[last][next] + dfs(mask | (1 << next), next, reqMask));
            }
        }
        
        memo[mask][last] = minCost;
        return minCost;
    }
}
```

**Key Insight:** Required nodes mask. Must visit all required nodes.

## 20. **Number of Ways to Reorder Array to Get Same BST (Leetcode 1569)**

**Instinct:**  
Number of ways to reorder array to get same BST. Combinatorics + DP.

**Approach:**  
1. Recursively count ways for left and right subtrees
2. Multiply by combination of interleaving

```java
class Solution {
    int MOD = 1_000_000_007;
    long[][] C;
    
    public int numOfWays(int[] nums) {
        int n = nums.length;
        C = new long[n][n];
        for (int i = 0; i < n; i++) {
            C[i][0] = C[i][i] = 1;
            for (int j = 1; j < i; j++) {
                C[i][j] = (C[i-1][j-1] + C[i-1][j]) % MOD;
            }
        }
        
        List<Integer> list = new ArrayList<>();
        for (int num : nums) list.add(num);
        long ans = dfs(list) - 1;
        return (int)((ans + MOD) % MOD);
    }
    
    private long dfs(List<Integer> nums) {
        if (nums.size() <= 2) return 1;
        
        int root = nums.get(0);
        List<Integer> left = new ArrayList<>();
        List<Integer> right = new ArrayList<>();
        
        for (int i = 1; i < nums.size(); i++) {
            if (nums.get(i) < root) left.add(nums.get(i));
            else right.add(nums.get(i));
        }
        
        long leftWays = dfs(left);
        long rightWays = dfs(right);
        
        int total = left.size() + right.size();
        return C[total][left.size()] * leftWays % MOD * rightWays % MOD;
    }
}
```

**Key Insight:** Recursive DP. Combinations for interleaving left and right.

## 🎯 Key Takeaways for Bitmask DP Pattern

### 1. **Core Bitmask Operations:**

| Operation | Code |
|-----------|------|
| Check if bit is set | `(mask & (1 << i)) != 0` |
| Set bit | `mask | (1 << i)` |
| Clear bit | `mask & ~(1 << i)` |
| Toggle bit | `mask ^ (1 << i)` |
| Count bits | `Integer.bitCount(mask)` |
| Iterate all masks | `for (int mask = 0; mask < (1 << n); mask++)` |
| Iterate subsets of mask | `for (int sub = mask; sub > 0; sub = (sub-1) & mask)` |

### 2. **Common State Patterns:**

| Problem Type | State | Transition |
|--------------|-------|------------|
| TSP | `(mask, last)` | `dp[mask][last] = min(dp[mask ^ (1<<last)][prev] + dist[prev][last])` |
| Assignment | `(mask)` | `dp[mask] = min(dp[mask ^ (1<<j)] + cost[bitCount][j])` |
| Subset Sum | `(mask, sum)` | `dp[mask][sum] = dp[mask ^ (1<<i)][sum - nums[i]]` |
| Row DP | `(row, mask)` | `dp[row][mask] = max(dp[row-1][prev] + count(mask))` |

### 3. **When to use Bitmask DP:**
- n ≤ 20 (2ⁿ is manageable)
- Need to track which elements are used/visited
- Order of processing matters
- Subset selection problems
- Permutation problems

### 4. **Bitmask DP Optimization Techniques:**

| Technique | Use Case |
|-----------|----------|
| Precompute valid masks | Row DP problems |
| Subset enumeration | `for (sub = mask; sub > 0; sub = (sub-1) & mask)` |
| Meet-in-the-middle | When 2ⁿ is too large (n ≤ 40) |
| BFS on mask | Unweighted graph problems |
| Floyd-Warshall + Bitmask | Precompute shortest paths |

### 5. **Common Bitmask Sizes:**

| n | 2ⁿ | Feasibility |
|---|-----|--------------|
| ≤ 15 | ≤ 32,768 | Very fast |
| ≤ 20 | ≤ 1,048,576 | Acceptable |
| ≤ 25 | ≤ 33 million | Might be slow |
| ≤ 30 | ≤ 1 billion | Too large |

### 6. **Bitmask DP Template:**

```java
// 1. State: mask (and sometimes last)
int[][] dp = new int[1 << n][n];
Arrays.fill(dp, INF);
dp[1 << start][start] = 0;

// 2. Iterate over masks
for (int mask = 0; mask < (1 << n); mask++) {
    for (int last = 0; last < n; last++) {
        if (dp[mask][last] == INF) continue;
        // Try all next states
        for (int next = 0; next < n; next++) {
            if ((mask & (1 << next)) == 0) {
                int newMask = mask | (1 << next);
                dp[newMask][next] = min(dp[newMask][next], 
                    dp[mask][last] + cost[last][next]);
            }
        }
    }
}
```


# Palindrome DP Pattern

## Palindrome DP ke 10 Main Variants:

| Variant | Description | Approach |
|---------|-------------|----------|
| **Longest Palindromic Subsequence (LPS)** | Longest palindrome (non-contiguous) | LCS of string and reverse |
| **Longest Palindromic Substring** | Longest palindrome (contiguous) | Expand around center or DP |
| **Palindromic Substrings Count** | Count of all palindromic substrings | Expand around center |
| **Minimum Insertions to Palindrome** | Min insertions to make palindrome | n - LPS |
| **Minimum Deletions to Palindrome** | Min deletions to make palindrome | n - LPS |
| **Palindrome Partitioning** | Partition into palindromes | DP + backtracking |
| **Palindrome Partitioning II (Min Cuts)** | Min cuts to make all parts palindrome | DP with palindrome check |
| **Palindrome Partitioning III** | Min changes to make k palindromes | DP with cost array |
| **Distinct Palindromic Subsequences** | Count distinct palindromic subsequences | DP with Set / 2 pointers |
| **Shortest Palindrome** | Add chars to front to make palindrome | KMP / Rolling hash |

## 1. **Longest Palindromic Subsequence (Leetcode 516)**

**Instinct:**  
String ka longest palindrome (non-contiguous). Palindromic property: first aur last char same hona chahiye, ya toh ek char ko skip karo. Memoization me **i aur j** track karo.

**Approach:**  
1. Memo[i][j] = LPS length for s[i..j]
2. Base: i > j → 0; i == j → 1
3. If s[i] == s[j] → 2 + dfs(i+1, j-1)
4. Else → max(dfs(i+1, j), dfs(i, j-1))

```java
class Solution {
    String s;
    int[][] memo;
    
    public int longestPalindromeSubseq(String s) {
        this.s = s;
        int n = s.length();
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        return dfs(0, n - 1);
    }
    
    private int dfs(int i, int j) {
        if (i > j) return 0;
        if (i == j) return 1;
        if (memo[i][j] != -1) return memo[i][j];
        
        int res;
        if (s.charAt(i) == s.charAt(j)) {
            res = 2 + dfs(i + 1, j - 1);
        } else {
            res = Math.max(dfs(i + 1, j), dfs(i, j - 1));
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** Classic LPS - either match ends or skip from left/right. O(n²) time.

## 2. **Longest Palindromic Substring (Leetcode 5)**

**Instinct:**  
String ka longest palindrome (contiguous). Expand around center approach is O(n²). Memoization bhi possible hai - check if s[i..j] is palindrome.

**Approach:**  
1. Memo[i][j] = is s[i..j] palindrome? (-1 unknown, 0 false, 1 true)
2. Base: i > j → true; i == j → true
3. If s[i] == s[j] and dfs(i+1, j-1) is true → palindrome
4. Track max length

```java
class Solution {
    String s;
    int[][] memo;
    int maxLen = 0;
    int start = 0;
    
    public String longestPalindrome(String s) {
        this.s = s;
        int n = s.length();
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        for (int i = 0; i < n; i++) {
            for (int j = i; j < n; j++) {
                if (isPalindrome(i, j) && j - i + 1 > maxLen) {
                    maxLen = j - i + 1;
                    start = i;
                }
            }
        }
        return s.substring(start, start + maxLen);
    }
    
    private boolean isPalindrome(int i, int j) {
        if (i > j) return true;
        if (i == j) return true;
        if (memo[i][j] != -1) return memo[i][j] == 1;
        
        boolean res = false;
        if (s.charAt(i) == s.charAt(j)) {
            res = isPalindrome(i + 1, j - 1);
        }
        
        memo[i][j] = res ? 1 : 0;
        return res;
    }
}
```

**Key Insight:** Substring check with memoization. Expand around center is more efficient for this problem.

## 3. **Palindromic Substrings (Leetcode 647)**

**Instinct:**  
String me kitne palindromic substrings hain (contiguous). Har substring check karo using DP.

**Approach:**  
1. Memo[i][j] = is s[i..j] palindrome?
2. Count all i,j where palindrome true

```java
class Solution {
    String s;
    int[][] memo;
    int count = 0;
    
    public int countSubstrings(String s) {
        this.s = s;
        int n = s.length();
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        for (int i = 0; i < n; i++) {
            for (int j = i; j < n; j++) {
                if (isPalindrome(i, j)) {
                    count++;
                }
            }
        }
        return count;
    }
    
    private boolean isPalindrome(int i, int j) {
        if (i > j) return true;
        if (i == j) return true;
        if (memo[i][j] != -1) return memo[i][j] == 1;
        
        boolean res = false;
        if (s.charAt(i) == s.charAt(j)) {
            res = isPalindrome(i + 1, j - 1);
        }
        
        memo[i][j] = res ? 1 : 0;
        return res;
    }
}
```

**Key Insight:** Count all palindromic substrings. Same palindrome check as before.

## 4. **Minimum Insertion Steps to Make a String Palindrome (Leetcode 1312)**

**Instinct:**  
String ko palindrome banane ke liye minimum insertions. Insertions = n - LPS length.

**Approach:**  
1. Find LPS length using LCS(s, reverse(s)) or interval DP
2. Answer = n - LPS

```java
class Solution {
    String s;
    int[][] memo;
    
    public int minInsertions(String s) {
        this.s = s;
        int n = s.length();
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        int lps = dfs(0, n - 1);
        return n - lps;
    }
    
    private int dfs(int i, int j) {
        if (i > j) return 0;
        if (i == j) return 1;
        if (memo[i][j] != -1) return memo[i][j];
        
        int res;
        if (s.charAt(i) == s.charAt(j)) {
            res = 2 + dfs(i + 1, j - 1);
        } else {
            res = Math.max(dfs(i + 1, j), dfs(i, j - 1));
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** Minimum insertions = n - LPS. Ek baar LPS mil gaya, baaki chars insert karne honge.

## 5. **Palindrome Partitioning (Leetcode 131)**

**Instinct:**  
String ko partition karna hai into palindromic substrings. Saare possible partitions return karne hain. Backtracking + palindrome check.

**Approach:**  
1. Memo[i] = list of partitions from index i
2. For each j >= i, if s[i..j] is palindrome, then add to result with dfs(j+1)

```java
class Solution {
    String s;
    List<List<String>>[] memo;
    
    public List<List<String>> partition(String s) {
        this.s = s;
        int n = s.length();
        memo = new List[n + 1];
        return dfs(0);
    }
    
    private List<List<String>> dfs(int start) {
        if (start == s.length()) {
            List<List<String>> empty = new ArrayList<>();
            empty.add(new ArrayList<>());
            return empty;
        }
        if (memo[start] != null) return memo[start];
        
        List<List<String>> res = new ArrayList<>();
        for (int end = start; end < s.length(); end++) {
            if (isPalindrome(start, end)) {
                String palindrome = s.substring(start, end + 1);
                List<List<String>> nextPartitions = dfs(end + 1);
                for (List<String> next : nextPartitions) {
                    List<String> current = new ArrayList<>();
                    current.add(palindrome);
                    current.addAll(next);
                    res.add(current);
                }
            }
        }
        
        memo[start] = res;
        return res;
    }
    
    private boolean isPalindrome(int i, int j) {
        while (i < j) {
            if (s.charAt(i) != s.charAt(j)) return false;
            i++;
            j--;
        }
        return true;
    }
}
```

**Key Insight:** Partition backtracking with memoization. Har palindrome milne par aage ke partitions join karo.

## 6. **Palindrome Partitioning II (Leetcode 132)**

**Instinct:**  
Minimum cuts to partition string into palindromes. Ye min cuts wala variant hai.

**Approach:**  
1. Memo[i] = min cuts from index i to end
2. For each j >= i, if s[i..j] is palindrome, then cuts = 1 + dfs(j+1)
3. Base: i == n → -1 (no cut needed at end)

```java
class Solution {
    String s;
    int[] memo;
    Boolean[][] isPal;
    
    public int minCut(String s) {
        this.s = s;
        int n = s.length();
        memo = new int[n];
        Arrays.fill(memo, -1);
        isPal = new Boolean[n][n];
        
        return dfs(0);
    }
    
    private int dfs(int i) {
        if (i == s.length()) return -1;
        if (memo[i] != -1) return memo[i];
        
        int minCuts = Integer.MAX_VALUE;
        for (int j = i; j < s.length(); j++) {
            if (isPalindrome(i, j)) {
                int cuts = 1 + dfs(j + 1);
                minCuts = Math.min(minCuts, cuts);
            }
        }
        
        memo[i] = minCuts;
        return minCuts;
    }
    
    private boolean isPalindrome(int i, int j) {
        if (i >= j) return true;
        if (isPal[i][j] != null) return isPal[i][j];
        
        boolean res = (s.charAt(i) == s.charAt(j)) && isPalindrome(i + 1, j - 1);
        isPal[i][j] = res;
        return res;
    }
}
```

**Key Insight:** Min cuts = min(1 + cuts from next index). Memoization on start index.

## 7. **Palindrome Partitioning III (Leetcode 1278)**

**Instinct:**  
String ko exactly k palindromic parts me divide karna hai. Minimum characters change karne honge to make each part palindrome.

**Approach:**  
1. Precompute cost[i][j] = changes needed to make s[i..j] palindrome
2. Memo[i][k] = min changes from index i with k parts remaining
3. Try all partitions

```java
class Solution {
    String s;
    int[][] cost;
    int[][] memo;
    
    public int palindromePartition(String s, int k) {
        this.s = s;
        int n = s.length();
        
        // Precompute cost to make s[i..j] palindrome
        cost = new int[n][n];
        for (int len = 2; len <= n; len++) {
            for (int i = 0; i + len - 1 < n; i++) {
                int j = i + len - 1;
                cost[i][j] = cost[i + 1][j - 1] + (s.charAt(i) == s.charAt(j) ? 0 : 1);
            }
        }
        
        memo = new int[n][k + 1];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        return dfs(0, k);
    }
    
    private int dfs(int i, int k) {
        if (i == s.length()) return 0;
        if (k == 1) return cost[i][s.length() - 1];
        if (memo[i][k] != -1) return memo[i][k];
        
        int minChanges = Integer.MAX_VALUE;
        for (int j = i; j <= s.length() - k; j++) {
            int changes = cost[i][j] + dfs(j + 1, k - 1);
            minChanges = Math.min(minChanges, changes);
        }
        
        memo[i][k] = minChanges;
        return minChanges;
    }
}
```

**Key Insight:** Precompute palindrome change cost. Then partition DP with k parts.

## 8. **Shortest Palindrome (Leetcode 214)**

**Instinct:**  
String ke aage characters add karke palindrome banana hai. Shortest possible palindrome. Find longest palindromic prefix.

**Approach:**  
1. Find longest prefix which is palindrome
2. Add reverse of remaining suffix to front

```java
class Solution {
    public String shortestPalindrome(String s) {
        if (s == null || s.length() <= 1) return s;
        
        int n = s.length();
        // Find longest palindromic prefix
        for (int i = n - 1; i >= 0; i--) {
            if (isPalindrome(s, 0, i)) {
                String suffix = s.substring(i + 1);
                String reversed = new StringBuilder(suffix).reverse().toString();
                return reversed + s;
            }
        }
        return "";
    }
    
    private boolean isPalindrome(String s, int i, int j) {
        while (i < j) {
            if (s.charAt(i) != s.charAt(j)) return false;
            i++;
            j--;
        }
        return true;
    }
}
```

**Key Insight:** Shortest palindrome = reverse(suffix) + s, where prefix is longest palindrome.

## 9. **Palindrome Removal (Leetcode 1246)**

**Instinct:**  
Array se palindromic subsequences remove karte ho. Minimum operations to remove all elements. Ek operation me palindrome subsequence remove kar sakte ho.

**Approach:**  
1. Memo[i][j] = min operations to remove arr[i..j]
2. Base: i > j → 0; i == j → 1
3. If arr[i] == arr[j] → ops = dfs(i+1, j-1)
4. Else → try all partitions

```java
class Solution {
    int[] arr;
    int[][] memo;
    
    public int minimumMoves(int[] arr) {
        this.arr = arr;
        int n = arr.length;
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        return dfs(0, n - 1);
    }
    
    private int dfs(int i, int j) {
        if (i > j) return 0;
        if (i == j) return 1;
        if (memo[i][j] != -1) return memo[i][j];
        
        int res = Integer.MAX_VALUE;
        
        // Option 1: Remove arr[i] alone
        res = Math.min(res, 1 + dfs(i + 1, j));
        
        // Option 2: Try to match arr[i] with some k where arr[k] == arr[i]
        for (int k = i + 1; k <= j; k++) {
            if (arr[k] == arr[i]) {
                // If adjacent, can remove together
                if (k == i + 1) {
                    res = Math.min(res, 1 + dfs(i + 2, j));
                } else {
                    res = Math.min(res, dfs(i + 1, k - 1) + dfs(k + 1, j));
                }
            }
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** Palindromic removal - match same values to remove together.

## 10. **Construct Palindrome (Longest Palindromic Subsequence with Constraints)**

**Instinct:**  
Two strings se palindrome build karna hai using subsequences. Maximum length possible.

**Approach:**  
1. Memo[i][j][k][l] = LPS length using s1[i..j] and s2[k..l]
2. Try taking from s1, s2, or both

```java
class Solution {
    String s1, s2;
    int[][][][] memo;
    
    public int longestPalindrome(String s1, String s2) {
        this.s1 = s1;
        this.s2 = s2;
        int n = s1.length(), m = s2.length();
        memo = new int[n][n][m][m];
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                for (int k = 0; k < m; k++) {
                    Arrays.fill(memo[i][j][k], -1);
                }
            }
        }
        return dfs(0, n - 1, 0, m - 1);
    }
    
    private int dfs(int i1, int j1, int i2, int j2) {
        if (i1 > j1 && i2 > j2) return 0;
        if (i1 > j1) {
            // Only s2 left, find LPS in s2[i2..j2]
            return lps(s2, i2, j2);
        }
        if (i2 > j2) {
            // Only s1 left
            return lps(s1, i1, j1);
        }
        if (memo[i1][j1][i2][j2] != -1) return memo[i1][j1][i2][j2];
        
        int res = 0;
        // Try matching from both strings
        if (s1.charAt(i1) == s2.charAt(j2)) {
            res = Math.max(res, 2 + dfs(i1 + 1, j1, i2, j2 - 1));
        }
        if (s1.charAt(j1) == s2.charAt(i2)) {
            res = Math.max(res, 2 + dfs(i1, j1 - 1, i2 + 1, j2));
        }
        // Try matching within same string
        res = Math.max(res, dfs(i1 + 1, j1, i2, j2));
        res = Math.max(res, dfs(i1, j1 - 1, i2, j2));
        res = Math.max(res, dfs(i1, j1, i2 + 1, j2));
        res = Math.max(res, dfs(i1, j1, i2, j2 - 1));
        
        memo[i1][j1][i2][j2] = res;
        return res;
    }
    
    private int lps(String s, int i, int j) {
        // Standard LPS for single string
        if (i > j) return 0;
        if (i == j) return 1;
        if (s.charAt(i) == s.charAt(j)) {
            return 2 + lps(s, i + 1, j - 1);
        }
        return Math.max(lps(s, i + 1, j), lps(s, i, j - 1));
    }
}
```

**Key Insight:** 4D DP for two strings. Complex but follows same palindrome logic.

## 11. **Valid Palindrome II (Leetcode 680)**

**Instinct:**  
String ko at most ek character delete karke palindrome ban sakta hai ya nahi. Two pointers with one skip allowed.

**Approach:**  
1. Two pointers i and j
2. If chars match → move both
3. If mismatch → try deleting from left or right

```java
class Solution {
    String s;
    int[][] memo;
    
    public boolean validPalindrome(String s) {
        this.s = s;
        int n = s.length();
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        return isPalindrome(0, n - 1, true);
    }
    
    private boolean isPalindrome(int i, int j, boolean canDelete) {
        if (i >= j) return true;
        if (memo[i][j] != -1 && !canDelete) return memo[i][j] == 1;
        
        boolean res;
        if (s.charAt(i) == s.charAt(j)) {
            res = isPalindrome(i + 1, j - 1, canDelete);
        } else if (canDelete) {
            res = isPalindrome(i + 1, j, false) || isPalindrome(i, j - 1, false);
        } else {
            res = false;
        }
        
        if (!canDelete) memo[i][j] = res ? 1 : 0;
        return res;
    }
}
```

**Key Insight:** One deletion allowed. Try both options when mismatch.

## 12. **Can Make Palindrome from Substring (Leetcode 1177)**

**Instinct:**  
Query: can substring be rearranged into palindrome with at most k replacements? For palindrome, at most one odd count allowed.

**Approach:**  
1. Prefix count of each character
2. For substring, count odd frequencies
3. Odd count / 2 <= k

```java
class Solution {
    public List<Boolean> canMakePaliQueries(String s, int[][] queries) {
        int n = s.length();
        int[][] prefix = new int[n + 1][26];
        
        for (int i = 0; i < n; i++) {
            for (int c = 0; c < 26; c++) {
                prefix[i + 1][c] = prefix[i][c];
            }
            prefix[i + 1][s.charAt(i) - 'a']++;
        }
        
        List<Boolean> res = new ArrayList<>();
        for (int[] q : queries) {
            int left = q[0], right = q[1], k = q[2];
            int oddCount = 0;
            for (int c = 0; c < 26; c++) {
                int freq = prefix[right + 1][c] - prefix[left][c];
                if (freq % 2 == 1) oddCount++;
            }
            res.add(oddCount / 2 <= k);
        }
        return res;
    }
}
```

**Key Insight:** Palindrome rearrangement condition: at most one odd frequency. Each replacement can fix two odds.

## 13. **Longest Palindromic Subsequence After at Most K Replacements**

**Instinct:**  
LPS but allowed to change up to K characters. Extra dimension for remaining changes.

**Approach:**  
1. Memo[i][j][k] = LPS length with k changes left
2. If s[i] == s[j] → 2 + dfs(i+1, j-1, k)
3. Else → try changing one of them (if k>0) or skip

```java
class Solution {
    String s;
    int[][][] memo;
    
    public int longestPalindromicSubsequenceWithChanges(String s, int k) {
        this.s = s;
        int n = s.length();
        memo = new int[n][n][k + 1];
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        return dfs(0, n - 1, k);
    }
    
    private int dfs(int i, int j, int k) {
        if (i > j) return 0;
        if (i == j) return 1;
        if (memo[i][j][k] != -1) return memo[i][j][k];
        
        int res = Math.max(dfs(i + 1, j, k), dfs(i, j - 1, k));
        
        if (s.charAt(i) == s.charAt(j)) {
            res = Math.max(res, 2 + dfs(i + 1, j - 1, k));
        } else if (k > 0) {
            // Change either s[i] or s[j] to match
            res = Math.max(res, 2 + dfs(i + 1, j - 1, k - 1));
        }
        
        memo[i][j][k] = res;
        return res;
    }
}
```

**Key Insight:** Changes allowed to make chars match. Extra dimension for remaining changes.

## 14. **Count Different Palindromic Subsequences (Leetcode 730)**

**Instinct:**  
Distinct palindromic subsequences count karni hai. Duplicates avoid karna hai.

**Approach:**  
1. Memo[i][j] = set of palindromes in s[i..j]
2. If s[i] == s[j], add new palindromes formed with s[i]
3. Use 2 pointers to find next same char to avoid duplicates

```java
class Solution {
    int MOD = 1_000_000_007;
    String s;
    Integer[][] memo;
    
    public int countPalindromicSubsequences(String s) {
        this.s = s;
        int n = s.length();
        memo = new Integer[n][n];
        return dfs(0, n - 1);
    }
    
    private int dfs(int i, int j) {
        if (i > j) return 0;
        if (i == j) return 1;
        if (memo[i][j] != null) return memo[i][j];
        
        long res = 0;
        if (s.charAt(i) == s.charAt(j)) {
            int left = i + 1, right = j - 1;
            while (left <= right && s.charAt(left) != s.charAt(i)) left++;
            while (left <= right && s.charAt(right) != s.charAt(i)) right--;
            
            if (left > right) {
                res = 2 * dfs(i + 1, j - 1) + 2;
            } else if (left == right) {
                res = 2 * dfs(i + 1, j - 1) + 1;
            } else {
                res = 2 * dfs(i + 1, j - 1) - dfs(left + 1, right - 1);
            }
        } else {
            res = dfs(i + 1, j) + dfs(i, j - 1) - dfs(i + 1, j - 1);
        }
        
        memo[i][j] = (int)((res + MOD) % MOD);
        return memo[i][j];
    }
}
```

**Key Insight:** Use inclusion-exclusion to avoid double counting. Find next/prev same char.

## 15. **Palindrome Pairs (Leetcode 336)**

**Instinct:**  
Words ki pairs (i,j) such that words[i] + words[j] is palindrome. HashMap use karo for reverse words.

**Approach:**  
1. HashMap of word → index
2. For each word, check all prefixes and suffixes
3. If prefix is palindrome, check if reverse of suffix exists

```java
class Solution {
    public List<List<Integer>> palindromePairs(String[] words) {
        List<List<Integer>> res = new ArrayList<>();
        Map<String, Integer> map = new HashMap<>();
        
        for (int i = 0; i < words.length; i++) {
            map.put(words[i], i);
        }
        
        for (int i = 0; i < words.length; i++) {
            String word = words[i];
            int n = word.length();
            
            // Check all possible splits
            for (int j = 0; j <= n; j++) {
                String prefix = word.substring(0, j);
                String suffix = word.substring(j);
                
                if (isPalindrome(prefix)) {
                    String revSuffix = new StringBuilder(suffix).reverse().toString();
                    if (map.containsKey(revSuffix) && map.get(revSuffix) != i) {
                        res.add(Arrays.asList(map.get(revSuffix), i));
                    }
                }
                
                if (j != n && isPalindrome(suffix)) {
                    String revPrefix = new StringBuilder(prefix).reverse().toString();
                    if (map.containsKey(revPrefix) && map.get(revPrefix) != i) {
                        res.add(Arrays.asList(i, map.get(revPrefix)));
                    }
                }
            }
        }
        return res;
    }
    
    private boolean isPalindrome(String s) {
        int i = 0, j = s.length() - 1;
        while (i < j) {
            if (s.charAt(i) != s.charAt(j)) return false;
            i++;
            j--;
        }
        return true;
    }
}
```

**Key Insight:** For each split, if one part is palindrome, check if reverse of other part exists.

## 16. **Minimum Number of Moves to Make Palindrome (Leetcode 2193)**

**Instinct:**  
String ko palindrome banane ke liye minimum adjacent swaps. Greedy approach - match from ends.

**Approach:**  
1. Two pointers from left and right
2. If chars match, move both
3. If not, find matching char from right and swap

```java
class Solution {
    public int minMovesToMakePalindrome(String s) {
        char[] arr = s.toCharArray();
        int n = arr.length;
        int moves = 0;
        int left = 0, right = n - 1;
        
        while (left < right) {
            if (arr[left] == arr[right]) {
                left++;
                right--;
            } else {
                // Find matching char for arr[left] from right side
                int mid = right;
                while (mid > left && arr[mid] != arr[left]) {
                    mid--;
                }
                
                if (mid == left) {
                    // Odd length palindrome, swap with next
                    char temp = arr[mid];
                    arr[mid] = arr[mid + 1];
                    arr[mid + 1] = temp;
                    moves++;
                    continue;
                }
                
                // Swap to bring matching char to right position
                for (int k = mid; k < right; k++) {
                    char temp = arr[k];
                    arr[k] = arr[k + 1];
                    arr[k + 1] = temp;
                    moves++;
                }
                left++;
                right--;
            }
        }
        return moves;
    }
}
```

**Key Insight:** Greedy matching from ends. If no match, swap with next (odd length case).

## 17. **Largest Palindromic Number (Leetcode 2384)**

**Instinct:**  
Digits se largest palindrome banani hai. Even counts use karo, ek odd count center me.

**Approach:**  
1. Count frequency of each digit
2. Build half palindrome using even counts
3. Add largest odd digit in center

```java
class Solution {
    public String largestPalindromic(String num) {
        int[] count = new int[10];
        for (char c : num.toCharArray()) {
            count[c - '0']++;
        }
        
        StringBuilder half = new StringBuilder();
        String middle = "";
        
        // Build half from largest digit to smallest
        for (int d = 9; d >= 0; d--) {
            if (count[d] > 0) {
                if (d != 0 || half.length() > 0) {
                    int pairs = count[d] / 2;
                    for (int i = 0; i < pairs; i++) {
                        half.append(d);
                    }
                }
                if (count[d] % 2 == 1 && middle.isEmpty()) {
                    middle = String.valueOf(d);
                }
            }
        }
        
        if (half.length() == 0 && middle.isEmpty()) return "0";
        
        return half.toString() + middle + half.reverse().toString();
    }
}
```

**Key Insight:** Palindrome from digits - use even counts, one odd in center.

## 18. **Break a Palindrome (Leetcode 1328)**

**Instinct:**  
Palindrome ko break karna hai by replacing one character to make it NOT palindrome. Smallest lexicographically.

**Approach:**  
1. Change first non-'a' char to 'a'
2. If all are 'a', change last char to 'b'

```java
class Solution {
    public String breakPalindrome(String palindrome) {
        int n = palindrome.length();
        if (n == 1) return "";
        
        char[] arr = palindrome.toCharArray();
        
        for (int i = 0; i < n / 2; i++) {
            if (arr[i] != 'a') {
                arr[i] = 'a';
                return new String(arr);
            }
        }
        
        arr[n - 1] = 'b';
        return new String(arr);
    }
}
```

**Key Insight:** Change first non-'a' to 'a'. If all 'a's, change last to 'b'.

## 20. **Palindromic Substrings in a Binary String**

**Instinct:**  
Binary string me palindromic substrings count. Same as regular palindrome count but with binary.

**Approach:**  
1. Same as count palindromic substrings
2. Expand around center

```java
class Solution {
    public int countBinaryPalindromicSubstrings(String s) {
        int n = s.length();
        int count = 0;
        
        // Odd length palindromes
        for (int center = 0; center < n; center++) {
            count += expandAroundCenter(s, center, center);
        }
        
        // Even length palindromes
        for (int center = 0; center < n - 1; center++) {
            count += expandAroundCenter(s, center, center + 1);
        }
        
        return count;
    }
    
    private int expandAroundCenter(String s, int left, int right) {
        int count = 0;
        while (left >= 0 && right < s.length() && s.charAt(left) == s.charAt(right)) {
            count++;
            left--;
            right++;
        }
        return count;
    }
}
```

**Key Insight:** Binary string palindromes same logic. Expand around center O(n²).

## 🎯 Key Takeaways for Palindrome DP Pattern

### 1. **Core Recurrence for LPS:**
```java
if (s[i] == s[j]) {
    dp[i][j] = 2 + dp[i+1][j-1];
} else {
    dp[i][j] = max(dp[i+1][j], dp[i][j-1]);
}
```

### 2. **Palindrome Check Recurrence:**
```java
isPalindrome(i, j) = (s[i] == s[j]) && isPalindrome(i+1, j-1)
```

### 3. **Common Patterns:**

| Problem Type | Approach |
|-------------|----------|
| Length/Count | Interval DP on (i,j) |
| Min cuts | DP[i] = min(1 + DP[j+1]) |
| All partitions | Backtracking + memoization |
| With changes | Extra dimension for changes |
| Two strings | LCS of string and reverse |

### 4. **Optimization Techniques:**
- **Expand around center**: O(n²) time, O(1) space for substring counting
- **Manacher's Algorithm**: O(n) time for longest palindromic substring
- **Prefix sums**: For range queries on palindrome rearrangement

### 5. **When to use Palindrome DP:**
- Need longest palindrome (subsequence/substring)
- Partition string into palindromes
- Min operations to make palindrome
- Count distinct palindromic subsequences
- Palindrome related queries

### 6. **State Space Patterns:**

```
Classic LPS:     (i, j)                    - O(n²)
With changes:    (i, j, k)                 - O(n²k)
Two strings:     (i1, j1, i2, j2)          - O(n⁴)
Partition DP:    (i, k)                    - O(nk)
```

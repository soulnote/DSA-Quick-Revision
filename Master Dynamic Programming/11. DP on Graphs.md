
# DP on Graphs Pattern 

## Graph DP ke 10 Main Variants:

| Variant | Description | Approach |
|---------|-------------|----------|
| **Shortest Path (DAG)** | Shortest path in DAG | Topological order + DP |
| **Longest Path in DAG** | Longest path in DAG | Topological order + DP |
| **Number of Paths** | Count paths from source to dest | DP on DAG |
| **Cheapest Flight with K Stops** | Shortest path with constraints | DP[stops][node] |
| **Minimum Cost to Reach Destination** | Path with minimum cost | Dijkstra / DP |
| **Graph Coloring** | Color graph with constraints | DP with bitmask |
| **Hamiltonian Path** | Path visiting all nodes exactly once | DP[mask][last] |
| **Traveling Salesman (TSP)** | Shortest cycle visiting all nodes | DP[mask][last] |
| **Shortest Path in Grid with Obstacles** | Grid path finding | BFS / DP |
| **Maximum Probability Path** | Path with max probability product | Dijkstra variant |

## 1. **Cheapest Flights Within K Stops (Leetcode 787)**

**Instinct:**  
Source se destination tak maximum K stops me cheapest price. DP[stops][node] = min cost to reach node with exactly 'stops' stops. Ya phir BFS/DP with stops as dimension.

**Approach:**  
1. Memo[stops][node] = min cost from node to destination with 'stops' remaining
2. Try all neighbors, use one stop
3. Base: if node == dest → 0
4. If stops == 0 → INF (except destination)

```java
class Solution {
    List<int[]>[] graph;
    int dest;
    int[][] memo;
    int INF = Integer.MAX_VALUE / 2;
    
    public int findCheapestPrice(int n, int[][] flights, int src, int dst, int k) {
        graph = new List[n];
        for (int i = 0; i < n; i++) graph[i] = new ArrayList<>();
        for (int[] f : flights) {
            graph[f[0]].add(new int[]{f[1], f[2]});
        }
        
        this.dest = dst;
        memo = new int[n][k + 2];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        int ans = dfs(src, k + 1);
        return ans >= INF ? -1 : ans;
    }
    
    private int dfs(int node, int stops) {
        if (node == dest) return 0;
        if (stops == 0) return INF;
        if (memo[node][stops] != -1) return memo[node][stops];
        
        int minCost = INF;
        for (int[] next : graph[node]) {
            int cost = next[1] + dfs(next[0], stops - 1);
            minCost = Math.min(minCost, cost);
        }
        
        memo[node][stops] = minCost;
        return minCost;
    }
}
```

**Key Insight:** Stops as dimension in DP. Try all neighbors with one stop remaining.

## 2. **Longest Path in DAG**

**Instinct:**  
Directed Acyclic Graph me longest path. Topological order + DP, ya DFS with memoization.

**Approach:**  
1. Memo[node] = longest path starting from node
2. For each neighbor, length = 1 + dfs(neighbor)
3. Take max over all neighbors

```java
class Solution {
    List<Integer>[] graph;
    int[] memo;
    
    public int longestPathInDAG(int n, int[][] edges) {
        graph = new List[n];
        for (int i = 0; i < n; i++) graph[i] = new ArrayList<>();
        for (int[] e : edges) {
            graph[e[0]].add(e[1]);
        }
        
        memo = new int[n];
        Arrays.fill(memo, -1);
        
        int maxLen = 0;
        for (int i = 0; i < n; i++) {
            maxLen = Math.max(maxLen, dfs(i));
        }
        return maxLen;
    }
    
    private int dfs(int node) {
        if (memo[node] != -1) return memo[node];
        
        int max = 0;
        for (int next : graph[node]) {
            max = Math.max(max, 1 + dfs(next));
        }
        
        memo[node] = max;
        return max;
    }
}
```

**Key Insight:** DAG cycles nahi hai, isliye memoization safe hai. Path length = 1 + max(neighbor lengths).

## 3. **Number of Paths in DAG (Source to Destination)**

**Instinct:**  
DAG me source se destination tak kitne paths hain. DP with memoization.

**Approach:**  
1. Memo[node] = number of paths from node to destination
2. Base: node == dest → 1
3. Sum over all neighbors

```java
class Solution {
    List<Integer>[] graph;
    int dest;
    int[] memo;
    
    public int countPaths(int n, int[][] edges, int src, int dst) {
        graph = new List[n];
        for (int i = 0; i < n; i++) graph[i] = new ArrayList<>();
        for (int[] e : edges) {
            graph[e[0]].add(e[1]);
        }
        
        this.dest = dst;
        memo = new int[n];
        Arrays.fill(memo, -1);
        
        return dfs(src);
    }
    
    private int dfs(int node) {
        if (node == dest) return 1;
        if (memo[node] != -1) return memo[node];
        
        int paths = 0;
        for (int next : graph[node]) {
            paths += dfs(next);
        }
        
        memo[node] = paths;
        return paths;
    }
}
```

**Key Insight:** DAG me cycle nahi hai. Sum over all neighbor paths.

## 4. **Hamiltonian Path (DP on Bitmask)**

**Instinct:**  
Graph me path jo har node exactly ek baar visit kare. DP[mask][last] = can we form path with visited nodes = mask ending at last.

**Approach:**  
1. Memo[mask][last] = is Hamiltonian path possible with visited mask ending at last
2. Try all unvisited nodes as next
3. Base: mask has all nodes → true

```java
class Solution {
    int n;
    boolean[][] graph;
    Boolean[][] memo;
    
    public boolean hasHamiltonianPath(int n, int[][] edges) {
        this.n = n;
        graph = new boolean[n][n];
        for (int[] e : edges) {
            graph[e[0]][e[1]] = true;
            graph[e[1]][e[0]] = true;
        }
        
        memo = new Boolean[1 << n][n];
        
        for (int start = 0; start < n; start++) {
            if (dfs(1 << start, start)) return true;
        }
        return false;
    }
    
    private boolean dfs(int mask, int last) {
        if (mask == (1 << n) - 1) return true;
        if (memo[mask][last] != null) return memo[mask][last];
        
        for (int next = 0; next < n; next++) {
            if ((mask & (1 << next)) == 0 && graph[last][next]) {
                if (dfs(mask | (1 << next), next)) {
                    memo[mask][last] = true;
                    return true;
                }
            }
        }
        
        memo[mask][last] = false;
        return false;
    }
}
```

**Key Insight:** Bitmask DP - mask tracks visited nodes, last tracks current node.

## 5. **Traveling Salesman Problem (TSP) - Shortest Path Visiting All Nodes**

**Instinct:**  
Har node exactly ek baar visit karte hue shortest cycle. DP[mask][last] = min cost to visit nodes in mask ending at last.

**Approach:**  
1. Memo[mask][last] = min cost to visit all nodes in mask ending at last
2. Try all unvisited nodes as next
3. Base: mask has all nodes → cost to return to start

```java
class Solution {
    int n;
    int[][] dist;
    int[][] memo;
    int INF = Integer.MAX_VALUE / 2;
    
    public int shortestPathVisitingAllNodes(int[][] graph) {
        n = graph.length;
        
        // Floyd-Warshall for shortest paths
        dist = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(dist[i], INF);
            dist[i][i] = 0;
            for (int j : graph[i]) {
                dist[i][j] = 1;
            }
        }
        for (int k = 0; k < n; k++) {
            for (int i = 0; i < n; i++) {
                for (int j = 0; j < n; j++) {
                    dist[i][j] = Math.min(dist[i][j], dist[i][k] + dist[k][j]);
                }
            }
        }
        
        memo = new int[1 << n][n];
        for (int i = 0; i < (1 << n); i++) {
            Arrays.fill(memo[i], -1);
        }
        
        int minCost = INF;
        for (int start = 0; start < n; start++) {
            minCost = Math.min(minCost, dfs(1 << start, start));
        }
        return minCost;
    }
    
    private int dfs(int mask, int last) {
        if (mask == (1 << n) - 1) return 0;
        if (memo[mask][last] != -1) return memo[mask][last];
        
        int minCost = INF;
        for (int next = 0; next < n; next++) {
            if ((mask & (1 << next)) == 0) {
                int cost = dist[last][next] + dfs(mask | (1 << next), next);
                minCost = Math.min(minCost, cost);
            }
        }
        
        memo[mask][last] = minCost;
        return minCost;
    }
}
```

**Key Insight:** TSP = DP[mask][last]. Precompute all-pairs shortest paths first.

## 6. **Shortest Path in Grid with Obstacles (Leetcode 1293)**

**Instinct:**  
Grid me (0,0) se (m-1,n-1) tak shortest path with at most k obstacles removed. BFS/DP with state (i, j, k).

**Approach:**  
1. Memo[i][j][k] = min steps to reach (i,j) with k eliminations left
2. BFS style DP or DFS with memoization

```java
class Solution {
    int m, n;
    int[][] grid;
    int[][][] memo;
    int INF = Integer.MAX_VALUE / 2;
    int[][] dirs = {{0,1},{1,0},{0,-1},{-1,0}};
    
    public int shortestPath(int[][] grid, int k) {
        this.grid = grid;
        m = grid.length;
        n = grid[0].length;
        memo = new int[m][n][k + 1];
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        
        int ans = dfs(0, 0, k);
        return ans >= INF ? -1 : ans;
    }
    
    private int dfs(int i, int j, int k) {
        if (i == m - 1 && j == n - 1) return 0;
        if (memo[i][j][k] != -1) return memo[i][j][k];
        
        int minSteps = INF;
        int originalVal = grid[i][j];
        
        // Mark as visited to avoid cycles
        grid[i][j] = -1;
        
        for (int[] dir : dirs) {
            int ni = i + dir[0];
            int nj = j + dir[1];
            if (ni >= 0 && ni < m && nj >= 0 && nj < n && grid[ni][nj] != -1) {
                if (grid[ni][nj] == 1 && k > 0) {
                    minSteps = Math.min(minSteps, 1 + dfs(ni, nj, k - 1));
                } else if (grid[ni][nj] == 0) {
                    minSteps = Math.min(minSteps, 1 + dfs(ni, nj, k));
                }
            }
        }
        
        grid[i][j] = originalVal;
        memo[i][j][k] = minSteps;
        return minSteps;
    }
}
```

**Key Insight:** State includes remaining eliminations. Mark visited to avoid cycles.

## 7. **Maximum Probability Path (Leetcode 1514)**

**Instinct:**  
Graph me source se destination tak maximum probability path (product of probabilities). Dijkstra variant with max heap.

**Approach:**  
1. Max heap (priority queue) for probabilities
2. DP[node] = max probability to reach node
3. Relax edges with product

```java
class Solution {
    public double maxProbability(int n, int[][] edges, double[] succProb, int start, int end) {
        List<double[]>[] graph = new List[n];
        for (int i = 0; i < n; i++) graph[i] = new ArrayList<>();
        for (int i = 0; i < edges.length; i++) {
            graph[edges[i][0]].add(new double[]{edges[i][1], succProb[i]});
            graph[edges[i][1]].add(new double[]{edges[i][0], succProb[i]});
        }
        
        double[] prob = new double[n];
        prob[start] = 1.0;
        
        PriorityQueue<double[]> pq = new PriorityQueue<>((a, b) -> Double.compare(b[1], a[1]));
        pq.offer(new double[]{start, 1.0});
        
        while (!pq.isEmpty()) {
            double[] curr = pq.poll();
            int node = (int) curr[0];
            double currProb = curr[1];
            
            if (node == end) return currProb;
            if (currProb < prob[node]) continue;
            
            for (double[] next : graph[node]) {
                int nextNode = (int) next[0];
                double nextProb = currProb * next[1];
                if (nextProb > prob[nextNode]) {
                    prob[nextNode] = nextProb;
                    pq.offer(new double[]{nextNode, nextProb});
                }
            }
        }
        
        return prob[end];
    }
}
```

**Key Insight:** Dijkstra variant with max heap. Probability product instead of sum.

## 8. **Minimum Cost to Connect All Nodes (Kruskal/Prim with DP)**

**Instinct:**  
Graph me sabhi nodes ko connect karne ka minimum cost. MST problem - greedy works, but DP possible with bitmask.

**Approach:**  
1. DP[mask] = min cost to connect nodes in mask
2. Try adding a new node connected to any node in mask

```java
class Solution {
    int n;
    int[][] dist;
    int[] memo;
    int INF = Integer.MAX_VALUE / 2;
    
    public int minCostToConnect(int n, int[][] edges) {
        this.n = n;
        dist = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(dist[i], INF);
            dist[i][i] = 0;
        }
        for (int[] e : edges) {
            dist[e[0]][e[1]] = Math.min(dist[e[0]][e[1]], e[2]);
            dist[e[1]][e[0]] = Math.min(dist[e[1]][e[0]], e[2]);
        }
        
        // Floyd-Warshall for shortest paths
        for (int k = 0; k < n; k++) {
            for (int i = 0; i < n; i++) {
                for (int j = 0; j < n; j++) {
                    dist[i][j] = Math.min(dist[i][j], dist[i][k] + dist[k][j]);
                }
            }
        }
        
        memo = new int[1 << n];
        Arrays.fill(memo, -1);
        memo[0] = 0;
        
        return dfs(1 << 0);
    }
    
    private int dfs(int mask) {
        if (mask == (1 << n) - 1) return 0;
        if (memo[mask] != -1) return memo[mask];
        
        int minCost = INF;
        for (int i = 0; i < n; i++) {
            if ((mask & (1 << i)) != 0) {
                for (int j = 0; j < n; j++) {
                    if ((mask & (1 << j)) == 0) {
                        minCost = Math.min(minCost, dist[i][j] + dfs(mask | (1 << j)));
                    }
                }
            }
        }
        
        memo[mask] = minCost;
        return minCost;
    }
}
```

**Key Insight:** Bitmask DP for MST. Add one node at a time to connected component.

## 9. **Number of Ways to Arrive at Destination (Leetcode 1976)**

**Instinct:**  
Graph me shortest path ki count. Dijkstra + DP for count.

**Approach:**  
1. Dijkstra to find shortest distances
2. DP[node] = number of shortest paths to node
3. Relax edges in order of distance

```java
class Solution {
    int MOD = 1_000_000_007;
    
    public int countPaths(int n, int[][] roads) {
        List<int[]>[] graph = new List[n];
        for (int i = 0; i < n; i++) graph[i] = new ArrayList<>();
        for (int[] r : roads) {
            graph[r[0]].add(new int[]{r[1], r[2]});
            graph[r[1]].add(new int[]{r[0], r[2]});
        }
        
        long[] dist = new long[n];
        Arrays.fill(dist, Long.MAX_VALUE);
        dist[0] = 0;
        
        int[] ways = new int[n];
        ways[0] = 1;
        
        PriorityQueue<long[]> pq = new PriorityQueue<>((a, b) -> Long.compare(a[1], b[1]));
        pq.offer(new long[]{0, 0});
        
        while (!pq.isEmpty()) {
            long[] curr = pq.poll();
            int node = (int) curr[0];
            long currDist = curr[1];
            
            if (currDist > dist[node]) continue;
            
            for (int[] next : graph[node]) {
                int nextNode = next[0];
                long nextDist = currDist + next[1];
                
                if (nextDist < dist[nextNode]) {
                    dist[nextNode] = nextDist;
                    ways[nextNode] = ways[node];
                    pq.offer(new long[]{nextNode, nextDist});
                } else if (nextDist == dist[nextNode]) {
                    ways[nextNode] = (ways[nextNode] + ways[node]) % MOD;
                }
            }
        }
        
        return ways[n - 1];
    }
}
```

**Key Insight:** Dijkstra + DP. When distances equal, add ways.

## 10. **Shortest Path with Alternating Colors (Leetcode 1129)**

**Instinct:**  
Graph me red and blue edges hain. Shortest path with alternating colors. State = (node, lastColor).

**Approach:**  
1. BFS with state (node, color)
2. Track visited[node][color]

```java
class Solution {
    public int[] shortestAlternatingPaths(int n, int[][] redEdges, int[][] blueEdges) {
        List<Integer>[] red = new List[n];
        List<Integer>[] blue = new List[n];
        for (int i = 0; i < n; i++) {
            red[i] = new ArrayList<>();
            blue[i] = new ArrayList<>();
        }
        for (int[] e : redEdges) red[e[0]].add(e[1]);
        for (int[] e : blueEdges) blue[e[0]].add(e[1]);
        
        int[][] dist = new int[n][2]; // 0=red, 1=blue
        for (int i = 0; i < n; i++) Arrays.fill(dist[i], -1);
        dist[0][0] = dist[0][1] = 0;
        
        Queue<int[]> q = new LinkedList<>();
        q.offer(new int[]{0, 0}); // node, color (0=red, 1=blue)
        q.offer(new int[]{0, 1});
        
        while (!q.isEmpty()) {
            int[] curr = q.poll();
            int node = curr[0];
            int color = curr[1];
            
            List<Integer> neighbors = (color == 0) ? blue[node] : red[node];
            int nextColor = 1 - color;
            
            for (int next : neighbors) {
                if (dist[next][nextColor] == -1) {
                    dist[next][nextColor] = dist[node][color] + 1;
                    q.offer(new int[]{next, nextColor});
                }
            }
        }
        
        int[] ans = new int[n];
        for (int i = 0; i < n; i++) {
            ans[i] = Math.min(
                dist[i][0] == -1 ? Integer.MAX_VALUE : dist[i][0],
                dist[i][1] == -1 ? Integer.MAX_VALUE : dist[i][1]
            );
            if (ans[i] == Integer.MAX_VALUE) ans[i] = -1;
        }
        return ans;
    }
}
```

**Key Insight:** Color as state. BFS on (node, color).

## 11. **Minimum Cost to Make at Least One Valid Path (Leetcode 1368)**

**Instinct:**  
Grid me har cell me direction arrow hai. Change direction with cost 1. Min cost to reach bottom-right. 0-1 BFS.

**Approach:**  
1. 0-1 BFS (deque) for min cost
2. Cost 0 if moving in arrow direction, else cost 1

```java
class Solution {
    public int minCost(int[][] grid) {
        int m = grid.length, n = grid[0].length;
        int[][] dirs = {{0,1},{0,-1},{1,0},{-1,0}};
        int[][] cost = new int[m][n];
        for (int i = 0; i < m; i++) Arrays.fill(cost[i], Integer.MAX_VALUE);
        cost[0][0] = 0;
        
        Deque<int[]> dq = new ArrayDeque<>();
        dq.offer(new int[]{0, 0});
        
        while (!dq.isEmpty()) {
            int[] curr = dq.poll();
            int i = curr[0], j = curr[1];
            
            for (int d = 0; d < 4; d++) {
                int ni = i + dirs[d][0];
                int nj = j + dirs[d][1];
                if (ni >= 0 && ni < m && nj >= 0 && nj < n) {
                    int newCost = cost[i][j] + (grid[i][j] == d + 1 ? 0 : 1);
                    if (newCost < cost[ni][nj]) {
                        cost[ni][nj] = newCost;
                        if (grid[i][j] == d + 1) {
                            dq.offerFirst(new int[]{ni, nj});
                        } else {
                            dq.offerLast(new int[]{ni, nj});
                        }
                    }
                }
            }
        }
        
        return cost[m-1][n-1];
    }
}
```

**Key Insight:** 0-1 BFS. Cost 0 for same direction, cost 1 for change.

## 12. **Second Minimum Time to Reach Destination (Leetcode 2045)**

**Instinct:**  
Graph me second shortest path (strictly longer than shortest). BFS with state (node, parity).

**Approach:**  
1. BFS to find shortest and second shortest distances
2. Track two best distances per node

```java
class Solution {
    public int secondMinimum(int n, int[][] edges, int time, int change) {
        List<Integer>[] graph = new List[n + 1];
        for (int i = 1; i <= n; i++) graph[i] = new ArrayList<>();
        for (int[] e : edges) {
            graph[e[0]].add(e[1]);
            graph[e[1]].add(e[0]);
        }
        
        int[][] dist = new int[n + 1][2];
        for (int i = 1; i <= n; i++) Arrays.fill(dist[i], Integer.MAX_VALUE);
        dist[1][0] = 0;
        
        Queue<int[]> q = new LinkedList<>();
        q.offer(new int[]{1, 0});
        
        while (!q.isEmpty()) {
            int[] curr = q.poll();
            int node = curr[0];
            int d = curr[1];
            
            for (int next : graph[node]) {
                if (dist[next][0] > d + 1) {
                    dist[next][1] = dist[next][0];
                    dist[next][0] = d + 1;
                    q.offer(new int[]{next, d + 1});
                } else if (dist[next][0] < d + 1 && dist[next][1] > d + 1) {
                    dist[next][1] = d + 1;
                    q.offer(new int[]{next, d + 1});
                }
            }
        }
        
        int steps = dist[n][1];
        int totalTime = 0;
        for (int i = 0; i < steps; i++) {
            if ((totalTime / change) % 2 == 1) {
                totalTime = ((totalTime / change) + 1) * change;
            }
            totalTime += time;
        }
        return totalTime;
    }
}
```

**Key Insight:** Track two best distances per node. Traffic signal simulation.

class Solution {

## 14. **Maximum Number of Achievable Transfer Requests (Leetcode 1601)**

**Instinct:**  
Building me transfer requests. Maximum requests satisfy such that net change = 0. Bitmask DP.

**Approach:**  
1. Try all subsets of requests using bitmask
2. Check if valid (net balance zero)
3. Track max size

```java
class Solution {
    public int maximumRequests(int n, int[][] requests) {
        int m = requests.length;
        int max = 0;
        
        for (int mask = 0; mask < (1 << m); mask++) {
            int[] balance = new int[n];
            int count = 0;
            
            for (int i = 0; i < m; i++) {
                if ((mask & (1 << i)) != 0) {
                    balance[requests[i][0]]--;
                    balance[requests[i][1]]++;
                    count++;
                }
            }
            
            boolean valid = true;
            for (int b : balance) {
                if (b != 0) {
                    valid = false;
                    break;
                }
            }
            
            if (valid) {
                max = Math.max(max, count);
            }
        }
        return max;
    }
}
```

**Key Insight:** Bitmask over requests. Check balance after each subset.

## 15. **Minimum Cost to Reach Destination with Time Constraint (Leetcode 1928)**

**Instinct:**  
Graph me min cost to reach destination within maxTime. DP[time][node] = min cost.

**Approach:**  
1. DP[time][node] = min cost to reach node at exactly 'time'
2. Try all neighbors

```java
class Solution {
    public int minCost(int maxTime, int[][] edges, int[] passingFees) {
        int n = passingFees.length;
        int INF = Integer.MAX_VALUE / 2;
        int[][] dp = new int[maxTime + 1][n];
        for (int i = 0; i <= maxTime; i++) {
            Arrays.fill(dp[i], INF);
        }
        dp[0][0] = passingFees[0];
        
        for (int t = 0; t <= maxTime; t++) {
            for (int[] e : edges) {
                int u = e[0], v = e[1], time = e[2];
                if (t + time <= maxTime) {
                    dp[t + time][v] = Math.min(dp[t + time][v], dp[t][u] + passingFees[v]);
                    dp[t + time][u] = Math.min(dp[t + time][u], dp[t][v] + passingFees[u]);
                }
            }
        }
        
        int ans = INF;
        for (int t = 0; t <= maxTime; t++) {
            ans = Math.min(ans, dp[t][n - 1]);
        }
        return ans == INF ? -1 : ans;
    }
}
```

**Key Insight:** Time as dimension. Floyd-Warshall style relaxation.

## 16. **Path with Maximum Minimum Value (Leetcode 1102)**

**Instinct:**  
Path from (0,0) to (m-1,n-1) jisme minimum value along path maximize karna hai. Max-heap BFS.

**Approach:**  
1. Priority queue based on max min value
2. Track max min value to reach each cell

```java
class Solution {
    public int maximumMinimumPath(int[][] grid) {
        int m = grid.length, n = grid[0].length;
        int[][] dirs = {{0,1},{1,0},{0,-1},{-1,0}};
        int[][] maxMin = new int[m][n];
        for (int i = 0; i < m; i++) Arrays.fill(maxMin[i], -1);
        maxMin[0][0] = grid[0][0];
        
        PriorityQueue<int[]> pq = new PriorityQueue<>((a, b) -> b[2] - a[2]);
        pq.offer(new int[]{0, 0, grid[0][0]});
        
        while (!pq.isEmpty()) {
            int[] curr = pq.poll();
            int i = curr[0], j = curr[1], val = curr[2];
            
            if (i == m - 1 && j == n - 1) return val;
            if (val < maxMin[i][j]) continue;
            
            for (int[] dir : dirs) {
                int ni = i + dir[0], nj = j + dir[1];
                if (ni >= 0 && ni < m && nj >= 0 && nj < n) {
                    int newVal = Math.min(val, grid[ni][nj]);
                    if (newVal > maxMin[ni][nj]) {
                        maxMin[ni][nj] = newVal;
                        pq.offer(new int[]{ni, nj, newVal});
                    }
                }
            }
        }
        return maxMin[m-1][n-1];
    }
}
```

**Key Insight:** Maximize the minimum = max-heap BFS. Track best min value per cell.

## 17. **Minimum Time to Visit All Nodes (Leetcode 847)**

**Instinct:**  
Graph me sabhi nodes visit karne ka minimum time. Shortest path visiting all nodes = TSP.

**Approach:**  
1. BFS on state (mask, last)
2. DP[mask][last] = min steps

```java
class Solution {
    public int shortestPathLength(int[][] graph) {
        int n = graph.length;
        int[][] dist = new int[1 << n][n];
        for (int i = 0; i < (1 << n); i++) {
            Arrays.fill(dist[i], Integer.MAX_VALUE);
        }
        
        Queue<int[]> q = new LinkedList<>();
        for (int i = 0; i < n; i++) {
            dist[1 << i][i] = 0;
            q.offer(new int[]{1 << i, i});
        }
        
        while (!q.isEmpty()) {
            int[] curr = q.poll();
            int mask = curr[0], node = curr[1];
            
            if (mask == (1 << n) - 1) return dist[mask][node];
            
            for (int next : graph[node]) {
                int newMask = mask | (1 << next);
                if (dist[newMask][next] > dist[mask][node] + 1) {
                    dist[newMask][next] = dist[mask][node] + 1;
                    q.offer(new int[]{newMask, next});
                }
            }
        }
        return -1;
    }
}
```

**Key Insight:** BFS on (mask, node). Mask tracks visited nodes.

## 18. **Maximum Cost Path in Grid with K Moves**

**Instinct:**  
Grid me (0,0) se (m-1,n-1) tak exactly k moves me max cost. DP[moves][i][j].

**Approach:**  
1. DP[k][i][j] = max cost with exactly k moves
2. From all 4 directions

```java
class Solution {
    int[][] grid;
    int m, n;
    int[][][] memo;
    int[][] dirs = {{0,1},{1,0},{0,-1},{-1,0}};
    
    public int maxCostWithKMoves(int[][] grid, int k) {
        this.grid = grid;
        m = grid.length;
        n = grid[0].length;
        memo = new int[k + 1][m][n];
        for (int i = 0; i <= k; i++) {
            for (int j = 0; j < m; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        
        return dfs(0, 0, k);
    }
    
    private int dfs(int i, int j, int moves) {
        if (i == m - 1 && j == n - 1 && moves == 0) return grid[i][j];
        if (moves == 0) return Integer.MIN_VALUE / 2;
        if (memo[moves][i][j] != -1) return memo[moves][i][j];
        
        int max = Integer.MIN_VALUE / 2;
        for (int[] dir : dirs) {
            int ni = i + dir[0];
            int nj = j + dir[1];
            if (ni >= 0 && ni < m && nj >= 0 && nj < n) {
                max = Math.max(max, grid[i][j] + dfs(ni, nj, moves - 1));
            }
        }
        
        memo[moves][i][j] = max;
        return max;
    }
}
```

**Key Insight:** Moves as dimension. Exactly k moves to reach destination.

## 19. **Number of Ways to Reach Destination with Exactly K Edges**

**Instinct:**  
Graph me source se destination tak exactly k edges me pahunchne ke ways. DP[k][node] = ways.

**Approach:**  
1. DP[edges][node] = ways to reach node with exactly 'edges' edges
2. From all incoming edges

```java
class Solution {
    int MOD = 1_000_000_007;
    
    public int numWaysToReachWithKEdges(int n, int[][] edges, int src, int dst, int k) {
        List<Integer>[] graph = new List[n];
        List<Integer>[] reverse = new List[n];
        for (int i = 0; i < n; i++) {
            graph[i] = new ArrayList<>();
            reverse[i] = new ArrayList<>();
        }
        for (int[] e : edges) {
            graph[e[0]].add(e[1]);
            reverse[e[1]].add(e[0]);
        }
        
        int[][] dp = new int[k + 1][n];
        dp[0][src] = 1;
        
        for (int e = 1; e <= k; e++) {
            for (int node = 0; node < n; node++) {
                for (int prev : reverse[node]) {
                    dp[e][node] = (dp[e][node] + dp[e - 1][prev]) % MOD;
                }
            }
        }
        
        return dp[k][dst];
    }
}
```

**Key Insight:** DP[edges][node]. Build from source outward.

## 20. **Minimum Score of a Path Between Two Nodes (Leetcode 2492)**

**Instinct:**  
Graph me two nodes ke beech path ka minimum edge weight. Connected component ka minimum edge.

**Approach:**  
1. DFS to find connected component containing source and destination
2. Find minimum edge in that component

```java
class Solution {
    List<int[]>[] graph;
    boolean[] visited;
    int minEdge = Integer.MAX_VALUE;
    
    public int minScore(int n, int[][] roads) {
        graph = new List[n + 1];
        for (int i = 1; i <= n; i++) graph[i] = new ArrayList<>();
        for (int[] r : roads) {
            graph[r[0]].add(new int[]{r[1], r[2]});
            graph[r[1]].add(new int[]{r[0], r[2]});
        }
        
        visited = new boolean[n + 1];
        dfs(1);
        return minEdge;
    }
    
    private void dfs(int node) {
        visited[node] = true;
        for (int[] next : graph[node]) {
            minEdge = Math.min(minEdge, next[1]);
            if (!visited[next[0]]) {
                dfs(next[0]);
            }
        }
    }
}
```

**Key Insight:** Path can go anywhere in connected component. Answer = min edge in component.

## 🎯 Key Takeaways for Graph DP Pattern

### 1. **Core Graph DP Patterns:**

| Pattern | State | Transition |
|---------|-------|------------|
| Shortest Path | `dist[node]` | `relax(node, neighbor)` |
| K Stops | `dp[stops][node]` | `dp[stops][node] = min(dp[stops-1][prev] + cost)` |
| Bitmask (TSP) | `dp[mask][last]` | `dp[mask][last] = min(dp[mask ^ (1<<last)][prev] + dist[prev][last])` |
| Grid DP | `dp[i][j]` | `dp[i][j] = min(dp[i-1][j], dp[i][j-1]) + grid[i][j]` |

### 2. **When to use Graph DP:**
- Path finding with constraints (stops, time, cost)
- Visiting all nodes (TSP, Hamiltonian)
- Counting paths
- Probability maximization
- Grid traversal with obstacles

### 3. **Graph DP vs Standard Algorithms:**

| Problem | Standard | DP Alternative |
|---------|----------|----------------|
| Shortest Path | Dijkstra | DP only for DAG |
| MST | Kruskal/Prim | Bitmask DP for small n |
| TSP | NP-hard | DP[mask][last] |
| K Stops | Bellman-Ford | DP[stops][node] |

### 4. **Common State Dimensions:**

```
Shortest Path:     (node)
K Stops:          (stops, node)
TSP:              (mask, last)
Grid + K:         (i, j, k)
Time Constraint:  (time, node)
Color Alternating: (node, color)
```

### 5. **Graph DP Optimization Techniques:**

| Technique | Use Case |
|-----------|----------|
| Bitmask | TSP, Hamiltonian path |
| Floyd-Warshall | Precompute all-pairs shortest paths |
| 0-1 BFS | Costs 0 or 1 |
| Dijkstra | Non-negative weights |
| Bellman-Ford | K stops constraint |
| Topological order | DAG shortest/longest path |

## 📝 Key Formulas to Remember

| Operation | Formula |
|-----------|---------|
| Shortest Path (DAG) | `dp[node] = min(dp[node], weight + dp[prev])` |
| K Stops DP | `dp[k][v] = min(dp[k-1][u] + cost)` |
| TSP | `dp[mask][last] = min(dp[mask ^ (1<<last)][prev] + dist[prev][last])` |
| Probability Path | `prob[next] = max(prob[next], prob[curr] * edgeProb)` |
| Path Count | `ways[next] += ways[curr]` if distances equal |

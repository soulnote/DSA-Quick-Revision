
# Digit DP Pattern

## Digit DP ke 10 Main Variants:

| Variant | Description | Approach |
|---------|-------------|----------|
| **Count Numbers with Digit Constraints** | Count numbers with digit sum constraints | DP[pos][tight][started][sum] |
| **Numbers with Non-Decreasing Digits** | Digits are non-decreasing | DP[pos][tight][prev] |
| **Numbers with Digit Sum Constraints** | Sum of digits = target | DP[pos][tight][sum] |
| **Numbers Divisible by K** | Number divisible by given K | DP[pos][tight][rem] |
| **Count Numbers with No Consecutive Ones** | Binary strings without "11" | DP[pos][tight][prev] |
| **Numbers with Even/Odd Digit Count** | Even number of odd digits | DP[pos][tight][parity] |
| **Numbers with Rotated Digits** | Good numbers (rotated becomes different) | DP[pos][tight][rotated] |
| **Count Special Numbers** | Numbers with certain properties | DP[pos][tight][state] |
| **Numbers with Digit Product Constraints** | Product of digits = target | DP[pos][tight][product] |
| **K-th Number with Digit Constraints** | Find K-th number satisfying property | Binary search + DP |

## 1. **Count Numbers with Digit Sum (Leetcode 357 variant)**

**Instinct:**  
0 se n tak numbers count karni hai jinka digit sum <= target. Har digit place pe, tight condition track karo ki kya hum bound ke equal chal rahe hain.

**Approach:**  
1. Convert number to digit array
2. DP[pos][tight][sum] = count of valid numbers from current position
3. Base: pos == n → return 1 if sum <= target else 0
4. Limit = tight ? digits[pos] : 9
5. Try all digits 0 to limit

```java
class Solution {
    int target;
    int[] digits;
    int[][][] memo;
    
    public int countDigitSum(int n, int target) {
        this.target = target;
        String s = Integer.toString(n);
        digits = new int[s.length()];
        for (int i = 0; i < s.length(); i++) {
            digits[i] = s.charAt(i) - '0';
        }
        
        memo = new int[s.length()][2][target + 1];
        for (int i = 0; i < s.length(); i++) {
            for (int j = 0; j < 2; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        
        return dfs(0, 1, 0);
    }
    
    private int dfs(int pos, int tight, int sum) {
        if (sum > target) return 0;
        if (pos == digits.length) return 1;
        if (memo[pos][tight][sum] != -1) return memo[pos][tight][sum];
        
        int limit = tight == 1 ? digits[pos] : 9;
        int count = 0;
        
        for (int dig = 0; dig <= limit; dig++) {
            count += dfs(pos + 1, (tight == 1 && dig == limit) ? 1 : 0, sum + dig);
        }
        
        memo[pos][tight][sum] = count;
        return count;
    }
}
```

**Key Insight:** Digit DP core = pos, tight, and property state (sum). Tight = 1 means bound me hai.

## 2. **Numbers with Non-Decreasing Digits (Leetcode 300 variant)**

**Instinct:**  
Digits non-decreasing hone chahiye (e.g., 123, 111, 135). State: previous digit.

**Approach:**  
1. DP[pos][tight][prev] = count
2. Try digits from prev to limit
3. Base: pos == n → 1

```java
class Solution {
    int[] digits;
    int[][][] memo;
    
    public int countNonDecreasing(int n) {
        String s = Integer.toString(n);
        digits = new int[s.length()];
        for (int i = 0; i < s.length(); i++) {
            digits[i] = s.charAt(i) - '0';
        }
        
        memo = new int[s.length()][2][10];
        for (int i = 0; i < s.length(); i++) {
            for (int j = 0; j < 2; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        
        return dfs(0, 1, 0, true);
    }
    
    private int dfs(int pos, int tight, int prev, boolean started) {
        if (pos == digits.length) return started ? 1 : 0;
        if (memo[pos][tight][prev] != -1) return memo[pos][tight][prev];
        
        int limit = tight == 1 ? digits[pos] : 9;
        int count = 0;
        
        for (int dig = 0; dig <= limit; dig++) {
            if (!started) {
                count += dfs(pos + 1, (tight == 1 && dig == limit) ? 1 : 0, dig, dig != 0);
            } else if (dig >= prev) {
                count += dfs(pos + 1, (tight == 1 && dig == limit) ? 1 : 0, dig, true);
            }
        }
        
        memo[pos][tight][prev] = count;
        return count;
    }
}
```

**Key Insight:** Non-decreasing = next digit >= previous digit. Started flag handles leading zeros.

## 3. **Numbers Divisible by K (Leetcode 1012 variant)**

**Instinct:**  
0 se n tak numbers count karni hai jo divisible by k ho. State: remainder.

**Approach:**  
1. DP[pos][tight][rem] = count
2. At each step, newRem = (rem * 10 + dig) % k
3. Base: pos == n → return 1 if rem == 0 else 0

```java
class Solution {
    int[] digits;
    int k;
    int[][][] memo;
    
    public int countDivisibleByK(int n, int k) {
        this.k = k;
        String s = Integer.toString(n);
        digits = new int[s.length()];
        for (int i = 0; i < s.length(); i++) {
            digits[i] = s.charAt(i) - '0';
        }
        
        memo = new int[s.length()][2][k];
        for (int i = 0; i < s.length(); i++) {
            for (int j = 0; j < 2; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        
        return dfs(0, 1, 0, true);
    }
    
    private int dfs(int pos, int tight, int rem, boolean started) {
        if (pos == digits.length) {
            return (started && rem == 0) ? 1 : 0;
        }
        if (memo[pos][tight][rem] != -1) return memo[pos][tight][rem];
        
        int limit = tight == 1 ? digits[pos] : 9;
        int count = 0;
        
        for (int dig = 0; dig <= limit; dig++) {
            int newRem = (rem * 10 + dig) % k;
            if (!started && dig == 0) {
                count += dfs(pos + 1, (tight == 1 && dig == limit) ? 1 : 0, 0, false);
            } else {
                count += dfs(pos + 1, (tight == 1 && dig == limit) ? 1 : 0, newRem, true);
            }
        }
        
        memo[pos][tight][rem] = count;
        return count;
    }
}
```

**Key Insight:** Remainder update: `newRem = (rem * 10 + dig) % k`. Started flag for leading zeros.

## 4. **Count Numbers with No Consecutive Ones (Leetcode 600)**

**Instinct:**  
Binary numbers from 1 to n with no consecutive 1s. Digit DP with base 2.

**Approach:**  
1. Convert n to binary string
2. DP[pos][tight][prev] = count (prev = 0 or 1)
3. Cannot place 1 if prev = 1

```java
class Solution {
    char[] bits;
    int[][][] memo;
    
    public int findIntegers(int n) {
        String binary = Integer.toBinaryString(n);
        bits = binary.toCharArray();
        memo = new int[bits.length][2][2];
        for (int i = 0; i < bits.length; i++) {
            for (int j = 0; j < 2; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        
        return dfs(0, 1, 0, true);
    }
    
    private int dfs(int pos, int tight, int prev, boolean started) {
        if (pos == bits.length) return started ? 1 : 0;
        if (memo[pos][tight][prev] != -1) return memo[pos][tight][prev];
        
        int limit = tight == 1 ? bits[pos] - '0' : 1;
        int count = 0;
        
        for (int dig = 0; dig <= limit; dig++) {
            if (prev == 1 && dig == 1) continue;
            if (!started && dig == 0) {
                count += dfs(pos + 1, (tight == 1 && dig == limit) ? 1 : 0, 0, false);
            } else {
                count += dfs(pos + 1, (tight == 1 && dig == limit) ? 1 : 0, dig, true);
            }
        }
        
        memo[pos][tight][prev] = count;
        return count;
    }
}
```

**Key Insight:** Binary digit DP. Cannot have '11' in the number.

## 5. **Count Numbers with Even Number of Odd Digits**

**Instinct:**  
Numbers jisme odd digits even count ho. State: parity (0 = even, 1 = odd).

**Approach:**  
1. DP[pos][tight][parity] = count
2. If digit is odd, flip parity
3. Base: pos == n → return 1 if parity == 0 else 0

```java
class Solution {
    int[] digits;
    int[][][] memo;
    
    public int countEvenOddDigits(int n) {
        String s = Integer.toString(n);
        digits = new int[s.length()];
        for (int i = 0; i < s.length(); i++) {
            digits[i] = s.charAt(i) - '0';
        }
        
        memo = new int[s.length()][2][2];
        for (int i = 0; i < s.length(); i++) {
            for (int j = 0; j < 2; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        
        return dfs(0, 1, 0, true);
    }
    
    private int dfs(int pos, int tight, int parity, boolean started) {
        if (pos == digits.length) {
            return (started && parity == 0) ? 1 : 0;
        }
        if (memo[pos][tight][parity] != -1) return memo[pos][tight][parity];
        
        int limit = tight == 1 ? digits[pos] : 9;
        int count = 0;
        
        for (int dig = 0; dig <= limit; dig++) {
            int newParity = parity;
            if (dig % 2 == 1) newParity ^= 1;
            
            if (!started && dig == 0) {
                count += dfs(pos + 1, (tight == 1 && dig == limit) ? 1 : 0, 0, false);
            } else {
                count += dfs(pos + 1, (tight == 1 && dig == limit) ? 1 : 0, newParity, true);
            }
        }
        
        memo[pos][tight][parity] = count;
        return count;
    }
}
```

**Key Insight:** Parity toggles for odd digits. Final answer when parity = 0 (even count).

## 6. **Numbers with Rotated Digits (Leetcode 788)**

**Instinct:**  
Good numbers: rotate 180 degrees becomes different number. Valid digits: 0,1,8 rotate to themselves; 2->5, 5->2, 6->9, 9->6. Invalid: 3,4,7.

**Approach:**  
1. State: 0 = no change yet, 1 = changed, 2 = invalid
2. Track if number has any changing digit

```java
class Solution {
    int[] digits;
    int[][][] memo;
    
    public int rotatedDigits(int n) {
        String s = Integer.toString(n);
        digits = new int[s.length()];
        for (int i = 0; i < s.length(); i++) {
            digits[i] = s.charAt(i) - '0';
        }
        
        memo = new int[s.length()][2][3];
        for (int i = 0; i < s.length(); i++) {
            for (int j = 0; j < 2; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        
        return dfs(0, 1, 0, true);
    }
    
    private int dfs(int pos, int tight, int state, boolean started) {
        if (pos == digits.length) {
            return (started && state == 1) ? 1 : 0;
        }
        if (memo[pos][tight][state] != -1) return memo[pos][tight][state];
        
        int limit = tight == 1 ? digits[pos] : 9;
        int count = 0;
        
        for (int dig = 0; dig <= limit; dig++) {
            int newState = state;
            if (dig == 3 || dig == 4 || dig == 7) {
                newState = 2; // invalid
            } else if (dig == 2 || dig == 5 || dig == 6 || dig == 9) {
                if (newState != 2) newState = 1; // changed
            }
            // 0,1,8: state remains same
            
            if (!started && dig == 0) {
                count += dfs(pos + 1, (tight == 1 && dig == limit) ? 1 : 0, 0, false);
            } else if (newState != 2) {
                count += dfs(pos + 1, (tight == 1 && dig == limit) ? 1 : 0, newState, true);
            }
        }
        
        memo[pos][tight][state] = count;
        return count;
    }
}
```

**Key Insight:** Three states: 0=no change, 1=changed, 2=invalid. At least one changing digit needed.

## 7. **K-th Number with Digit Sum Constraint**

**Instinct:**  
Find K-th number (starting from 1) whose digit sum <= target. Binary search + count function.

**Approach:**  
1. Binary search on answer
2. Count numbers <= mid with digit sum <= target
3. Find smallest x such that count >= K

```java
class Solution {
    int target;
    
    public int findKthNumberWithDigitSum(int k, int target) {
        this.target = target;
        int left = 1, right = Integer.MAX_VALUE / 2;
        int ans = -1;
        
        while (left <= right) {
            int mid = left + (right - left) / 2;
            if (count(mid) >= k) {
                ans = mid;
                right = mid - 1;
            } else {
                left = mid + 1;
            }
        }
        return ans;
    }
    
    private int count(int n) {
        String s = Integer.toString(n);
        int[] digits = new int[s.length()];
        for (int i = 0; i < s.length(); i++) {
            digits[i] = s.charAt(i) - '0';
        }
        
        int[][][] memo = new int[s.length()][2][target + 1];
        for (int i = 0; i < s.length(); i++) {
            for (int j = 0; j < 2; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        
        return dfs(digits, 0, 1, 0, memo);
    }
    
    private int dfs(int[] digits, int pos, int tight, int sum, int[][][] memo) {
        if (sum > target) return 0;
        if (pos == digits.length) return 1;
        if (memo[pos][tight][sum] != -1) return memo[pos][tight][sum];
        
        int limit = tight == 1 ? digits[pos] : 9;
        int count = 0;
        
        for (int dig = 0; dig <= limit; dig++) {
            count += dfs(digits, pos + 1, (tight == 1 && dig == limit) ? 1 : 0, sum + dig, memo);
        }
        
        memo[pos][tight][sum] = count;
        return count;
    }
}
```

**Key Insight:** Binary search + digit DP count function. O(log N × len × target × 2).

## 8. **Count Numbers with Unique Digits (Leetcode 357)**

**Instinct:**  
Numbers with all digits distinct. State: bitmask of used digits.

**Approach:**  
1. DP[pos][tight][mask] = count
2. Track which digits have been used
3. Cannot use digit if already used

```java
class Solution {
    public int countNumbersWithUniqueDigits(int n) {
        if (n == 0) return 1;
        
        int[] digits = new int[10];
        for (int i = 0; i < 10; i++) {
            digits[i] = i;
        }
        
        int[][][] memo = new int[10][2][1 << 10];
        for (int i = 0; i < 10; i++) {
            for (int j = 0; j < 2; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        
        return dfs(digits, 0, 1, 0, true, n, memo) + 1; // +1 for 0
    }
    
    private int dfs(int[] digits, int pos, int tight, int mask, boolean started, int n, int[][][] memo) {
        if (pos == n) return started ? 1 : 0;
        if (memo[pos][tight][mask] != -1) return memo[pos][tight][mask];
        
        int limit = tight == 1 ? digits[pos] : 9;
        int count = 0;
        
        for (int dig = 0; dig <= limit; dig++) {
            if (!started && dig == 0) {
                count += dfs(digits, pos + 1, (tight == 1 && dig == limit) ? 1 : 0, mask, false, n, memo);
            } else if ((mask & (1 << dig)) == 0) {
                count += dfs(digits, pos + 1, (tight == 1 && dig == limit) ? 1 : 0, mask | (1 << dig), true, n, memo);
            }
        }
        
        memo[pos][tight][mask] = count;
        return count;
    }
}
```

**Key Insight:** Bitmask for used digits. Cannot reuse digit if already used.

## 9. **Count Numbers with Digit Product = K**

**Instinct:**  
Numbers jinka digit product exactly K hai. State: product (but product can be large, so use map).

**Approach:**  
1. DP[pos][tight][product] using map for product
2. newProduct = product * dig
3. Base: pos == n → return 1 if product == K else 0

```java
class Solution {
    int[] digits;
    int K;
    Map<String, Integer> memo = new HashMap<>();
    
    public int countDigitProduct(int n, int k) {
        this.K = k;
        String s = Integer.toString(n);
        digits = new int[s.length()];
        for (int i = 0; i < s.length(); i++) {
            digits[i] = s.charAt(i) - '0';
        }
        
        return dfs(0, 1, 1, true);
    }
    
    private int dfs(int pos, int tight, int product, boolean started) {
        if (product > K && K != 0) return 0;
        if (pos == digits.length) {
            return (started && product == K) ? 1 : 0;
        }
        
        String key = pos + "|" + tight + "|" + product + "|" + started;
        if (memo.containsKey(key)) return memo.get(key);
        
        int limit = tight == 1 ? digits[pos] : 9;
        int count = 0;
        
        for (int dig = 0; dig <= limit; dig++) {
            if (!started && dig == 0) {
                count += dfs(pos + 1, (tight == 1 && dig == limit) ? 1 : 0, 1, false);
            } else {
                count += dfs(pos + 1, (tight == 1 && dig == limit) ? 1 : 0, product * dig, true);
            }
        }
        
        memo.put(key, count);
        return count;
    }
}
```

**Key Insight:** Product as state. Product can grow large, so prune when > K.

## 10. **Count Numbers with At Least K Repeating Digits**

**Instinct:**  
Numbers jisme koi digit at least K times repeat ho. Complement approach: count with all digits at most K-1 times.

**Approach:**  
1. Count numbers with each digit frequency <= K-1
2. Subtract from total

```java
class Solution {
    int[] digits;
    int K;
    int[][][] memo;
    
    public int countAtLeastKRepeating(int n, int k) {
        this.K = k;
        String s = Integer.toString(n);
        digits = new int[s.length()];
        for (int i = 0; i < s.length(); i++) {
            digits[i] = s.charAt(i) - '0';
        }
        
        // Count numbers with each digit frequency < K
        memo = new int[s.length()][2][1 << 20]; // too large, use map instead
        // Simplified: use frequency array as state
        return n + 1 - countWithMaxFreqLessThanK();
    }
    
    private int countWithMaxFreqLessThanK() {
        // Implementation with frequency array state
        // For simplicity, using map
        Map<String, Integer> dp = new HashMap<>();
        return dfsWithFreq(0, 1, new int[10], true, dp);
    }
    
    private int dfsWithFreq(int pos, int tight, int[] freq, boolean started, Map<String, Integer> dp) {
        if (pos == digits.length) return started ? 1 : 0;
        
        String key = pos + "|" + tight + "|" + Arrays.toString(freq) + "|" + started;
        if (dp.containsKey(key)) return dp.get(key);
        
        int limit = tight == 1 ? digits[pos] : 9;
        int count = 0;
        
        for (int dig = 0; dig <= limit; dig++) {
            if (!started && dig == 0) {
                count += dfsWithFreq(pos + 1, (tight == 1 && dig == limit) ? 1 : 0, freq, false, dp);
            } else {
                freq[dig]++;
                if (freq[dig] < K) {
                    count += dfsWithFreq(pos + 1, (tight == 1 && dig == limit) ? 1 : 0, freq, true, dp);
                }
                freq[dig]--;
            }
        }
        
        dp.put(key, count);
        return count;
    }
}
```

**Key Insight:** Complement counting. Track frequency array as state.

## 11. **Count Numbers with Digit Sum Between A and B**

**Instinct:**  
Numbers jinka digit sum [A, B] range me ho. Use prefix sum: count(B) - count(A-1).

**Approach:**  
1. Function count(n, target) returns numbers with digit sum <= target
2. Answer = count(n, B) - count(n, A-1)

```java
class Solution {
    public int countDigitSumInRange(int n, int A, int B) {
        return countLessThanEqual(n, B) - countLessThanEqual(n, A - 1);
    }
    
    private int countLessThanEqual(int n, int target) {
        if (target < 0) return 0;
        String s = Integer.toString(n);
        int[] digits = new int[s.length()];
        for (int i = 0; i < s.length(); i++) {
            digits[i] = s.charAt(i) - '0';
        }
        
        int[][][] memo = new int[s.length()][2][target + 1];
        for (int i = 0; i < s.length(); i++) {
            for (int j = 0; j < 2; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        
        return dfs(digits, 0, 1, 0, target, memo);
    }
    
    private int dfs(int[] digits, int pos, int tight, int sum, int target, int[][][] memo) {
        if (sum > target) return 0;
        if (pos == digits.length) return 1;
        if (memo[pos][tight][sum] != -1) return memo[pos][tight][sum];
        
        int limit = tight == 1 ? digits[pos] : 9;
        int count = 0;
        
        for (int dig = 0; dig <= limit; dig++) {
            count += dfs(digits, pos + 1, (tight == 1 && dig == limit) ? 1 : 0, sum + dig, target, memo);
        }
        
        memo[pos][tight][sum] = count;
        return count;
    }
}
```

**Key Insight:** Range query using prefix sums. Count(n, B) - count(n, A-1).

## 13. **Numbers with All Digits Even**

**Instinct:**  
Numbers jisme saare digits even ho. Digits: 0,2,4,6,8.

**Approach:**  
1. DP[pos][tight][started] = count
2. Only even digits allowed

```java
class Solution {
    int[] digits;
    int[][][] memo;
    int[] evenDigits = {0, 2, 4, 6, 8};
    
    public int countAllEvenDigits(int n) {
        String s = Integer.toString(n);
        digits = new int[s.length()];
        for (int i = 0; i < s.length(); i++) {
            digits[i] = s.charAt(i) - '0';
        }
        
        memo = new int[s.length()][2][2];
        for (int i = 0; i < s.length(); i++) {
            for (int j = 0; j < 2; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        
        return dfs(0, 1, false);
    }
    
    private int dfs(int pos, int tight, boolean started) {
        if (pos == digits.length) return started ? 1 : 0;
        if (memo[pos][tight][started ? 1 : 0] != -1) return memo[pos][tight][started ? 1 : 0];
        
        int limit = tight == 1 ? digits[pos] : 9;
        int count = 0;
        
        for (int dig : evenDigits) {
            if (dig > limit) break;
            if (!started && dig == 0) {
                count += dfs(pos + 1, (tight == 1 && dig == limit) ? 1 : 0, false);
            } else {
                count += dfs(pos + 1, (tight == 1 && dig == limit) ? 1 : 0, true);
            }
        }
        
        memo[pos][tight][started ? 1 : 0] = count;

    }
}
```

**Key Insight:** Restrict digits to even set only. Simple constraint.

## 14. **Numbers with Digit Sum Divisible by K**

**Instinct:**  
Numbers jinka digit sum divisible by K. State: remainder of digit sum modulo K.

**Approach:**  
1. DP[pos][tight][rem] = count
2. newRem = (rem + dig) % k
3. Base: pos == n → return 1 if rem == 0 else 0

```java
class Solution {
    int[] digits;
    int k;
    int[][][] memo;
    
    public int countDigitSumDivisibleByK(int n, int k) {
        this.k = k;
        String s = Integer.toString(n);
        digits = new int[s.length()];
        for (int i = 0; i < s.length(); i++) {
            digits[i] = s.charAt(i) - '0';
        }
        
        memo = new int[s.length()][2][k];
        for (int i = 0; i < s.length(); i++) {
            for (int j = 0; j < 2; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        
        return dfs(0, 1, 0, true);
    }
    
    private int dfs(int pos, int tight, int rem, boolean started) {
        if (pos == digits.length) {
            return (started && rem == 0) ? 1 : 0;
        }
        if (memo[pos][tight][rem] != -1) return memo[pos][tight][rem];
        
        int limit = tight == 1 ? digits[pos] : 9;
        int count = 0;
        
        for (int dig = 0; dig <= limit; dig++) {
            int newRem = (rem + dig) % k;
            if (!started && dig == 0) {
                count += dfs(pos + 1, (tight == 1 && dig == limit) ? 1 : 0, 0, false);
            } else {
                count += dfs(pos + 1, (tight == 1 && dig == limit) ? 1 : 0, newRem, true);
            }
        }
        
        memo[pos][tight][rem] = count;
        return count;
    }
}
```

**Key Insight:** Digit sum remainder modulo K. Classic digit DP with sum remainder.

## 15. **Numbers with All Digits Distinct (Leetcode 357 - Another approach)**

**Instinct:**  
Numbers with all digits distinct. State: bitmask of used digits.

**Approach:**  
1. DP[pos][tight][mask] = count
2. Track used digits via bitmask

```java
class Solution {
    public int countNumbersWithUniqueDigits(int n) {
        if (n == 0) return 1;
        int[][][] memo = new int[n + 1][2][1 << 10];
        for (int i = 0; i <= n; i++) {
            for (int j = 0; j < 2; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        return dfs(0, 1, 0, true, n, memo);
    }
    
    private int dfs(int pos, int tight, int mask, boolean started, int n, int[][][] memo) {
        if (pos == n) return started ? 1 : 0;
        if (memo[pos][tight][mask] != -1) return memo[pos][tight][mask];
        
        int limit = tight == 1 ? 9 : 9;
        // For first position, limit might be different, but simplified
        int count = 0;
        
        for (int dig = 0; dig <= limit; dig++) {
            if (!started && dig == 0) {
                count += dfs(pos + 1, 0, mask, false, n, memo);
            } else if ((mask & (1 << dig)) == 0) {
                count += dfs(pos + 1, 0, mask | (1 << dig), true, n, memo);
            }
        }
        
        memo[pos][tight][mask] = count;
        return count;
    }
}
```

**Key Insight:** Bitmask tracks used digits. n-digit numbers only.

## 16. **Numbers with Increasing Digits (Strictly Increasing)**

**Instinct:**  
Digits strictly increasing (e.g., 123, 467). State: previous digit.

**Approach:**  
1. DP[pos][tight][prev] = count
2. Next digit > previous digit
3. No zero at start

```java
class Solution {
    int[] digits;
    int[][][] memo;
    
    public int countStrictlyIncreasing(int n) {
        String s = Integer.toString(n);
        digits = new int[s.length()];
        for (int i = 0; i < s.length(); i++) {
            digits[i] = s.charAt(i) - '0';
        }
        
        memo = new int[s.length()][2][10];
        for (int i = 0; i < s.length(); i++) {
            for (int j = 0; j < 2; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        
        return dfs(0, 1, 0, false);
    }
    
    private int dfs(int pos, int tight, int prev, boolean started) {
        if (pos == digits.length) return started ? 1 : 0;
        if (memo[pos][tight][prev] != -1) return memo[pos][tight][prev];
        
        int limit = tight == 1 ? digits[pos] : 9;
        int count = 0;
        
        for (int dig = (started ? prev + 1 : 1); dig <= limit; dig++) {

    }
```

**Key Insight:** Strictly increasing = next digit > previous. Start from digit 1.

## 17. **Count Numbers with Alternate Even/Odd Digits**

**Instinct:**  
Digits alternate between even and odd. State: previous digit parity.

**Approach:**  
1. DP[pos][tight][prevParity] = count
2. Current digit parity must be opposite of previous

```java
class Solution {
    int[] digits;
    int[][][] memo;
    
    public int countAlternateParity(int n) {
        String s = Integer.toString(n);
        digits = new int[s.length()];
        for (int i = 0; i < s.length(); i++) {
            digits[i] = s.charAt(i) - '0';
        }
        
        memo = new int[s.length()][2][3]; // 0=even, 1=odd, 2=start
        for (int i = 0; i < s.length(); i++) {
            for (int j = 0; j < 2; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        
        return dfs(0, 1, 2, false);
    }
    
    private int dfs(int pos, int tight, int prevParity, boolean started) {
        if (pos == digits.length) return started ? 1 : 0;
        if (memo[pos][tight][prevParity] != -1) return memo[pos][tight][prevParity];
        
        int limit = tight == 1 ? digits[pos] : 9;
        int count = 0;
        
        for (int dig = 0; dig <= limit; dig++) {
            int currParity = dig % 2;
            if (!started) {
                if (dig == 0) {
                    count += dfs(pos + 1, (tight == 1 && dig == limit) ? 1 : 0, 2, false);
                } else {
                    count += dfs(pos + 1, (tight == 1 && dig == limit) ? 1 : 0, currParity, true);
                }
            } else if (currParity != prevParity) {
                count += dfs(pos + 1, (tight == 1 && dig == limit) ? 1 : 0, currParity, true);
            }
        }
        
        memo[pos][tight][prevParity] = count;
        return count;
    }
}
```

**Key Insight:** Track parity of previous digit. Current parity must be different.

## 18. **Numbers that are Palindromes (Leetcode 906 variant)**

**Instinct:**  
Count palindromic numbers up to n. Construct palindromes instead of digit DP.

**Approach:**  
1. Generate palindromes by taking half and mirroring
2. Count how many <= n

```java
class Solution {
    public int countPalindromicNumbers(int n) {
        if (n < 0) return 0;
        if (n < 10) return n + 1;
        
        String s = Integer.toString(n);
        int len = s.length();
        int count = 9; // 1-9 single digit palindromes
        
        // Count palindromes with length < len
        for (int l = 2; l < len; l++) {
            count += 9 * (int) Math.pow(10, (l - 1) / 2);
        }
        
        // Count palindromes with length = len
        String half = s.substring(0, (len + 1) / 2);
        long halfNum = Long.parseLong(half);
        
        for (long i = (len == 1 ? 1 : (long) Math.pow(10, (len - 1) / 2)); i <= halfNum; i++) {
            String left = Long.toString(i);
            String right = new StringBuilder(left).reverse().toString();
            String palindrome = left + right.substring(len % 2);
            if (Long.parseLong(palindrome) <= n) {
                count++;
            }
        }
        
        return count;
    }
}
```

**Key Insight:** Generate palindromes by constructing from half. Not standard digit DP.

Numbers jisme adjacent digits same nahi hai. State: previous digit.

1. DP[pos][tight][prev] = count

## 20. **K-th Number with Digit Sum Divisible by K**

**Instinct:**  
Find K-th number (from 1) jiska digit sum divisible by K. Binary search + DP.

**Approach:**  
1. Binary search on answer
2. Count function returns numbers <= mid with digit sum % K == 0
3. Find smallest x with count >= K

```java
class Solution {
    int k;
    
    public int findKthDigitSumDivisible(int k, int K) {
        this.k = K;
        int left = 1, right = Integer.MAX_VALUE / 2;
        int ans = -1;
        
        while (left <= right) {
            int mid = left + (right - left) / 2;
            if (count(mid) >= k) {
                ans = mid;
                right = mid - 1;
            } else {
                left = mid + 1;
            }
        }
        return ans;
    }
    
    private int count(int n) {
        String s = Integer.toString(n);
        int[] digits = new int[s.length()];
        for (int i = 0; i < s.length(); i++) {
            digits[i] = s.charAt(i) - '0';
        }
        
        int[][][] memo = new int[s.length()][2][k];
        for (int i = 0; i < s.length(); i++) {
            for (int j = 0; j < 2; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        
        return dfs(digits, 0, 1, 0, true, memo);
    }
    
    private int dfs(int[] digits, int pos, int tight, int rem, boolean started, int[][][] memo) {
        if (pos == digits.length) {
            return (started && rem == 0) ? 1 : 0;
        }
        if (memo[pos][tight][rem] != -1) return memo[pos][tight][rem];
        
        int limit = tight == 1 ? digits[pos] : 9;
        int count = 0;
        
        for (int dig = 0; dig <= limit; dig++) {
            int newRem = (rem + dig) % k;
            if (!started && dig == 0) {
                count += dfs(digits, pos + 1, (tight == 1 && dig == limit) ? 1 : 0, 0, false, memo);
            } else {
                count += dfs(digits, pos + 1, (tight == 1 && dig == limit) ? 1 : 0, newRem, true, memo);
            }
        }
        
        memo[pos][tight][rem] = count;
        return count;
    }
}
```

**Key Insight:** Binary search + digit DP count. Find K-th number.

## 🎯 Key Takeaways for Digit DP Pattern

### 1. **Core Digit DP State:**
```java
dfs(pos, tight, started, ...additional state)
```
- `pos`: current digit position (0 to len-1)
- `tight`: 1 if prefix equals bound, 0 if already smaller
- `started`: 1 if number has started (non-leading zeros)
- `...`: additional property (sum, rem, mask, prev, etc.)

### 2. **Standard Digit DP Template:**
```java
int dfs(int pos, int tight, int started, int state) {
    if (pos == n) return started ? baseCase : 0;
    if (memo[pos][tight][started][state] != -1) return memo[...];
    
    int limit = tight == 1 ? digits[pos] : 9;
    int count = 0;
    
    for (int dig = 0; dig <= limit; dig++) {
        if (!started && dig == 0) {
            count += dfs(pos+1, (tight==1 && dig==limit)?1:0, 0, initialState);
        } else {
            int newState = update(state, dig);
            count += dfs(pos+1, (tight==1 && dig==limit)?1:0, 1, newState);
        }
    }
    
    memo[pos][tight][started][state] = count;
    return count;
}
```

### 3. **Common State Updates:**

| Constraint | State Update |
|------------|--------------|
| Digit Sum | `newSum = sum + dig` |
| Remainder | `newRem = (rem * 10 + dig) % k` |
| Non-decreasing | `newPrev = dig` (must be >= prev) |
| Unique Digits | `newMask = mask | (1 << dig)` |
| Parity | `newParity = parity ^ (dig % 2)` |

### 4. **When to use Digit DP:**
- Counting numbers in range [0, n] with digit constraints
- Digit sum, product, or remainder constraints
- Digit order constraints (increasing, non-decreasing)
- Digit frequency constraints
- Finding K-th number satisfying digit properties

### 5. **Digit DP Optimization Techniques:**

| Technique | Use Case |
|-----------|----------|
| Memoization | All digit DP problems |
| State compression | Pack multiple states into one integer |
| Pruning | Skip when sum > target, product > target |
| Binary search | Finding K-th element |
| Prefix sums | Range queries [L, R] = f(R) - f(L-1) |

### 6. **Common State Sizes:**

| Problem | State Dimensions | Size |
|---------|-----------------|------|
| Digit Sum | len × 2 × (9*len) | ~10 × 2 × 100 = 2000 |
| Remainder | len × 2 × k | ~10 × 2 × 100 = 2000 |
| Bitmask | len × 2 × 2^10 | ~10 × 2 × 1024 = 20k |
| Prev Digit | len × 2 × 10 | ~10 × 2 × 10 = 200 |

### 7. **Digit DP Complexity:**
- Time: O(len × 2 × state × 10)
- Usually len ≤ 10 (for 32-bit integers) or len ≤ 18 (for 64-bit)
- Very efficient for 10^9 range


# 🔤 Longest Common Subsequence (LCS) Pattern 

## LCS ke 10 Main Variants:

| Variant | Description | Approach |
|---------|-------------|----------|
| **Classic LCS** | Longest common subsequence | Match/mismatch on characters |
| **Shortest Common Supersequence (SCS)** | Shortest string containing both | LCS length = m+n - LCS |
| **Minimum Insertions/Deletions** | Convert string A to B | m + n - 2*LCS |
| **Longest Palindromic Subsequence (LPS)** | Longest palindrome in string | LCS of string and its reverse |
| **Longest Repeating Subsequence** | Same string, indices different | LCS with i != j |
| **Edit Distance (Levenshtein)** | Insert/delete/replace operations | DP with 3 operations |
| **Distinct Subsequences** | Count of string t in s as subsequence | Take/skip with counting |
| **Interleaving Strings** | Check if s3 is interleaving of s1,s2 | 2D DP on indices |
| **Wildcard Matching** | Pattern with * and ? | DP on pattern and string |
| **Regular Expression Matching** | Pattern with . and * | DP with char matching |

## 1. **Longest Common Subsequence (Leetcode 1143)**

**Instinct:**  
Do strings di hain, longest common subsequence nikaalni hai (non-contiguous, order same). Har char match ho ya na ho. Memoization me **two indices** track karo - i for text1, j for text2.

**Approach:**  
1. Memo[i][j] = LCS length for text1[i..] and text2[j..]
2. Base: i == n ya j == m → 0
3. If chars match → 1 + dfs(i+1, j+1)
4. Else → max(dfs(i+1, j), dfs(i, j+1))

```java
class Solution {
    String s1, s2;
    int[][] memo;
    
    public int longestCommonSubsequence(String text1, String text2) {
        this.s1 = text1;
        this.s2 = text2;
        int n = s1.length(), m = s2.length();
        memo = new int[n][m];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        return dfs(0, 0);
    }
    
    private int dfs(int i, int j) {
        if (i == s1.length() || j == s2.length()) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int res;
        if (s1.charAt(i) == s2.charAt(j)) {
            res = 1 + dfs(i + 1, j + 1);
        } else {
            res = Math.max(dfs(i + 1, j), dfs(i, j + 1));
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** Classic LCS - 2D DP on indices. Match → diagonal, mismatch → max(up, left).

## 2. **Shortest Common Supersequence (Leetcode 1092)**

**Instinct:**  
Do strings ko merge karna hai such that both are subsequences. Shortest length = m + n - LCS. Print bhi karna hai toh LCS ke basis pe build karo.

**Approach:**  
1. First find LCS using DP
2. Build supersequence by traversing backwards
3. Agar chars match → include once
4. Agar nahi match → include char from string with larger remaining

```java
class Solution {
    String s1, s2;
    int[][] memo;
    
    public String shortestCommonSupersequence(String str1, String str2) {
        this.s1 = str1;
        this.s2 = str2;
        int n = s1.length(), m = s2.length();
        memo = new int[n][m];
        for (int i = 0; i < n; i++) Arrays.fill(memo[i], -1);
        
        // First compute LCS length (for DP table)
        dfs(0, 0);
        
        // Build the supersequence
        StringBuilder sb = new StringBuilder();
        int i = 0, j = 0;
        while (i < n && j < m) {
            if (s1.charAt(i) == s2.charAt(j)) {
                sb.append(s1.charAt(i));
                i++;
                j++;
            } else {
                // Check which direction gives longer LCS
                int down = (i + 1 < n) ? getLen(i + 1, j) : 0;
                int right = (j + 1 < m) ? getLen(i, j + 1) : 0;
                if (down >= right) {
                    sb.append(s1.charAt(i));
                    i++;
                } else {
                    sb.append(s2.charAt(j));
                    j++;
                }
            }
        }
        
        // Add remaining characters
        while (i < n) sb.append(s1.charAt(i++));
        while (j < m) sb.append(s2.charAt(j++));
        
        return sb.toString();
    }
    
    private int dfs(int i, int j) {
        if (i == s1.length() || j == s2.length()) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int res;
        if (s1.charAt(i) == s2.charAt(j)) {
            res = 1 + dfs(i + 1, j + 1);
        } else {
            res = Math.max(dfs(i + 1, j), dfs(i, j + 1));
        }
        memo[i][j] = res;
        return res;
    }
    
    private int getLen(int i, int j) {
        if (i == s1.length() || j == s2.length()) return 0;
        return memo[i][j];
    }
}
```

**Key Insight:** SCS length = m + n - LCS. Construction uses DP table to decide next char.

## 3. **Minimum Insertion Steps to Make a String Palindrome (Leetcode 1312)**

**Instinct:**  
String ko palindrome banane ke liye minimum insertions. Insertions = n - LPS (Longest Palindromic Subsequence). LPS = LCS of string and its reverse.

**Approach:**  
1. Reverse string
2. Find LCS of original and reverse
3. Answer = n - LCS

```java
class Solution {
    String s, rev;
    int[][] memo;
    
    public int minInsertions(String s) {
        this.s = s;
        this.rev = new StringBuilder(s).reverse().toString();
        int n = s.length();
        memo = new int[n][n];
        for (int i = 0; i < n; i++) Arrays.fill(memo[i], -1);
        
        int lps = dfs(0, 0);
        return n - lps;
    }
    
    private int dfs(int i, int j) {
        if (i == s.length() || j == rev.length()) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int res;
        if (s.charAt(i) == rev.charAt(j)) {
            res = 1 + dfs(i + 1, j + 1);
        } else {
            res = Math.max(dfs(i + 1, j), dfs(i, j + 1));
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** Minimum insertions to make palindrome = n - LPS. LPS = LCS(s, reverse(s)).

## 7. **Edit Distance (Leetcode 72)**

**Instinct:**  
Word1 ko Word2 me convert karne ke liye minimum operations (insert, delete, replace). Ye LCS ka generalization hai - 3 operations allowed.

**Approach:**  
1. Memo[i][j] = min operations for word1[i..] and word2[j..]
2. If chars match → dfs(i+1, j+1)
3. Else → min(insert, delete, replace) + 1
   - Insert: dfs(i, j+1)
   - Delete: dfs(i+1, j)
   - Replace: dfs(i+1, j+1)

```java
class Solution {
    String w1, w2;
    int[][] memo;
    
    public int minDistance(String word1, String word2) {
        this.w1 = word1;
        this.w2 = word2;
        int n = w1.length(), m = w2.length();
        memo = new int[n][m];
        for (int i = 0; i < n; i++) Arrays.fill(memo[i], -1);
        
        return dfs(0, 0);
    }
    
    private int dfs(int i, int j) {
        if (i == w1.length()) return w2.length() - j; // insert all remaining
        if (j == w2.length()) return w1.length() - i; // delete all remaining
        if (memo[i][j] != -1) return memo[i][j];
        
        int res;
        if (w1.charAt(i) == w2.charAt(j)) {
            res = dfs(i + 1, j + 1);
        } else {
            int insert = dfs(i, j + 1);
            int delete = dfs(i + 1, j);
            int replace = dfs(i + 1, j + 1);
            res = 1 + Math.min(insert, Math.min(delete, replace));
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** Edit distance = LCS with 3 operations. Base cases handle remaining chars.

## 8. **Distinct Subsequences (Leetcode 115)**

**Instinct:**  
Count of times string t appears as subsequence in string s. Har char ya toh match karo (if equal) ya skip karo.

**Approach:**  
1. Memo[i][j] = ways to form t[j..] from s[i..]
2. If j == t.length() → 1 (found)
3. If i == s.length() → 0 (not found)
4. If s[i] == t[j] → take + skip
5. Else → skip

```java
class Solution {
    String s, t;
    int[][] memo;
    
    public int numDistinct(String s, String t) {
        this.s = s;
        this.t = t;
        int n = s.length(), m = t.length();
        memo = new int[n][m];
        for (int i = 0; i < n; i++) Arrays.fill(memo[i], -1);
        
        return dfs(0, 0);
    }
    
    private int dfs(int i, int j) {
        if (j == t.length()) return 1;
        if (i == s.length()) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int res = dfs(i + 1, j); // skip
        if (s.charAt(i) == t.charAt(j)) {
            res += dfs(i + 1, j + 1); // take
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** Count subsequences = take/skip DP. Take only when chars match.

## 9. **Interleaving Strings (Leetcode 97)**

**Instinct:**  
Check if s3 is interleaving of s1 and s2 (order preserved, but interleaved). 2D DP on indices of s1 and s2.

**Approach:**  
1. Memo[i][j] = can we form s3[i+j..] from s1[i..] and s2[j..]
2. If i+j == s3.length() → true
3. Try taking from s1 if s1[i] == s3[i+j]
4. Try taking from s2 if s2[j] == s3[i+j]

```java
class Solution {
    String s1, s2, s3;
    int[][] memo;
    
    public boolean isInterleave(String s1, String s2, String s3) {
        this.s1 = s1;
        this.s2 = s2;
        this.s3 = s3;
        if (s1.length() + s2.length() != s3.length()) return false;
        
        int n = s1.length(), m = s2.length();
        memo = new int[n + 1][m + 1];
        for (int i = 0; i <= n; i++) Arrays.fill(memo[i], -1);
        
        return dfs(0, 0);
    }
    
    private boolean dfs(int i, int j) {
        if (i + j == s3.length()) return true;
        if (memo[i][j] != -1) return memo[i][j] == 1;
        
        boolean res = false;
        if (i < s1.length() && s1.charAt(i) == s3.charAt(i + j)) {
            res = res || dfs(i + 1, j);
        }
        if (j < s2.length() && s2.charAt(j) == s3.charAt(i + j)) {
            res = res || dfs(i, j + 1);
        }
        
        memo[i][j] = res ? 1 : 0;
        return res;
    }
}
```

**Key Insight:** Interleaving = pick from s1 or s2 at each step. 2D DP on indices.

## 10. **Wildcard Matching (Leetcode 44)**

**Instinct:**  
Pattern with '?' (any single char) and '*' (any sequence). DP on indices of string and pattern.

**Approach:**  
1. Memo[i][j] = can pattern[j..] match string[i..]
2. If j == pattern.length() → i == string.length()
3. If i == string.length() → check if remaining pattern are all '*'
4. If pattern[j] == '*' → match zero or more chars
5. If pattern[j] == '?' or match char → move both

```java
class Solution {
    String s, p;
    int[][] memo;
    
    public boolean isMatch(String s, String p) {
        this.s = s;
        this.p = p;
        int n = s.length(), m = p.length();
        memo = new int[n + 1][m + 1];
        for (int i = 0; i <= n; i++) Arrays.fill(memo[i], -1);
        
        return dfs(0, 0);
    }
    
    private boolean dfs(int i, int j) {
        if (j == p.length()) return i == s.length();
        if (memo[i][j] != -1) return memo[i][j] == 1;
        
        boolean res;
        if (p.charAt(j) == '*') {
            // match zero or more characters
            res = dfs(i, j + 1) || (i < s.length() && dfs(i + 1, j));
        } else {
            boolean match = i < s.length() && 
                (p.charAt(j) == '?' || p.charAt(j) == s.charAt(i));
            res = match && dfs(i + 1, j + 1);
        }
        
        memo[i][j] = res ? 1 : 0;
        return res;
    }
}
```

**Key Insight:** Wildcard '*' can match zero or more. '?' matches exactly one char.

## 11. **Regular Expression Matching (Leetcode 10)**

**Instinct:**  
Pattern with '.' (any char) and '*' (zero or more of previous char). More complex than wildcard.

**Approach:**  
1. Memo[i][j] = can pattern[j..] match string[i..]
2. If j == p.length() → i == s.length()
3. Check if next char is '*'
4. If yes → match zero or more of current char
5. If no → match single char (with '.' support)

```java
class Solution {
    String s, p;
    int[][] memo;
    
    public boolean isMatch(String s, String p) {
        this.s = s;
        this.p = p;
        int n = s.length(), m = p.length();
        memo = new int[n + 1][m + 1];
        for (int i = 0; i <= n; i++) Arrays.fill(memo[i], -1);
        
        return dfs(0, 0);
    }
    
    private boolean dfs(int i, int j) {
        if (j == p.length()) return i == s.length();
        if (memo[i][j] != -1) return memo[i][j] == 1;
        
        boolean firstMatch = i < s.length() && 
            (p.charAt(j) == '.' || p.charAt(j) == s.charAt(i));
        
        boolean res;
        if (j + 1 < p.length() && p.charAt(j + 1) == '*') {
            // match zero or more of current char
            res = dfs(i, j + 2) || (firstMatch && dfs(i + 1, j));
        } else {
            res = firstMatch && dfs(i + 1, j + 1);
        }
        
        memo[i][j] = res ? 1 : 0;
        return res;
    }
}
```

**Key Insight:** Regex '*' applies to previous character. Need to check next char for '*'.

## 12. **Longest Common Substring (Contiguous)**

**Instinct:**  
LCS ka contiguous version - subarray, not subsequence. Agar mismatch ho toh chain toot jaati hai.

**Approach:**  
1. Memo[i][j] = length of common substring ending at i in s1 and j in s2
2. If chars match → 1 + dfs(i-1, j-1)
3. Else → 0
4. Track max during recursion

```java
class Solution {
    String s1, s2;
    int[][] memo;
    int maxLen = 0;
    
    public int longestCommonSubstring(String text1, String text2) {
        this.s1 = text1;
        this.s2 = text2;
        int n = s1.length(), m = s2.length();
        memo = new int[n][m];
        for (int i = 0; i < n; i++) Arrays.fill(memo[i], -1);
        
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                maxLen = Math.max(maxLen, dfs(i, j));
            }
        }
        return maxLen;
    }
    
    private int dfs(int i, int j) {
        if (i < 0 || j < 0) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int res = 0;
        if (s1.charAt(i) == s2.charAt(j)) {
            res = 1 + dfs(i - 1, j - 1);
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** Substring = contiguous. Mismatch pe chain break (0 return).

## 14. **Minimum ASCII Delete Sum for Two Strings (Leetcode 712)**

**Instinct:**  
Delete karna hai, but cost = ASCII value. Min cost to make strings equal. Ye weighted LCS hai - maximize sum of common chars, then total sum - 2*commonSum.

**Approach:**  
1. Memo[i][j] = max ASCII sum of common subsequence
2. If match → sum + dfs(i+1, j+1)
3. Else → max(dfs(i+1, j), dfs(i, j+1))
4. Answer = totalSum1 + totalSum2 - 2 * maxCommonSum

```java
class Solution {
    String s1, s2;
    Integer[][] memo;
    
    public int minimumDeleteSum(String s1, String s2) {
        this.s1 = s1;
        this.s2 = s2;
        int n = s1.length(), m = s2.length();
        memo = new Integer[n][m];
        
        int totalSum = 0;
        for (char c : s1.toCharArray()) totalSum += c;
        for (char c : s2.toCharArray()) totalSum += c;
        
        int commonSum = dfs(0, 0);
        return totalSum - 2 * commonSum;
    }
    
    private int dfs(int i, int j) {
        if (i == s1.length() || j == s2.length()) return 0;
        if (memo[i][j] != null) return memo[i][j];
        
        int res;
        if (s1.charAt(i) == s2.charAt(j)) {
            res = s1.charAt(i) + dfs(i + 1, j + 1);
        } else {
            res = Math.max(dfs(i + 1, j), dfs(i, j + 1));
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** Weighted LCS - maximize sum instead of length.

## 15. **Longest Common Subsequence with at most K changes**

**Instinct:**  
LCS but allowed to change up to K characters in one string to match the other. Extra state for remaining changes.

**Approach:**  
1. Memo[i][j][k] = LCS length with k changes left
2. If chars match → 1 + dfs(i+1, j+1, k)
3. If chars don't match and k > 0 → 1 + dfs(i+1, j+1, k-1)
4. Else → max(dfs(i+1, j, k), dfs(i, j+1, k))

```java
class Solution {
    String s1, s2;
    int[][][] memo;
    
    public int longestCommonSubsequenceWithChanges(String text1, String text2, int k) {
        this.s1 = text1;
        this.s2 = text2;
        int n = s1.length(), m = s2.length();
        memo = new int[n][m][k + 1];
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        return dfs(0, 0, k);
    }
    
    private int dfs(int i, int j, int k) {
        if (i == s1.length() || j == s2.length()) return 0;
        if (memo[i][j][k] != -1) return memo[i][j][k];
        
        int res;
        if (s1.charAt(i) == s2.charAt(j)) {
            res = 1 + dfs(i + 1, j + 1, k);
        } else {
            int change = 0;
            if (k > 0) {
                change = 1 + dfs(i + 1, j + 1, k - 1);
            }
            int skip1 = dfs(i + 1, j, k);
            int skip2 = dfs(i, j + 1, k);
            res = Math.max(change, Math.max(skip1, skip2));
        }
        
        memo[i][j][k] = res;
        return res;
    }
}
```

**Key Insight:** Extra dimension for remaining changes. Change allowed when chars don't match.

## 16. **Maximum Length of Repeated Subarray (Leetcode 718) - LCS on arrays**

**Instinct:**  
Two arrays me longest common subarray (contiguous). Same as longest common substring but for arrays.

**Approach:**  
1. Memo[i][j] = length of common subarray starting at i in nums1 and j in nums2
2. If nums1[i] == nums2[j] → 1 + dfs(i+1, j+1)
3. Else → 0

```java
class Solution {
    int[] n1, n2;
    int[][] memo;
    
    public int findLength(int[] nums1, int[] nums2) {
        this.n1 = nums1;
        this.n2 = nums2;
        int n = n1.length, m = n2.length;
        memo = new int[n][m];
        for (int i = 0; i < n; i++) Arrays.fill(memo[i], -1);
        
        int maxLen = 0;
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                maxLen = Math.max(maxLen, dfs(i, j));
            }
        }
        return maxLen;
    }
    
    private int dfs(int i, int j) {
        if (i >= n1.length || j >= n2.length) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int res = 0;
        if (n1[i] == n2[j]) {
            res = 1 + dfs(i + 1, j + 1);
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** Subarray = contiguous. Mismatch pe chain toot jaati hai.

## 17. **Longest Common Subsequence of Three Strings**

**Instinct:**  
Three strings ka LCS. 3D DP on indices i, j, k.

**Approach:**  
1. Memo[i][j][k] = LCS length for three strings
2. If all three chars match → 1 + dfs(i+1, j+1, k+1)
3. Else → max of all three skipping one each

```java
class Solution {
    String s1, s2, s3;
    int[][][] memo;
    
    public int longestCommonSubsequenceThree(String text1, String text2, String text3) {
        this.s1 = text1;
        this.s2 = text2;
        this.s3 = text3;
        int n = s1.length(), m = s2.length(), l = s3.length();
        memo = new int[n][m][l];
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        return dfs(0, 0, 0);
    }
    
    private int dfs(int i, int j, int k) {
        if (i == s1.length() || j == s2.length() || k == s3.length()) return 0;
        if (memo[i][j][k] != -1) return memo[i][j][k];
        
        int res;
        if (s1.charAt(i) == s2.charAt(j) && s2.charAt(j) == s3.charAt(k)) {
            res = 1 + dfs(i + 1, j + 1, k + 1);
        } else {
            res = Math.max(
                Math.max(dfs(i + 1, j, k), dfs(i, j + 1, k)),
                dfs(i, j, k + 1)
            );
        }
        
        memo[i][j][k] = res;
        return res;
    }
}
```

**Key Insight:** 3D LCS - extension of 2D LCS. All three must match for diagonal.

## 18. **Count Different Palindromic Subsequences (Leetcode 730)**

**Instinct:**  
Count of distinct palindromic subsequences in a string. LCS based approach with set for uniqueness.

**Approach:**  
1. Memo[i][j] = set of distinct palindromes in s[i..j]
2. If i > j → empty set
3. If i == j → {s[i]}
4. If s[i] == s[j] → new palindromes = s[i] + inner + s[i]
5. Else → union of left and right

```java
class Solution {
    int MOD = 1_000_000_007;
    String s;
    Set<String>[][] memo;
    
    public int countPalindromicSubsequences(String s) {
        this.s = s;
        int n = s.length();
        memo = new Set[n][n];
        Set<String> all = dfs(0, n - 1);
        return all.size() % MOD;
    }
    
    private Set<String> dfs(int i, int j) {
        if (i > j) return new HashSet<>();
        if (i == j) {
            Set<String> set = new HashSet<>();
            set.add(s.substring(i, i + 1));
            return set;
        }
        if (memo[i][j] != null) return memo[i][j];
        
        Set<String> res = new HashSet<>();
        if (s.charAt(i) == s.charAt(j)) {
            Set<String> inner = dfs(i + 1, j - 1);
            res.add(s.substring(i, i + 1));
            res.add(s.substring(i, i + 1) + s.substring(i, i + 1));
            for (String str : inner) {
                res.add(s.charAt(i) + str + s.charAt(j));
            }
        } else {
            res.addAll(dfs(i + 1, j));
            res.addAll(dfs(i, j - 1));
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** Distinct palindromic subsequences - use Set to avoid duplicates.

## 19. **Maximum Sum of Two Non-Overlapping Subsequences (LCS variant)**

**Instinct:**  
Two non-overlapping subsequences with max sum. LCS jaisa but with sum and overlap constraint.

**Approach:**  
1. Memo[i][j][taken1][taken2] = max sum
2. Track whether each subsequence has started/ended

```java
class Solution {
    int[] nums;
    int k1, k2;
    Integer[][][][] memo;
    
    public int maxSumTwoSubsequences(int[] nums, int k1, int k2) {
        this.nums = nums;
        this.k1 = k1;
        this.k2 = k2;
        int n = nums.length;
        memo = new Integer[n][n][k1 + 1][k2 + 1];
        return dfs(0, -1, k1, k2);
    }
    
    private int dfs(int i, int prev, int rem1, int rem2) {
        if (i == nums.length) return 0;
        if (rem1 == 0 && rem2 == 0) return 0;
        
        int prevIdx = prev + 1;
        if (memo[i][prevIdx][rem1][rem2] != null) return memo[i][prevIdx][rem1][rem2];
        
        int skip = dfs(i + 1, prev, rem1, rem2);
        int take = 0;
        
        if (prev == -1 || nums[i] > nums[prev]) {
            if (rem1 > 0) {
                take = Math.max(take, nums[i] + dfs(i + 1, i, rem1 - 1, rem2));
            }
            if (rem2 > 0) {
                take = Math.max(take, nums[i] + dfs(i + 1, i, rem1, rem2 - 1));
            }
        }
        
        memo[i][prevIdx][rem1][rem2] = Math.max(skip, take);
        return memo[i][prevIdx][rem1][rem2];
    }
}
```

**Key Insight:** Two subsequences simultaneously - track remaining lengths for both.

## 20. **Longest Common Subsequence with Wildcard (Pattern Matching)**

**Instinct:**  
Pattern with '?' (any char) and '*' (any sequence). LCS jaisa but pattern can have wildcards.

**Approach:**  
1. Memo[i][j] = max LCS length
2. If pattern char is '?' or matches → 1 + dfs(i+1, j+1)
3. If pattern char is '*' → max of match zero or more chars
4. Else → max(skip pattern, skip string)

```java
class Solution {
    String s, p;
    int[][] memo;
    
    public int longestCommonSubsequenceWithWildcard(String s, String pattern) {
        this.s = s;
        this.p = pattern;
        int n = s.length(), m = pattern.length();
        memo = new int[n][m];
        for (int i = 0; i < n; i++) Arrays.fill(memo[i], -1);
        
        return dfs(0, 0);
    }
    
    private int dfs(int i, int j) {
        if (j == p.length()) return 0;
        if (i == s.length()) {
            // Check if remaining pattern are all '*'
            for (int k = j; k < p.length(); k++) {
                if (p.charAt(k) != '*') return 0;
            }
            return 0;
        }
        if (memo[i][j] != -1) return memo[i][j];
        
        int res;
        if (p.charAt(j) == '*') {
            // match zero or more chars
            res = Math.max(dfs(i + 1, j), dfs(i, j + 1));
        } else if (p.charAt(j) == '?' || p.charAt(j) == s.charAt(i)) {
            res = 1 + dfs(i + 1, j + 1);
        } else {
            res = Math.max(dfs(i + 1, j), dfs(i, j + 1));
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** Wildcard '*' can match any sequence. '?' matches exactly one char.

## 🎯 Key Takeaways for LCS Pattern (Memoization)

### 1. **Classic LCS Structure:**
```java
dfs(i, j) {
    if (i == n || j == m) return 0;
    if (s1[i] == s2[j]) {
        return 1 + dfs(i+1, j+1);
    } else {
        return Math.max(dfs(i+1, j), dfs(i, j+1));
    }
}
```

### 2. **Common Variations Table:**

| Variation | Change in Recurrence |
|-----------|---------------------|
| Shortest Common Supersequence | Length = m+n - LCS |
| Edit Distance | Min(insert, delete, replace) + 1 |
| Distinct Subsequences | Count = skip + (take if match) |
| Interleaving | OR of pick from s1 or s2 |
| Wildcard | Handle '*' specially |
| Weighted LCS | Sum instead of +1 |

### 3. **State Space:**
- Classic LCS: `(i, j)` - O(n²)
- With K changes: `(i, j, k)` - O(n²k)
- Three strings: `(i, j, k)` - O(n³)

### 4. **Base Cases Important:**
```java
if (i == n) return 0;                    // string exhausted
if (j == m) return 0;                    // pattern exhausted
if (i == n) return remaining insertions; // for edit distance
```

### 5. **When to use LCS Pattern:**
- Two sequences ko compare karna hai
- Order preservation important hai
- Need longest common subsequence/substring
- Edit distance / pattern matching problems

### 6. **Optimization Techniques:**
- For LCS length only: 2 rows enough (space O(min(m,n)))
- For LCS string: need full table for reconstruction
- Memoization vs Tabulation: both O(n²) time

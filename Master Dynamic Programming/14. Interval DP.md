# Interval DP Pattern - Complete Guide

## Interval DP ke 10 Main Variants:

| Variant | Description | Approach |
|---------|-------------|----------|
| **Matrix Chain Multiplication** | Min cost to multiply matrices | Try all partition points |
| **Burst Balloons** | Max coins by bursting balloons | Last balloon approach |
| **Minimum Cost to Cut Stick** | Min cost to cut at given positions | Interval DP on cuts |
| **Palindrome Partitioning** | Min cuts to make all palindromes | DP + palindrome check |
| **Longest Palindromic Subsequence** | Longest palindrome (non-contiguous) | Match ends or skip |
| **Stone Game** | Max difference in stone game | Min-max DP |
| **Predict the Winner** | Predict winner in game | Min-max DP |
| **Optimal BST** | Min search cost for BST | Try each root |
| **Scramble String** | Check if string is scrambled | Partition + swap/no swap |
| **Egg Drop** | Min moves with eggs | Binary search + DP |

## 1. **Matrix Chain Multiplication**

**Instinct:**  
Matrices ko multiply karne ka minimum cost. Cost = dimensions[i-1] * dimensions[k] * dimensions[j]. Har partition point k try karo - left part + right part + current cost.

**Approach:**  
1. Memo[i][j] = min cost to multiply matrices from i to j
2. Base: i == j → 0
3. Try all k from i to j-1
4. cost = dp[i][k] + dp[k+1][j] + dims[i-1] * dims[k] * dims[j]

```java
class Solution {
    int[] dims;
    int[][] memo;
    int INF = Integer.MAX_VALUE / 2;
    
    public int matrixChainMultiplication(int[] dimensions) {
        this.dims = dimensions;
        int n = dimensions.length - 1;
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        return dfs(0, n - 1);
    }
    
    private int dfs(int i, int j) {
        if (i >= j) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int minCost = INF;
        for (int k = i; k < j; k++) {
            int cost = dfs(i, k) + dfs(k + 1, j) + dims[i] * dims[k + 1] * dims[j + 1];
            minCost = Math.min(minCost, cost);
        }
        
        memo[i][j] = minCost;
        return minCost;
    }
}
```

**Key Insight:** Partition DP - try all split points. Cost = left + right + combine cost.

## 2. **Burst Balloons (Leetcode 312)**

**Instinct:**  
Balloons burst karne se coins milte hain = nums[left] * nums[i] * nums[right]. Last balloon socho jo burst hoga - uske left aur right independent ho jaate hain.

**Approach:**  
1. Add 1 at boundaries: [1, nums, 1]
2. Memo[i][j] = max coins from i+1 to j-1
3. Try each k as last balloon in range

```java
class Solution {
    int[] arr;
    int[][] memo;
    
    public int maxCoins(int[] nums) {
        int n = nums.length;
        arr = new int[n + 2];
        arr[0] = arr[n + 1] = 1;
        for (int i = 1; i <= n; i++) {
            arr[i] = nums[i - 1];
        }
        
        memo = new int[n + 2][n + 2];
        for (int i = 0; i < n + 2; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        return dfs(1, n);
    }
    
    private int dfs(int i, int j) {
        if (i > j) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int max = 0;
        for (int k = i; k <= j; k++) {
            int coins = arr[i - 1] * arr[k] * arr[j + 1];
            int total = coins + dfs(i, k - 1) + dfs(k + 1, j);
            max = Math.max(max, total);
        }
        
        memo[i][j] = max;
        return max;
    }
}
```

**Key Insight:** Last balloon approach - partition around k. Left and right become independent.

## 3. **Minimum Cost to Cut a Stick (Leetcode 1547)**

**Instinct:**  
Stick ko given cuts pe cut karna hai. Cost = length of current stick. Order of cuts se total cost change hoti hai. Interval DP.

**Approach:**  
1. Add 0 and n to cuts array, sort
2. Memo[i][j] = min cost to cut between cuts[i] and cuts[j]
3. Try each k as first cut

```java
class Solution {
    int[] cuts;
    int[][] memo;
    
    public int minCost(int n, int[] cuts) {
        int m = cuts.length;
        this.cuts = new int[m + 2];
        this.cuts[0] = 0;
        this.cuts[m + 1] = n;
        for (int i = 0; i < m; i++) {
            this.cuts[i + 1] = cuts[i];
        }
        Arrays.sort(this.cuts);
        
        memo = new int[m + 2][m + 2];
        for (int i = 0; i < m + 2; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        return dfs(0, m + 1);
    }
    
    private int dfs(int i, int j) {
        if (i + 1 >= j) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int minCost = Integer.MAX_VALUE;
        for (int k = i + 1; k < j; k++) {
            int cost = (cuts[j] - cuts[i]) + dfs(i, k) + dfs(k, j);
            minCost = Math.min(minCost, cost);
        }
        
        memo[i][j] = minCost;
        return minCost;
    }
}
```

**Key Insight:** Interval DP on cut positions. Cost = current length + left + right.

## 4. **Palindrome Partitioning II (Leetcode 132)**

**Instinct:**  
String ko minimum cuts me palindrome parts me divide karna hai. Har partition point check karo agar palindrome hai toh.

**Approach:**  
1. Precompute isPalindrome[i][j]
2. Memo[i] = min cuts from index i to end
3. For each j >= i, if s[i..j] is palindrome, cuts = 1 + dfs(j+1)

```java
class Solution {
    String s;
    int[] memo;
    Boolean[][] isPal;
    
    public int minCut(String s) {
        this.s = s;
        int n = s.length();
        memo = new int[n];
        Arrays.fill(memo, -1);
        isPal = new Boolean[n][n];
        
        return dfs(0);
    }
    
    private int dfs(int i) {
        if (i == s.length()) return -1;
        if (memo[i] != -1) return memo[i];
        
        int minCuts = Integer.MAX_VALUE;
        for (int j = i; j < s.length(); j++) {
            if (isPalindrome(i, j)) {
                minCuts = Math.min(minCuts, 1 + dfs(j + 1));
            }
        }
        
        memo[i] = minCuts;
        return minCuts;
    }
    
    private boolean isPalindrome(int i, int j) {
        if (i >= j) return true;
        if (isPal[i][j] != null) return isPal[i][j];
        
        boolean res = (s.charAt(i) == s.charAt(j)) && isPalindrome(i + 1, j - 1);
        isPal[i][j] = res;
        return res;
    }
}
```

**Key Insight:** 1D DP on start index. Palindrome check with memoization.

## 5. **Longest Palindromic Subsequence (Leetcode 516)**

**Instinct:**  
String ka longest palindromic subsequence (non-contiguous). Interval DP on indices.

**Approach:**  
1. Memo[i][j] = LPS length for s[i..j]
2. Base: i > j → 0; i == j → 1
3. If s[i] == s[j] → 2 + dfs(i+1, j-1)
4. Else → max(dfs(i+1, j), dfs(i, j-1))

```java
class Solution {
    String s;
    int[][] memo;
    
    public int longestPalindromeSubseq(String s) {
        this.s = s;
        int n = s.length();
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        return dfs(0, n - 1);
    }
    
    private int dfs(int i, int j) {
        if (i > j) return 0;
        if (i == j) return 1;
        if (memo[i][j] != -1) return memo[i][j];
        
        int res;
        if (s.charAt(i) == s.charAt(j)) {
            res = 2 + dfs(i + 1, j - 1);
        } else {
            res = Math.max(dfs(i + 1, j), dfs(i, j - 1));
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** Classic LPS - match ends or skip from left/right.

## 6. **Stone Game (Leetcode 877)**

**Instinct:**  
Two players pick stones from ends. Max difference for current player. Interval DP on array.

**Approach:**  
1. Memo[i][j] = max difference current player can get from piles[i..j]
2. Base: i == j → piles[i]
3. Pick left: piles[i] - dfs(i+1, j)
4. Pick right: piles[j] - dfs(i, j-1)

```java
class Solution {
    int[] piles;
    int[][] memo;
    
    public boolean stoneGame(int[] piles) {
        this.piles = piles;
        int n = piles.length;
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        int diff = dfs(0, n - 1);
        return diff > 0;
    }
    
    private int dfs(int i, int j) {
        if (i > j) return 0;
        if (i == j) return piles[i];
        if (memo[i][j] != -1) return memo[i][j];
        
        int pickLeft = piles[i] - dfs(i + 1, j);
        int pickRight = piles[j] - dfs(i, j - 1);
        
        memo[i][j] = Math.max(pickLeft, pickRight);
        return memo[i][j];
    }
}
```

**Key Insight:** Min-max DP. Current player's advantage = pick - opponent's advantage.

## 7. **Predict the Winner (Leetcode 486)**

**Instinct:**  
Same as stone game but array can have odd length. Player 1 wins if difference >= 0.

**Approach:**  
Same as stone game.

```java
class Solution {
    int[] nums;
    int[][] memo;
    
    public boolean predictTheWinner(int[] nums) {
        this.nums = nums;
        int n = nums.length;
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        int diff = dfs(0, n - 1);
        return diff >= 0;
    }
    
    private int dfs(int i, int j) {
        if (i > j) return 0;
        if (i == j) return nums[i];
        if (memo[i][j] != -1) return memo[i][j];
        
        int left = nums[i] - dfs(i + 1, j);
        int right = nums[j] - dfs(i, j - 1);
        
        memo[i][j] = Math.max(left, right);
        return memo[i][j];
    }
}
```

**Key Insight:** Same min-max DP. Player 1 wins if final difference >= 0.

## 8. **Optimal Binary Search Tree**

**Instinct:**  
Keys with probabilities, build BST with minimum search cost. Har root ke liye left subtree cost + right subtree cost + root's contribution.

**Approach:**  
1. Memo[i][j] = min cost for keys[i..j]
2. Try each k as root
3. cost = left cost + right cost + sum of probabilities in range

```java
class Solution {
    int[] keys;
    int[] freq;
    int[][] memo;
    int[] prefixSum;
    
    public int optimalBST(int[] keys, int[] freq) {
        this.keys = keys;
        this.freq = freq;
        int n = keys.length;
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        prefixSum = new int[n + 1];
        for (int i = 0; i < n; i++) {
            prefixSum[i + 1] = prefixSum[i] + freq[i];
        }
        
        return dfs(0, n - 1);
    }
    
    private int dfs(int i, int j) {
        if (i > j) return 0;
        if (i == j) return freq[i];
        if (memo[i][j] != -1) return memo[i][j];
        
        int totalFreq = prefixSum[j + 1] - prefixSum[i];
        int minCost = Integer.MAX_VALUE;
        
        for (int k = i; k <= j; k++) {
            int leftCost = dfs(i, k - 1);
            int rightCost = dfs(k + 1, j);
            int cost = leftCost + rightCost + totalFreq;
            minCost = Math.min(minCost, cost);
        }
        
        memo[i][j] = minCost;
        return minCost;
    }
}
```

**Key Insight:** Each root adds its depth * freq. Total freq = sum of frequencies in range.

## 9. **Scramble String (Leetcode 87)**

**Instinct:**  
String ko scramble karke doosri string ban sakti hai ya nahi. Partition string into two parts, swap or not swap.

**Approach:**  
1. Memo[i][j][len] = can s1[i..i+len-1] be scrambled to s2[j..j+len-1]
2. Try all partition points
3. Check (s1 left == s2 left AND s1 right == s2 right) OR (swap case)

```java
class Solution {
    String s1, s2;
    Boolean[][][] memo;
    
    public boolean isScramble(String s1, String s2) {
        this.s1 = s1;
        this.s2 = s2;
        int n = s1.length();
        memo = new Boolean[n][n][n + 1];
        return dfs(0, 0, n);
    }
    
    private boolean dfs(int i, int j, int len) {
        if (len == 0) return true;
        if (len == 1) return s1.charAt(i) == s2.charAt(j);
        if (memo[i][j][len] != null) return memo[i][j][len];
        
        // Quick check: same character count
        int[] count = new int[26];
        for (int k = 0; k < len; k++) {
            count[s1.charAt(i + k) - 'a']++;
            count[s2.charAt(j + k) - 'a']--;
        }
        for (int c : count) {
            if (c != 0) {
                memo[i][j][len] = false;
                return false;
            }
        }
        
        for (int k = 1; k < len; k++) {
            // No swap case
            if (dfs(i, j, k) && dfs(i + k, j + k, len - k)) {
                memo[i][j][len] = true;
                return true;
            }
            // Swap case
            if (dfs(i, j + len - k, k) && dfs(i + k, j, len - k)) {
                memo[i][j][len] = true;
                return true;
            }
        }
        
        memo[i][j][len] = false;
        return false;
    }
}
```

**Key Insight:** 3D interval DP (i, j, len). Partition + two possibilities: swap or not swap.

## 10. **Super Egg Drop (Leetcode 887)**

**Instinct:**  
K eggs, N floors. Minimum moves to find critical floor. Try each floor, if egg breaks go down, else go up.

**Approach:**  
1. Memo[k][n] = min moves with k eggs and n floors
2. Base: n == 0 → 0; k == 1 → n
3. Binary search optimization to find optimal floor

```java
class Solution {
    int[][] memo;
    
    public int superEggDrop(int k, int n) {
        memo = new int[k + 1][n + 1];
        for (int i = 0; i <= k; i++) {
            Arrays.fill(memo[i], -1);
        }
        return dfs(k, n);
    }
    
    private int dfs(int k, int n) {
        if (n == 0) return 0;
        if (k == 1) return n;
        if (memo[k][n] != -1) return memo[k][n];
        
        int lo = 1, hi = n;
        int res = Integer.MAX_VALUE;
        
        while (lo <= hi) {
            int mid = lo + (hi - lo) / 2;
            int broken = dfs(k - 1, mid - 1);
            int notBroken = dfs(k, n - mid);
            
            if (broken > notBroken) {
                hi = mid - 1;
                res = Math.min(res, broken + 1);
            } else {
                lo = mid + 1;
                res = Math.min(res, notBroken + 1);
            }
        }
        
        memo[k][n] = res;
        return res;
    }
}
```

**Key Insight:** Binary search on floor to minimize worst case. Partition DP on floors.

## 11. **Minimum Cost Tree From Leaf Values (Leetcode 1130)**

**Instinct:**  
Binary tree from leaf values, internal node value = product of max leaves in subtree. Min sum of internal nodes.

**Approach:**  
1. Memo[i][j] = min cost for leaves[i..j]
2. Try each partition point k
3. cost = max(i..k) * max(k+1..j) + leftCost + rightCost

```java
class Solution {
    int[] arr;
    int[][] memo;
    int[][] maxVal;
    
    public int mctFromLeafValues(int[] arr) {
        this.arr = arr;
        int n = arr.length;
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        // Precompute max in any range
        maxVal = new int[n][n];
        for (int i = 0; i < n; i++) {
            maxVal[i][i] = arr[i];
            for (int j = i + 1; j < n; j++) {
                maxVal[i][j] = Math.max(maxVal[i][j - 1], arr[j]);
            }
        }
        
        return dfs(0, n - 1);
    }
    
    private int dfs(int i, int j) {
        if (i >= j) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int minCost = Integer.MAX_VALUE;
        for (int k = i; k < j; k++) {
            int cost = maxVal[i][k] * maxVal[k + 1][j] + dfs(i, k) + dfs(k + 1, j);
            minCost = Math.min(minCost, cost);
        }
        
        memo[i][j] = minCost;
        return minCost;
    }
}
```

**Key Insight:** Partition DP with precomputed max values. Internal node cost = product of maxes.

## 12. **Number of Ways to Cut a Stick (Leetcode 1547 - Count ways)**

**Instinct:**  
Stick ko given cuts pe cut karna hai. Number of different ways to cut (order matters). Partition DP with counting.

**Approach:**  
1. Add 0 and n to cuts, sort
2. Memo[i][j] = number of ways to cut between cuts[i] and cuts[j]
3. Sum over all k as first cut

```java
class Solution {
    int MOD = 1_000_000_007;
    int[] cuts;
    Long[][] memo;
    
    public int numberOfWaysToCut(int n, int[] cuts) {
        int m = cuts.length;
        this.cuts = new int[m + 2];
        this.cuts[0] = 0;
        this.cuts[m + 1] = n;
        for (int i = 0; i < m; i++) {
            this.cuts[i + 1] = cuts[i];
        }
        Arrays.sort(this.cuts);
        
        memo = new Long[m + 2][m + 2];
        return (int)(dfs(0, m + 1) % MOD);
    }
    
    private long dfs(int i, int j) {
        if (i + 1 >= j) return 1;
        if (memo[i][j] != null) return memo[i][j];
        
        long ways = 0;
        for (int k = i + 1; k < j; k++) {
            ways = (ways + dfs(i, k) * dfs(k, j)) % MOD;
        }
        
        memo[i][j] = ways;
        return ways;
    }
}
```

**Key Insight:** Count ways = product of left and right ways. Sum over all first cuts.

## 13. **Longest Palindromic Substring (Leetcode 5) - Interval DP**

**Instinct:**  
String ka longest palindromic substring (contiguous). Interval DP on indices.

**Approach:**  
1. Memo[i][j] = is s[i..j] palindrome?
2. If s[i] == s[j] and (j-i <= 2 or dp[i+1][j-1]) then palindrome
3. Track max length

```java
class Solution {
    String s;
    Boolean[][] memo;
    
    public String longestPalindrome(String s) {
        this.s = s;
        int n = s.length();
        memo = new Boolean[n][n];
        
        int start = 0, maxLen = 1;
        
        for (int i = 0; i < n; i++) {
            for (int j = i; j < n; j++) {
                if (isPalindrome(i, j) && j - i + 1 > maxLen) {
                    maxLen = j - i + 1;
                    start = i;
                }
            }
        }
        
        return s.substring(start, start + maxLen);
    }
    
    private boolean isPalindrome(int i, int j) {
        if (i >= j) return true;
        if (memo[i][j] != null) return memo[i][j];
        
        boolean res = (s.charAt(i) == s.charAt(j)) && isPalindrome(i + 1, j - 1);
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** Interval DP for palindrome check. Substring property.

## 14. **Remove Boxes (Leetcode 546)**

**Instinct:**  
Boxes remove karne hai. Removing k consecutive same color boxes gives k*k points. Max points.

**Approach:**  
1. Memo[i][j][k] = max points from boxes[i..j] with k extra boxes of color boxes[i] on left
2. Either remove boxes[i] with the k extras, or try to merge with same color later

```java
class Solution {
    int[] boxes;
    int[][][] memo;
    
    public int removeBoxes(int[] boxes) {
        this.boxes = boxes;
        int n = boxes.length;
        memo = new int[n][n][n];
        return dfs(0, n - 1, 0);
    }
    
    private int dfs(int i, int j, int k) {
        if (i > j) return 0;
        if (memo[i][j][k] != 0) return memo[i][j][k];
        
        // Optimize: merge same consecutive boxes
        while (i + 1 <= j && boxes[i + 1] == boxes[i]) {
            i++;
            k++;
        }
        
        // Option 1: remove boxes[i] with the k extras
        int res = (k + 1) * (k + 1) + dfs(i + 1, j, 0);
        
        // Option 2: try to merge with same color later
        for (int m = i + 1; m <= j; m++) {
            if (boxes[m] == boxes[i]) {
                res = Math.max(res, dfs(i + 1, m - 1, 0) + dfs(m, j, k + 1));
            }
        }
        
        memo[i][j][k] = res;
        return res;
    }
}
```

**Key Insight:** 3D interval DP (i, j, k). k = extra same boxes on left.

## 15. **Strange Printer (Leetcode 664)**

**Instinct:**  
Printer can print same character in one turn. Min turns to print string. Interval DP.

**Approach:**  
1. Memo[i][j] = min turns for s[i..j]
2. Base: i == j → 1
3. If s[i] == s[j], can print together: turns = dfs(i, j-1)
4. Else try all partitions

```java
class Solution {
    String s;
    int[][] memo;
    
    public int strangePrinter(String s) {
        this.s = s;
        int n = s.length();
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        return dfs(0, n - 1);
    }
    
    private int dfs(int i, int j) {
        if (i > j) return 0;
        if (i == j) return 1;
        if (memo[i][j] != -1) return memo[i][j];
        
        // Default: print s[i] separately
        int res = 1 + dfs(i + 1, j);
        
        // Try to merge s[i] with same character later
        for (int k = i + 1; k <= j; k++) {
            if (s.charAt(k) == s.charAt(i)) {
                res = Math.min(res, dfs(i + 1, k - 1) + dfs(k, j));
            }
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** If s[i] == s[k], can print s[i] and s[k] in same turn.

## 16. **Minimum Score Triangulation of Polygon (Leetcode 1039)**

**Instinct:**  
Polygon triangulation ka minimum score. Score = product of three vertices. Interval DP.

**Approach:**  
1. Memo[i][j] = min score for polygon from vertex i to j
2. Base: j - i < 2 → 0 (triangle needs 3 vertices)
3. Try all k between i and j as third vertex

```java
class Solution {
    int[] values;
    int[][] memo;
    
    public int minScoreTriangulation(int[] values) {
        this.values = values;
        int n = values.length;
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        return dfs(0, n - 1);
    }
    
    private int dfs(int i, int j) {
        if (j - i < 2) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int minScore = Integer.MAX_VALUE;
        for (int k = i + 1; k < j; k++) {
            int score = values[i] * values[k] * values[j] + dfs(i, k) + dfs(k, j);
            minScore = Math.min(minScore, score);
        }
        
        memo[i][j] = minScore;
        return minScore;
    }
}
```

**Key Insight:** Triangulation = partition at vertex k. Score = triangle (i,k,j) + left + right.

## 17. **Burst Balloons Variation with Different Scoring**

**Instinct:**  
Variation where coins = nums[i] (without left/right multiplication). Simpler interval DP.

**Approach:**  
1. Memo[i][j] = max coins from i to j
2. Try each k as last balloon
3. Coins = nums[k] + left + right

```java
class Solution {
    int[] nums;
    int[][] memo;
    
    public int maxCoinsSimple(int[] nums) {
        this.nums = nums;
        int n = nums.length;
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        return dfs(0, n - 1);
    }
    
    private int dfs(int i, int j) {
        if (i > j) return 0;
        if (i == j) return nums[i];
        if (memo[i][j] != -1) return memo[i][j];
        
        int max = 0;
        for (int k = i; k <= j; k++) {
            int coins = nums[k] + dfs(i, k - 1) + dfs(k + 1, j);
            max = Math.max(max, coins);
        }
        
        memo[i][j] = max;
        return max;
    }
}
```

**Key Insight:** Simplified version without boundary multiplication.

## 19. **Minimum Cost to Merge Stones (Leetcode 1000)**

**Instinct:**  
Stones ko k groups me merge karna hai. Cost = sum of stones in group. Minimum total cost. Interval DP.

**Approach:**  
1. Memo[i][j][k] = min cost to merge stones[i..j] into k piles
2. Base: if i == j → cost 0 if k == 1 else INF
3. Try partition at mid

```java
class Solution {
    int[] prefix;
    int K;
    int[][][] memo;
    int INF = Integer.MAX_VALUE / 2;
    
    public int mergeStones(int[] stones, int k) {
        int n = stones.length;
        if ((n - 1) % (k - 1) != 0) return -1;
        
        this.K = k;
        prefix = new int[n + 1];
        for (int i = 0; i < n; i++) {
            prefix[i + 1] = prefix[i] + stones[i];
        }
        
        memo = new int[n][n][k + 1];
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        
        return dfs(0, n - 1, 1);
    }
    
    private int dfs(int i, int j, int piles) {
        if (i == j) return piles == 1 ? 0 : INF;
        if (memo[i][j][piles] != -1) return memo[i][j][piles];
        
        if (piles == 1) {
            int val = dfs(i, j, K);
            if (val >= INF) return INF;
            return val + prefix[j + 1] - prefix[i];
        }
        
        int min = INF;
        for (int mid = i; mid < j; mid += (K - 1)) {
            int left = dfs(i, mid, piles - 1);
            int right = dfs(mid + 1, j, 1);
            min = Math.min(min, left + right);
        }
        
        memo[i][j][piles] = min;
        return min;
    }
}
```

**Key Insight:** Interval DP with piles dimension. Can only merge K piles into 1.

## 20. **Count Different Palindromic Subsequences (Leetcode 730)**

**Instinct:**  
Distinct palindromic subsequences count. Interval DP with inclusion-exclusion.

**Approach:**  
1. Memo[i][j] = count of distinct palindromes in s[i..j]
2. If s[i] == s[j], find next and prev same char
3. Use formula to avoid duplicates

```java
class Solution {
    int MOD = 1_000_000_007;
    String s;
    int[][] memo;
    
    public int countPalindromicSubsequences(String s) {
        this.s = s;
        int n = s.length();
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        return dfs(0, n - 1);
    }
    
    private int dfs(int i, int j) {
        if (i > j) return 0;
        if (i == j) return 1;
        if (memo[i][j] != -1) return memo[i][j];
        
        long res = 0;
        if (s.charAt(i) == s.charAt(j)) {
            int left = i + 1, right = j - 1;
            while (left <= right && s.charAt(left) != s.charAt(i)) left++;
            while (left <= right && s.charAt(right) != s.charAt(i)) right--;
            
            if (left > right) {
                res = 2 * dfs(i + 1, j - 1) + 2;
            } else if (left == right) {
                res = 2 * dfs(i + 1, j - 1) + 1;
            } else {
                res = 2 * dfs(i + 1, j - 1) - dfs(left + 1, right - 1);
            }
        } else {
            res = dfs(i + 1, j) + dfs(i, j - 1) - dfs(i + 1, j - 1);
        }
        
        memo[i][j] = (int)((res + MOD) % MOD);
        return memo[i][j];
    }
}
```

**Key Insight:** Inclusion-exclusion to avoid duplicates. Find next/prev same char.

## 🎯 Key Takeaways for Interval DP Pattern

### 1. **Core Interval DP Structure:**
```java
dfs(i, j) {
    if (i >= j) return base;
    if (memo[i][j] != null) return memo[i][j];
    
    int best = initial;
    for (int k = i; k <= j; k++) {
        // Try all partition points
        int cost = dfs(i, k-1) + dfs(k+1, j) + combine(i, k, j);
        best = min/max(best, cost);
    }
    
    memo[i][j] = best;
    return best;
}
```

### 2. **Common Combine Operations:**

| Problem | Combine Operation |
|---------|-------------------|
| Matrix Chain | `dims[i-1] * dims[k] * dims[j]` |
| Burst Balloons | `arr[i-1] * arr[k] * arr[j+1]` |
| Cut Stick | `cuts[j] - cuts[i]` |
| Stone Game | `nums[i] - dp[i+1][j]` or `nums[j] - dp[i][j-1]` |
| Triangulation | `values[i] * values[k] * values[j]` |

### 3. **When to use Interval DP:**
- Problem involves array/string and operations on subarrays
- Order of operations matters
- Can divide into independent subproblems
- Need to try all possible partition points
- Game theory with picks from ends

### 4. **Interval DP Optimization Techniques:**

| Technique | Use Case |
|-----------|----------|
| Quadrangle Inequality (Knuth) | Matrix chain, optimal BST |
| Divide and Conquer DP | When cost function is monotonic |
| Binary Search | Egg drop problem |
| Precomputation | Max/min in ranges, prefix sums |

### 5. **Common State Dimensions:**

```
2D Interval: (i, j)                    - O(n²)
3D Interval: (i, j, k)                 - O(n³)
1D + Interval: (i) with precomputed    - O(n²)
Game DP: (i, j) with min-max           - O(n²)
```

### 6. **Interval DP vs Other Patterns:**

| Pattern | State | Transition |
|---------|-------|------------|
| Interval DP | (i, j) | Partition at k |
| LIS | (i, prev) | Pick or skip |
| LCS | (i, j) | Match or mismatch |
| Knapsack | (i, capacity) | Take or skip |

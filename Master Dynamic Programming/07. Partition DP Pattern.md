
# Partition DP Pattern - Complete Guide

## Partition DP ke 10 Main Variants:

| Variant | Description | Approach |
|---------|-------------|----------|
| **Matrix Chain Multiplication** | Min cost to multiply matrices | Try all partition points |
| **Burst Balloons** | Max coins by bursting balloons | Last balloon approach |
| **Minimum Cost to Cut Stick** | Min cost to cut at given positions | Interval DP |
| **Palindrome Partitioning** | Min cuts to make all palindromes | DP + palindrome check |
| **Optimal BST** | Min search cost for BST | Try each root |
| **Scramble String** | Check if string is scrambled | Partition + compare |
| **Stone Game** | Max difference in stone game | Interval DP |
| **Predict the Winner** | Predict winner in game | Min-max DP |
| **Egg Drop** | Min moves with eggs | Try each floor |
| **Maximum Score from Multiplication** | Max score from operations | Partition on indices |

## 1. **Matrix Chain Multiplication (Leetcode 312 - Burst Balloons variant)**

**Instinct:**  
Matrices ko multiply karne ka minimum cost chahiye. Cost = dimensions[i-1] * dimensions[k] * dimensions[j]. Har partition point k try karo - left part + right part + current cost.

**Approach:**  
1. Memo[i][j] = min cost to multiply matrices from i to j
2. Base: i == j → 0
3. Try all k from i to j-1
4. cost = dp[i][k] + dp[k+1][j] + dims[i-1] * dims[k] * dims[j]

```java
class Solution {
    int[] dims;
    int[][] memo;
    
    public int matrixChainMultiplication(int[] dimensions) {
        this.dims = dimensions;
        int n = dimensions.length - 1; // number of matrices
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        return dfs(0, n - 1);
    }
    
    private int dfs(int i, int j) {
        if (i >= j) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int minCost = Integer.MAX_VALUE;
        for (int k = i; k < j; k++) {
            int cost = dfs(i, k) + dfs(k + 1, j) + dims[i] * dims[k + 1] * dims[j + 1];
            minCost = Math.min(minCost, cost);
        }
        
        memo[i][j] = minCost;
        return minCost;
    }
}
```

**Key Insight:** Classic partition DP - try all partition points. Cost = left + right + combine cost.

## 2. **Burst Balloons (Leetcode 312)**

**Instinct:**  
Balloons burst karne se coins milte hain = nums[left] * nums[i] * nums[right]. Last balloon ko socho jo burst hoga - uske left aur right independent ho jaate hain.

**Approach:**  
1. Add 1 at boundaries: [1, nums, 1]
2. Memo[i][j] = max coins from i+1 to j-1
3. Try each k as last balloon in range

```java
class Solution {
    int[] arr;
    int[][] memo;
    
    public int maxCoins(int[] nums) {
        int n = nums.length;
        arr = new int[n + 2];
        arr[0] = arr[n + 1] = 1;
        for (int i = 1; i <= n; i++) {
            arr[i] = nums[i - 1];
        }
        
        memo = new int[n + 2][n + 2];
        for (int i = 0; i < n + 2; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        return dfs(1, n);
    }
    
    private int dfs(int i, int j) {
        if (i > j) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int max = 0;
        for (int k = i; k <= j; k++) {
            int coins = arr[i - 1] * arr[k] * arr[j + 1];
            int total = coins + dfs(i, k - 1) + dfs(k + 1, j);
            max = Math.max(max, total);
        }
        
        memo[i][j] = max;
        return max;
    }
}
```

**Key Insight:** Last balloon approach - partition around k. Left and right become independent.

## 3. **Minimum Cost to Cut a Stick (Leetcode 1547)**

**Instinct:**  
Stick ko given cuts pe cut karna hai. Cost = length of current stick. Order of cuts se total cost change hoti hai. Interval DP.

**Approach:**  
1. Add 0 and n to cuts array, sort
2. Memo[i][j] = min cost to cut between cuts[i] and cuts[j]
3. Try each k as first cut

```java
class Solution {
    int[] cuts;
    int[][] memo;
    
    public int minCost(int n, int[] cuts) {
        int m = cuts.length;
        this.cuts = new int[m + 2];
        this.cuts[0] = 0;
        this.cuts[m + 1] = n;
        for (int i = 0; i < m; i++) {
            this.cuts[i + 1] = cuts[i];
        }
        Arrays.sort(this.cuts);
        
        memo = new int[m + 2][m + 2];
        for (int i = 0; i < m + 2; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        return dfs(0, m + 1);
    }
    
    private int dfs(int i, int j) {
        if (i + 1 >= j) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int minCost = Integer.MAX_VALUE;
        for (int k = i + 1; k < j; k++) {
            int cost = (cuts[j] - cuts[i]) + dfs(i, k) + dfs(k, j);
            minCost = Math.min(minCost, cost);
        }
        
        memo[i][j] = minCost;
        return minCost;
    }
}
```

**Key Insight:** Interval DP on cut positions. Cost = current length + left + right.

## 4. **Palindrome Partitioning II (Leetcode 132)**

**Instinct:**  
String ko minimum cuts me palindrome parts me divide karna hai. Har partition point check karo agar palindrome hai toh.

**Approach:**  
1. Memo[i] = min cuts from index i to end
2. For each j >= i, if s[i..j] is palindrome, cuts = 1 + dfs(j+1)
3. Base: i == n → -1

```java
class Solution {
    String s;
    int[] memo;
    Boolean[][] isPal;
    
    public int minCut(String s) {
        this.s = s;
        int n = s.length();
        memo = new int[n];
        Arrays.fill(memo, -1);
        isPal = new Boolean[n][n];
        
        return dfs(0);
    }
    
    private int dfs(int i) {
        if (i == s.length()) return -1;
        if (memo[i] != -1) return memo[i];
        
        int minCuts = Integer.MAX_VALUE;
        for (int j = i; j < s.length(); j++) {
            if (isPalindrome(i, j)) {
                minCuts = Math.min(minCuts, 1 + dfs(j + 1));
            }
        }
        
        memo[i] = minCuts;
        return minCuts;
    }
    
    private boolean isPalindrome(int i, int j) {
        if (i >= j) return true;
        if (isPal[i][j] != null) return isPal[i][j];
        
        boolean res = (s.charAt(i) == s.charAt(j)) && isPalindrome(i + 1, j - 1);
        isPal[i][j] = res;
        return res;
    }
}
```

**Key Insight:** 1D DP on start index. Palindrome check with memoization.

## 5. **Optimal Binary Search Tree (Leetcode 96 - Unique BST count)**

**Instinct:**  
Keys with probabilities, build BST with minimum search cost. Har root ke liye left subtree cost + right subtree cost + root's contribution.

**Approach:**  
1. Memo[i][j] = min cost for keys[i..j]
2. Try each k as root
3. cost = left cost + right cost + sum of probabilities in range

```java
class Solution {
    int[] keys;
    int[] freq;
    int[][] memo;
    int[] prefixSum;
    
    public int optimalBST(int[] keys, int[] freq) {
        this.keys = keys;
        this.freq = freq;
        int n = keys.length;
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        prefixSum = new int[n + 1];
        for (int i = 0; i < n; i++) {
            prefixSum[i + 1] = prefixSum[i] + freq[i];
        }
        
        return dfs(0, n - 1);
    }
    
    private int dfs(int i, int j) {
        if (i > j) return 0;
        if (i == j) return freq[i];
        if (memo[i][j] != -1) return memo[i][j];
        
        int totalFreq = prefixSum[j + 1] - prefixSum[i];
        int minCost = Integer.MAX_VALUE;
        
        for (int k = i; k <= j; k++) {
            int leftCost = dfs(i, k - 1);
            int rightCost = dfs(k + 1, j);
            int cost = leftCost + rightCost + totalFreq;
            minCost = Math.min(minCost, cost);
        }
        
        memo[i][j] = minCost;
        return minCost;
    }
}
```

**Key Insight:** Each root adds its depth * freq. Total freq = sum of frequencies in range.

## 6. **Scramble String (Leetcode 87)**

**Instinct:**  
String ko scramble karke doosri string ban sakti hai ya nahi. Partition string into two parts, swap or not swap.

**Approach:**  
1. Memo[s1][s2] = can s1 be scrambled to s2
2. Try all partition points
3. Check (s1 left == s2 left AND s1 right == s2 right) OR (swap case)

```java
class Solution {
    Map<String, Boolean> memo = new HashMap<>();
    
    public boolean isScramble(String s1, String s2) {
        if (s1.equals(s2)) return true;
        if (s1.length() != s2.length()) return false;
        
        String key = s1 + "#" + s2;
        if (memo.containsKey(key)) return memo.get(key);
        
        int n = s1.length();
        
        // Quick check: same character count
        int[] count = new int[26];
        for (int i = 0; i < n; i++) {
            count[s1.charAt(i) - 'a']++;
            count[s2.charAt(i) - 'a']--;
        }
        for (int c : count) {
            if (c != 0) {
                memo.put(key, false);
                return false;
            }
        }
        
        for (int i = 1; i < n; i++) {
            // No swap case
            if (isScramble(s1.substring(0, i), s2.substring(0, i)) &&
                isScramble(s1.substring(i), s2.substring(i))) {
                memo.put(key, true);
                return true;
            }
            
            // Swap case
            if (isScramble(s1.substring(0, i), s2.substring(n - i)) &&
                isScramble(s1.substring(i), s2.substring(0, n - i))) {
                memo.put(key, true);
                return true;
            }
        }
        
        memo.put(key, false);
        return false;
    }
}
```

**Key Insight:** Partition + recursion. Two possibilities: swap or not swap.

## 7. **Stone Game (Leetcode 877) - Interval DP**

**Instinct:**  
Two players pick stones from ends. Max difference for current player. Interval DP on array.

**Approach:**  
1. Memo[i][j] = max difference current player can get from piles[i..j]
2. Base: i == j → piles[i]
3. Pick left: piles[i] - dfs(i+1, j)
4. Pick right: piles[j] - dfs(i, j-1)

```java
class Solution {
    int[] piles;
    int[][] memo;
    
    public boolean stoneGame(int[] piles) {
        this.piles = piles;
        int n = piles.length;
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        int diff = dfs(0, n - 1);
        return diff > 0;
    }
    
    private int dfs(int i, int j) {
        if (i > j) return 0;
        if (i == j) return piles[i];
        if (memo[i][j] != -1) return memo[i][j];
        
        int pickLeft = piles[i] - dfs(i + 1, j);
        int pickRight = piles[j] - dfs(i, j - 1);
        
        memo[i][j] = Math.max(pickLeft, pickRight);
        return memo[i][j];
    }
}
```

**Key Insight:** Min-max DP. Current player's advantage = pick - opponent's advantage.

## 8. **Predict the Winner (Leetcode 486)**

**Instinct:**  
Same as stone game but array can have odd length. Player 1 wins if difference >= 0.

**Approach:**  
Same as stone game.

```java
class Solution {
    int[] nums;
    int[][] memo;
    
    public boolean predictTheWinner(int[] nums) {
        this.nums = nums;
        int n = nums.length;
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        int diff = dfs(0, n - 1);
        return diff >= 0;
    }
    
    private int dfs(int i, int j) {
        if (i > j) return 0;
        if (i == j) return nums[i];
        if (memo[i][j] != -1) return memo[i][j];

        return memo[i][j];
    }
}
```

**Key Insight:** Same min-max DP. Player 1 wins if final difference >= 0.

## 9. **Super Egg Drop (Leetcode 887)**

**Instinct:**  
K eggs, N floors. Minimum moves to find critical floor. Try each floor, if egg breaks go down, else go up.

**Approach:**  
1. Memo[k][n] = min moves with k eggs and n floors
2. Base: n == 0 → 0; k == 1 → n
3. Binary search optimization to find optimal floor

```java
class Solution {
    int[][] memo;
    
    public int superEggDrop(int k, int n) {
        memo = new int[k + 1][n + 1];
        for (int i = 0; i <= k; i++) {
            Arrays.fill(memo[i], -1);
        }
        return dfs(k, n);
    }
    
    private int dfs(int k, int n) {
        if (n == 0) return 0;
        if (k == 1) return n;
        if (memo[k][n] != -1) return memo[k][n];
        
        int lo = 1, hi = n;
        int res = Integer.MAX_VALUE;
        
        while (lo <= hi) {
            int mid = lo + (hi - lo) / 2;
            int broken = dfs(k - 1, mid - 1);
            int notBroken = dfs(k, n - mid);
            
            if (broken > notBroken) {
                hi = mid - 1;
                res = Math.min(res, broken + 1);
            } else {
                lo = mid + 1;
                res = Math.min(res, notBroken + 1);
            }
        }
        
        memo[k][n] = res;
        return res;
    }
}
```

**Key Insight:** Binary search on floor to minimize worst case. Partition DP on floors.

## 10. **Maximum Score from Multiplication (Leetcode 1770)**

**Instinct:**  
Array nums aur multipliers diye hain. Multiply karo, nums se ya toh left end ya right end pick karo. Max score.

**Approach:**  
1. Memo[i][j] = max score with i picks from left, j picks from right
2. But i+j = operation index
3. Better: Memo[op][left] = max score

```java
class Solution {
    int[] nums, multipliers;
    int n, m;
    Integer[][] memo;
    
    public int maximumScore(int[] nums, int[] multipliers) {
        this.nums = nums;
        this.multipliers = multipliers;
        n = nums.length;
        m = multipliers.length;
        memo = new Integer[m][m];
        
        return dfs(0, 0);
    }
    
    private int dfs(int op, int left) {
        if (op == m) return 0;
        if (memo[op][left] != null) return memo[op][left];
        
        int right = (n - 1) - (op - left);
        
        int pickLeft = nums[left] * multipliers[op] + dfs(op + 1, left + 1);
        int pickRight = nums[right] * multipliers[op] + dfs(op + 1, left);
        
        memo[op][left] = Math.max(pickLeft, pickRight);
        return memo[op][left];
    }
}
```

**Key Insight:** State = (operations done, picks from left). Right = n-1 - (op - left).

## 11. **Minimum Cost Tree From Leaf Values (Leetcode 1130)**

**Instinct:**  
Binary tree from leaf values, internal node value = product of max leaves in subtree. Min sum of internal nodes.

**Approach:**  
1. Memo[i][j] = min cost for leaves[i..j]
2. Try each partition point k
3. cost = max(i..k) * max(k+1..j) + leftCost + rightCost

```java
class Solution {
    int[] arr;
    int[][] memo;
    int[][] maxVal;
    
    public int mctFromLeafValues(int[] arr) {
        this.arr = arr;
        int n = arr.length;
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        // Precompute max in any range
        maxVal = new int[n][n];
        for (int i = 0; i < n; i++) {
            maxVal[i][i] = arr[i];
            for (int j = i + 1; j < n; j++) {
                maxVal[i][j] = Math.max(maxVal[i][j - 1], arr[j]);
            }
        }
        
        return dfs(0, n - 1);
    }
    
    private int dfs(int i, int j) {
        if (i >= j) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int minCost = Integer.MAX_VALUE;
        for (int k = i; k < j; k++) {
            int cost = maxVal[i][k] * maxVal[k + 1][j] + dfs(i, k) + dfs(k + 1, j);
            minCost = Math.min(minCost, cost);
        }
        
        memo[i][j] = minCost;
        return minCost;
    }
}
```

**Key Insight:** Partition DP with precomputed max values. Internal node cost = product of maxes.

## 12. **Maximum Profit in Job Scheduling (Leetcode 1235)**

**Instinct:**  
Jobs with start, end, profit. Max profit without overlapping. Sort by end time, then partition on non-conflicting jobs.

**Approach:**  
1. Sort jobs by end time
2. Memo[i] = max profit from index i
3. Either take current + next non-conflicting, or skip

```java
class Solution {
    int[][] jobs;
    int n;
    Integer[] memo;
    
    public int jobScheduling(int[] startTime, int[] endTime, int[] profit) {
        n = startTime.length;
        jobs = new int[n][3];
        for (int i = 0; i < n; i++) {
            jobs[i] = new int[]{startTime[i], endTime[i], profit[i]};
        }
        Arrays.sort(jobs, (a, b) -> a[1] - b[1]);
        
        memo = new Integer[n];
        return dfs(0);
    }
    
    private int dfs(int i) {
        if (i >= n) return 0;
        if (memo[i] != null) return memo[i];
        
        int skip = dfs(i + 1);
        
        int nextIndex = findNext(i);
        int take = jobs[i][2] + dfs(nextIndex);
        
        memo[i] = Math.max(skip, take);
        return memo[i];
    }
    
    private int findNext(int i) {
        int lo = i + 1, hi = n;
        while (lo < hi) {
            int mid = lo + (hi - lo) / 2;
            if (jobs[mid][0] >= jobs[i][1]) {
                hi = mid;
            } else {
                lo = mid + 1;
            }
        }
        return lo;
    }
}
```

**Key Insight:** Weighted interval scheduling = partition on next non-conflicting job.

## 13. **Minimum Difficulty of a Job Schedule (Leetcode 1335)**

**Instinct:**  
Jobs ko d days me partition karna hai. Har day ka difficulty = max job in that day. Min sum of difficulties.

**Approach:**  
1. Memo[i][d] = min difficulty from index i with d days left
2. Try each partition point j as end of current day

```java
class Solution {
    int[] jobDifficulty;
    int n;
    Integer[][] memo;
    
    public int minDifficulty(int[] jobDifficulty, int d) {
        this.jobDifficulty = jobDifficulty;
        n = jobDifficulty.length;
        if (n < d) return -1;
        
        memo = new Integer[n][d + 1];
        return dfs(0, d);
    }
    
    private int dfs(int i, int daysLeft) {
        if (i == n && daysLeft == 0) return 0;
        if (i == n || daysLeft == 0) return Integer.MAX_VALUE / 2;
        if (memo[i][daysLeft] != null) return memo[i][daysLeft];
        
        int maxSoFar = 0;
        int minDiff = Integer.MAX_VALUE / 2;
        
        for (int j = i; j <= n - daysLeft; j++) {
            maxSoFar = Math.max(maxSoFar, jobDifficulty[j]);
            int rest = dfs(j + 1, daysLeft - 1);
            minDiff = Math.min(minDiff, maxSoFar + rest);
        }
        
        memo[i][daysLeft] = minDiff;
        return minDiff;
    }
}
```

**Key Insight:** Partition jobs into d days. Each day's difficulty = max in that segment.

## 14. **Maximum Number of Events That Can Be Attended II (Leetcode 1751)**

**Instinct:**  
Events with start, end, value. Choose at most k events without overlap. Max value.

**Approach:**  
1. Sort by end time
2. Memo[i][k] = max value from index i with k events left
3. Take current + next non-conflicting OR skip

```java
class Solution {
    int[][] events;
    int n;
    Integer[][] memo;
    
    public int maxValue(int[][] events, int k) {
        this.events = events;
        n = events.length;
        Arrays.sort(events, (a, b) -> a[1] - b[1]);
        
        memo = new Integer[n][k + 1];
        return dfs(0, k);

    private int dfs(int i, int k) {
        if (i >= n || k == 0) return 0;
        if (memo[i][k] != null) return memo[i][k];
        
        int skip = dfs(i + 1, k);
        
        int nextIndex = findNext(i);
        int take = events[i][2] + dfs(nextIndex, k - 1);
        
        memo[i][k] = Math.max(skip, take);
        return memo[i][k];
    }
    
    private int findNext(int i) {
        int lo = i + 1, hi = n;
        while (lo < hi) {
            int mid = lo + (hi - lo) / 2;
            if (events[mid][0] > events[i][1]) {
                hi = mid;
            } else {
                lo = mid + 1;
            }
        }
        return lo;
    }
}
```

**Key Insight:** Weighted interval scheduling with at most k events.

## 15. **Number of Ways to Cut a Stick (Leetcode 1547 - Count ways)**

**Instinct:**  
Stick ko given cuts pe cut karna hai. Number of different ways to cut (order matters). Partition DP with counting.

**Approach:**  
1. Add 0 and n to cuts, sort
2. Memo[i][j] = number of ways to cut between cuts[i] and cuts[j]
3. Sum over all k as first cut

```java
class Solution {
    int MOD = 1_000_000_007;
    int[] cuts;
    Long[][] memo;
    
    public int numberOfWaysToCut(int n, int[] cuts) {
        int m = cuts.length;
        this.cuts = new int[m + 2];
        this.cuts[0] = 0;
        this.cuts[m + 1] = n;
        for (int i = 0; i < m; i++) {
            this.cuts[i + 1] = cuts[i];
        }
        Arrays.sort(this.cuts);
        
        memo = new Long[m + 2][m + 2];
        return (int)(dfs(0, m + 1) % MOD);
    }
    
    private long dfs(int i, int j) {
        if (i + 1 >= j) return 1;
        if (memo[i][j] != null) return memo[i][j];
        
        long ways = 0;
        for (int k = i + 1; k < j; k++) {
            ways = (ways + dfs(i, k) * dfs(k, j)) % MOD;
        }
        
        memo[i][j] = ways;
        return ways;
    }
}
```

**Key Insight:** Count ways = product of left and right ways. Sum over all first cuts.

## 16. **Maximum Sum of 3 Non-Overlapping Subarrays (Leetcode 689)**

**Instinct:**  
Array me 3 non-overlapping subarrays of length k. Max sum. Partition into three parts.

**Approach:**  
1. Prefix sum array
2. Memo[i][count] = max sum from i with count subarrays left
3. Either take subarray at i or skip

```java
class Solution {
    int[] nums;
    int k;
    int[] prefix;
    int[][] memo;
    int[] result;
    
    public int[] maxSumOfThreeSubarrays(int[] nums, int k) {
        this.nums = nums;
        this.k = k;
        int n = nums.length;
        
        prefix = new int[n + 1];
        for (int i = 0; i < n; i++) {
            prefix[i + 1] = prefix[i] + nums[i];
        }
        
        memo = new int[n][4];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        result = new int[3];
        dfs(0, 3);
        return result;
    }
    
    private int dfs(int i, int count) {
        if (count == 0) return 0;
        if (i > nums.length - k) return 0;
        if (memo[i][count] != -1) return memo[i][count];
        
        int takeSum = prefix[i + k] - prefix[i];
        int take = takeSum + dfs(i + k, count - 1);
        int skip = dfs(i + 1, count);
        
        if (take >= skip) {
            memo[i][count] = take;
            if (count == 3) result[3 - count] = i;
        } else {
            memo[i][count] = skip;
        }
        
        return memo[i][count];
    }
}
```

**Key Insight:** Partition array into non-overlapping subarrays of fixed length.

## 17. **Frog Jump (Leetcode 403) - Partition on jumps**

**Instinct:**  
Frog stones pe jump kar raha hai. Last jump k units tha toh next jump k-1, k, or k+1 units. Partition DP on positions.

**Approach:**  
1. Memo[i][k] = can frog reach end from stone i with last jump k
2. Try k-1, k, k+1 jumps

```java
class Solution {
    int[] stones;
    Map<Integer, Integer> indexMap;
    Map<String, Boolean> memo;
    
    public boolean canCross(int[] stones) {
        this.stones = stones;
        indexMap = new HashMap<>();
        for (int i = 0; i < stones.length; i++) {
            indexMap.put(stones[i], i);
        }
        memo = new HashMap<>();
        
        return dfs(0, 0);
    }
    
    private boolean dfs(int pos, int lastJump) {
        if (pos == stones.length - 1) return true;
        
        String key = pos + "#" + lastJump;
        if (memo.containsKey(key)) return memo.get(key);
        
        for (int jump = lastJump - 1; jump <= lastJump + 1; jump++) {
            if (jump <= 0) continue;
            int nextStone = stones[pos] + jump;
            if (indexMap.containsKey(nextStone)) {
                if (dfs(indexMap.get(nextStone), jump)) {
                    memo.put(key, true);
                    return true;
                }
            }
        }
        
        memo.put(key, false);
        return false;
    }
}
```

**Key Insight:** State = (position, last jump). Try all possible next jumps.

## 18. **Minimum Number of Removals to Make Mountain Array (Leetcode 1671)**

**Instinct:**  
Array ko mountain banane ke liye min removals. Mountain = increase then decrease. Partition at peak.

**Approach:**  
1. LIS from left and LIS from right
2. For each peak, mountain length = left[i] + right[i] - 1
3. Answer = n - max mountain length

```java
class Solution {
    public int minimumMountainRemovals(int[] nums) {
        int n = nums.length;
        int[] left = new int[n];
        int[] right = new int[n];
        
        // LIS from left
        Arrays.fill(left, 1);
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < i; j++) {
                if (nums[j] < nums[i]) {
                    left[i] = Math.max(left[i], left[j] + 1);
                }
            }
        }
        
        // LIS from right (LDS)
        Arrays.fill(right, 1);
        for (int i = n - 1; i >= 0; i--) {
            for (int j = n - 1; j > i; j--) {
                if (nums[j] < nums[i]) {
                    right[i] = Math.max(right[i], right[j] + 1);
                }
            }
        }
        
        int maxMountain = 0;
        for (int i = 1; i < n - 1; i++) {
            if (left[i] > 1 && right[i] > 1) {
                maxMountain = Math.max(maxMountain, left[i] + right[i] - 1);
            }
        }
        
        return n - maxMountain;
    }
}
```

**Key Insight:** Mountain = LIS from left + LIS from right - 1. Partition at peak.

## 19. **Maximum Score from Grid (Leetcode 2008 - Partition on operations)**

**Instinct:**  
Grid me max score with k operations. Partition on which rows/columns to pick.

**Approach:**  
1. Memo[i][k] = max score from row i with k operations left
2. Try all possible picks in current row

```java
class Solution {
    int[][] grid;
    int m, n;
    int[][] memo;
    
    public int maxScoreFromGrid(int[][] grid, int k) {
        this.grid = grid;
        m = grid.length;
        n = grid[0].length;
        memo = new int[m][k + 1];
        for (int i = 0; i < m; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        return dfs(0, k);
    }
    
    private int dfs(int row, int k) {
        if (row >= m || k == 0) return 0;
        if (memo[row][k] != -1) return memo[row][k];
        
        int max = 0;
        // Pick some columns in current row
        for (int pick = 0; pick <= Math.min(n, k); pick++) {
            int sum = getMaxSumInRow(row, pick);
            max = Math.max(max, sum + dfs(row + 1, k - pick));
        }
        
        memo[row][k] = max;
        return max;
    }
    
    private int getMaxSumInRow(int row, int pick) {
        // Return max sum by picking 'pick' elements from row
        // This could be greedy or DP
        int[] sorted = Arrays.copyOf(grid[row], n);
        Arrays.sort(sorted);
        int sum = 0;
        for (int i = n - 1; i >= n - pick; i--) {
            sum += sorted[i];
        }
        return sum;
    }
}
```

**Key Insight:** Partition operations across rows. Each row me pick k elements.

## 20. **Ways to Split Array into K Subarrays with Equal Sum**

**Instinct:**  
Array ko k parts me divide karna hai with equal sum. Each part ka sum = total/k. Partition DP with prefix sums.

**Approach:**  
1. Check if total % k == 0
2. target = total / k
3. Memo[i][k] = ways to split from i into k parts
4. Try all j where sum(i..j) == target

```java
class Solution {
    int[] nums;
    int target;
    int[][] memo;
    
    public int waysToSplit(int[] nums, int k) {
        this.nums = nums;
        int total = 0;
        for (int num : nums) total += num;
        if (total % k != 0) return 0;
        
        target = total / k;
        int n = nums.length;
        memo = new int[n][k + 1];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        return dfs(0, k);
    }
    
    private int dfs(int i, int k) {
        if (i == nums.length && k == 0) return 1;
        if (i == nums.length || k == 0) return 0;
        if (memo[i][k] != -1) return memo[i][k];
        
        int sum = 0;
        int ways = 0;
        for (int j = i; j < nums.length; j++) {
            sum += nums[j];
            if (sum == target) {
                ways += dfs(j + 1, k - 1);
            }
        }
        
        memo[i][k] = ways;
        return ways;
    }
}
```

**Key Insight:** Partition array into k parts with equal sum. Use prefix sums.

## 🎯 Key Takeaways for Partition DP Pattern

### 1. **Core Structure for Interval DP:**
```java
dfs(i, j) {
    if (i >= j) return base;
    for (int k = i; k <= j; k++) {
        cost = left + right + combine(i, k, j);
        best = min/max(best, cost);
    }
    return best;
}
```

### 2. **Common Recurrence Patterns:**

| Problem Type | Recurrence |
|-------------|------------|
| Matrix Chain | `dp[i][j] = min(dp[i][k] + dp[k+1][j] + dims[i-1]*dims[k]*dims[j])` |
| Burst Balloons | `dp[i][j] = max(dp[i][k-1] + dp[k+1][j] + arr[i-1]*arr[k]*arr[j+1])` |
| Cut Stick | `dp[i][j] = min(dp[i][k] + dp[k][j] + cuts[j] - cuts[i])` |
| Min-Max Game | `dp[i][j] = max(nums[i] - dp[i+1][j], nums[j] - dp[i][j-1])` |

### 3. **State Space Patterns:**

```
2D Interval DP: (i, j)                    - O(n²)
With extra dimension: (i, j, k)           - O(n²k)
1D DP on start: (i)                       - O(n)
With count: (i, count)                    - O(nk)
```

### 4. **When to use Partition DP:**
- Array/string ko parts me divide karna hai
- Order of operations matters
- Need to try all possible partition points
- Problem has optimal substructure with partitions

### 5. **Optimization Techniques:**
- **Knuth Optimization**: For quadrangle inequality (MCM)
- **Divide and Conquer DP**: For certain cost functions
- **Binary Search**: For egg drop type problems
- **Prefix Sums**: For range sum queries

### 6. **Common Partition DP Variants:**

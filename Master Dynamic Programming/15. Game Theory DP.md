

# Game Theory DP Pattern 

## Game Theory DP ke 10 Main Variants:

| Variant | Description | Approach |
|---------|-------------|----------|
| **Take from Ends** | Players pick from left/right ends | Max difference = pick - opponent's best |
| **Take from Anywhere** | Players pick any remaining element | DP on remaining set (bitmask) |
| **Predict the Winner** | Can first player win? | Difference >= 0 |
| **Stone Games** | Various stone picking rules | State = (i, j) or (i, M) |
| **Nim Games** | Remove stones from piles | XOR theory / DP on piles |
| **Optimal Play with Multipliers** | Multiply and pick | DP with operation index |
| **Card/Coin Games** | Pick from ends with multipliers | DP with left/right pointers |
| **Geography/Graph Games** | Move on graph | DP[node] with state |
| **Partition Games** | Split into parts | Interval DP |
| **Chocolate/Chalk Games** | Take 1 to K stones | DP[n] with move options |

## 1. **Predict the Winner (Leetcode 486)**

**Instinct:**  
Two players pick numbers from ends. Each wants to maximize sum. Current player's advantage = pick - opponent's best from remaining.

**Approach:**  
1. Memo[i][j] = max difference current player can get from nums[i..j]
2. Base: i == j → nums[i]
3. Pick left: nums[i] - dfs(i+1, j)
4. Pick right: nums[j] - dfs(i, j-1)
5. Return max of both

```java
class Solution {
    int[] nums;
    int[][] memo;
    
    public boolean predictTheWinner(int[] nums) {
        this.nums = nums;
        int n = nums.length;
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        int diff = dfs(0, n - 1);
        return diff >= 0;
    }
    
    private int dfs(int i, int j) {
        if (i > j) return 0;
        if (i == j) return nums[i];
        if (memo[i][j] != -1) return memo[i][j];
        
        int pickLeft = nums[i] - dfs(i + 1, j);
        int pickRight = nums[j] - dfs(i, j - 1);
        
        memo[i][j] = Math.max(pickLeft, pickRight);
        return memo[i][j];
    }
}
```

**Key Insight:** Min-max DP. Current player's advantage = pick - opponent's advantage.

## 2. **Stone Game (Leetcode 877)**

**Instinct:**  
Same as Predict the Winner but array length even. First player always wins.

**Approach:**  
Same DP, but return true (always win) or compute diff.

```java
class Solution {

    int[][] memo;
    
    public boolean stoneGame(int[] piles) {
        this.piles = piles;
        int n = piles.length;
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        int diff = dfs(0, n - 1);
        return diff > 0;
    }
    
    private int dfs(int i, int j) {
        if (i > j) return 0;
        if (i == j) return piles[i];
        if (memo[i][j] != -1) return memo[i][j];
        
        int pickLeft = piles[i] - dfs(i + 1, j);
        int pickRight = piles[j] - dfs(i, j - 1);
        
        memo[i][j] = Math.max(pickLeft, pickRight);
        return memo[i][j];
    }
}
```

**Key Insight:** Since piles.length is even, first player can always force a win.

## 3. **Stone Game II (Leetcode 1140)**

**Instinct:**  
Players can take 1 to 2M stones from start, where M = max(M, taken). State = (index, M).

**Approach:**  
1. Memo[i][M] = max stones current player can get from index i with M
2. Suffix sum for quick range sum
3. Try taking x = 1 to 2M stones
4. Stones taken = sum[i..i+x-1] + (total - sum[i..i+x-1] - opponent's best)

```java
class Solution {
    int[] piles;
    int[] suffixSum;
    int[][] memo;
    
    public int stoneGameII(int[] piles) {
        this.piles = piles;
        int n = piles.length;
        suffixSum = new int[n + 1];
        for (int i = n - 1; i >= 0; i--) {
            suffixSum[i] = suffixSum[i + 1] + piles[i];
        }
        
        memo = new int[n][n + 1];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        return dfs(0, 1);
    }
    
    private int dfs(int i, int M) {
        if (i >= piles.length) return 0;
        if (memo[i][M] != -1) return memo[i][M];
        
        int maxStones = 0;
        for (int x = 1; x <= 2 * M; x++) {
            if (i + x > piles.length) break;
            int opponent = dfs(i + x, Math.max(M, x));
            maxStones = Math.max(maxStones, suffixSum[i] - opponent);
        }
        
        memo[i][M] = maxStones;
        return maxStones;
    }
}
```

**Key Insight:** M updates to max(M, x). Suffix sum for O(1) range sum.

## 4. **Stone Game III (Leetcode 1406)**

**Instinct:**  
Players can take 1, 2, or 3 stones from start. State = (index).

**Approach:**  
1. Memo[i] = max score difference from index i
2. Try taking 1, 2, or 3 stones
3. Score = sum of taken - opponent's best

```java
class Solution {
    int[] stoneValue;
    int[] memo;
    int[] suffixSum;
    
    public String stoneGameIII(int[] stoneValue) {
        this.stoneValue = stoneValue;
        int n = stoneValue.length;
        suffixSum = new int[n + 1];
        for (int i = n - 1; i >= 0; i--) {
            suffixSum[i] = suffixSum[i + 1] + stoneValue[i];
        }
        
        memo = new int[n];
        Arrays.fill(memo, -1);
        
        int diff = dfs(0);
        if (diff > 0) return "Alice";
        if (diff < 0) return "Bob";
        return "Tie";
    }
    
    private int dfs(int i) {
        if (i >= stoneValue.length) return 0;
        if (memo[i] != -1) return memo[i];
        
        int maxDiff = Integer.MIN_VALUE;
        int sum = 0;
        
        for (int x = 1; x <= 3; x++) {
            if (i + x - 1 >= stoneValue.length) break;
            sum += stoneValue[i + x - 1];
            int diff = sum - dfs(i + x);
            maxDiff = Math.max(maxDiff, diff);
        }
        
        memo[i] = maxDiff;
        return maxDiff;
    }
}
```

**Key Insight:** Take 1, 2, or 3 stones. Diff = sum taken - opponent's best.

## 5. **Stone Game IV (Leetcode 1510)**

**Instinct:**  
Remove square numbers of stones. State = (n) = current player win or lose.

**Approach:**  
1. Memo[n] = can current player win with n stones
2. Try removing all square numbers <= n
3. If any move leads to opponent losing, current wins

```java
class Solution {
    Boolean[] memo;
    
    public boolean winnerSquareGame(int n) {
        memo = new Boolean[n + 1];
        return dfs(n);
    }
    
    private boolean dfs(int n) {
        if (n == 0) return false;
        if (memo[n] != null) return memo[n];
        
        for (int i = 1; i * i <= n; i++) {
            if (!dfs(n - i * i)) {
                memo[n] = true;
                return true;
            }
        }
        
        memo[n] = false;
        return false;
    }
}
```

**Key Insight:** Remove square numbers. If any move leads to opponent losing, current wins.

## 6. **Stone Game V (Leetcode 1563)**

**Instinct:**  
Split array into two parts, take larger sum side. State = (i, j).

**Approach:**  
1. Memo[i][j] = max score from stone[i..j]
2. Try all split points k
3. If left sum < right sum, take left; if right < left, take right; if equal, take max of both

```java
class Solution {
    int[] stoneValue;
    int[] prefix;
    int[][] memo;
    
    public int stoneGameV(int[] stoneValue) {
        this.stoneValue = stoneValue;
        int n = stoneValue.length;
        prefix = new int[n + 1];
        for (int i = 0; i < n; i++) {
            prefix[i + 1] = prefix[i] + stoneValue[i];
        }
        
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        return dfs(0, n - 1);
    }
    
    private int dfs(int i, int j) {
        if (i == j) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int maxScore = 0;
        for (int k = i; k < j; k++) {
            int leftSum = prefix[k + 1] - prefix[i];
            int rightSum = prefix[j + 1] - prefix[k + 1];
            
            if (leftSum < rightSum) {
                maxScore = Math.max(maxScore, leftSum + dfs(i, k));
            } else if (rightSum < leftSum) {
                maxScore = Math.max(maxScore, rightSum + dfs(k + 1, j));
            } else {
                maxScore = Math.max(maxScore, leftSum + Math.max(dfs(i, k), dfs(k + 1, j)));
            }
        }
        
        memo[i][j] = maxScore;
        return maxScore;
    }
}
```

**Key Insight:** Split at k. Take smaller sum side; if equal, take max of both.

## 7. **Stone Game VI (Leetcode 1686)**

**Instinct:**  
Each stone has value for Alice and Bob. Players take turns picking stones. Max total difference.

**Approach:**  
1. Sort stones by (aliceValue + bobValue) descending
2. Greedy: each player picks stone with highest sum

```java
class Solution {
    public int stoneGameVI(int[] aliceValues, int[] bobValues) {
        int n = aliceValues.length;
        Integer[] indices = new Integer[n];
        for (int i = 0; i < n; i++) indices[i] = i;
        
        Arrays.sort(indices, (a, b) -> 
            (aliceValues[b] + bobValues[b]) - (aliceValues[a] + bobValues[a]));
        
        int aliceScore = 0, bobScore = 0;
        for (int i = 0; i < n; i++) {
            if (i % 2 == 0) {
                aliceScore += aliceValues[indices[i]];
            } else {
                bobScore += bobValues[indices[i]];
            }
        }
        
        return Integer.compare(aliceScore, bobScore);
    }
}
```

**Key Insight:** Greedy by sum of values. Not DP, but classic game theory.

## 8. **Stone Game VII (Leetcode 1690)**

**Instinct:**  
Remove from ends, score = sum of remaining after removal. State = (i, j).

**Approach:**  
1. Memo[i][j] = max score difference
2. Remove left: sum(i+1, j) - dfs(i+1, j)
3. Remove right: sum(i, j-1) - dfs(i, j-1)

```java
class Solution {
    int[] stones;
    int[] prefix;
    int[][] memo;
    
    public int stoneGameVII(int[] stones) {
        this.stones = stones;
        int n = stones.length;
        prefix = new int[n + 1];
        for (int i = 0; i < n; i++) {
            prefix[i + 1] = prefix[i] + stones[i];
        }
        
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        return dfs(0, n - 1);
    }
    
    private int dfs(int i, int j) {
        if (i == j) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int sum = prefix[j + 1] - prefix[i];
        int removeLeft = (sum - stones[i]) - dfs(i + 1, j);
        int removeRight = (sum - stones[j]) - dfs(i, j - 1);
        
        memo[i][j] = Math.max(removeLeft, removeRight);
        return memo[i][j];
    }
}
```

**Key Insight:** Score = sum of remaining after removal. Difference based DP.

## 9. **Stone Game VIII (Leetcode 1872)**

**Instinct:**  
Prefix sums, players take first X stones, then game continues. State = (index, turn).

**Approach:**  
1. Memo[i] = max difference from index i
2. Try taking k stones from start

```java
class Solution {
    int[] stones;
    int[] prefix;
    int[] memo;
    
    public int stoneGameVIII(int[] stones) {
        this.stones = stones;
        int n = stones.length;
        prefix = new int[n + 1];
        for (int i = 0; i < n; i++) {
            prefix[i + 1] = prefix[i] + stones[i];
        }
        
        memo = new int[n];
        Arrays.fill(memo, -1);
        
        return dfs(1);
    }
    
    private int dfs(int i) {
        if (i == stones.length) return 0;
        if (memo[i] != -1) return memo[i];
        
        int maxDiff = Integer.MIN_VALUE;
        for (int k = i; k < stones.length; k++) {
            int diff = prefix[k + 1] - dfs(k + 1);
            maxDiff = Math.max(maxDiff, diff);
        }
        
        memo[i] = maxDiff;
        return maxDiff;
    }
}
```

**Key Insight:** Players take prefix sum. Next turn starts at next index.

## 10. **Can I Win (Leetcode 464)**

**Instinct:**  
Players pick numbers 1 to maxChoosableInteger. First to reach target sum wins. State = (used mask, current sum).

**Approach:**  
1. Memo[mask] = can current player win
2. Try all unused numbers
3. If number + sum >= target → win
4. Else if opponent loses with new mask → win

```java
class Solution {
    int maxInt;
    int target;
    Boolean[] memo;
    
    public boolean canIWin(int maxChoosableInteger, int desiredTotal) {
        if (desiredTotal <= 0) return true;
        if (maxChoosableInteger * (maxChoosableInteger + 1) / 2 < desiredTotal) return false;
        
        this.maxInt = maxChoosableInteger;
        this.target = desiredTotal;
        memo = new Boolean[1 << maxInt];
        
        return dfs(0, 0);
    }
    
    private boolean dfs(int mask, int sum) {
        if (sum >= target) return false;
        if (memo[mask] != null) return memo[mask];
        
        for (int i = 1; i <= maxInt; i++) {
            int bit = 1 << (i - 1);
            if ((mask & bit) != 0) continue;
            
            if (sum + i >= target || !dfs(mask | bit, sum + i)) {
                memo[mask] = true;
                return true;
            }
        }
        
        memo[mask] = false;
        return false;
    }
}
```

**Key Insight:** Bitmask tracks used numbers. If any move leads to opponent losing, win.

## 11. **Divisor Game (Leetcode 1025)**

**Instinct:**  
Choose divisor of N, subtract from N. State = (N) = current player win or lose.

**Approach:**  
1. Memo[n] = can current player win with n
2. Try all divisors d < n where n % d == 0
3. If any move leads to opponent losing, win

```java
class Solution {
    Boolean[] memo;
    
    public boolean divisorGame(int n) {
        memo = new Boolean[n + 1];
        return dfs(n);
    }
    
    private boolean dfs(int n) {
        if (n == 1) return false;
        if (memo[n] != null) return memo[n];
        
        for (int d = 1; d < n; d++) {
            if (n % d == 0 && !dfs(n - d)) {
                memo[n] = true;
                return true;
            }
        }
        
        memo[n] = false;
        return false;
    }
}
```

**Key Insight:** Mathematical solution: Alice wins if n is even. DP works too.

## 12. **Nim Game (Leetcode 292)**

**Instinct:**  
Remove 1, 2, or 3 stones. Last to move wins. State = (n).

**Approach:**  
1. Memo[n] = can current player win
2. Try removing 1, 2, or 3 stones
3. If any move leads to opponent losing, win

```java
class Solution {

        if (n <= 0) return false;
        if (memo[n] != null) return memo[n];

                return true;
            }
        }
        
        memo[n] = false;
        return false;
    }
}
```

**Key Insight:** Mathematical solution: n % 4 != 0. DP works for small n.

## 14. **Maximum Score from Multiplication (Leetcode 1770)**

**Instinct:**  
Pick from left or right end, multiply with multiplier. State = (op, left).

**Approach:**  
1. Memo[op][left] = max score with op operations done, left picks
2. Right = n - 1 - (op - left)
3. Pick left or right

```java
class Solution {
    int[] nums, multipliers;
    int n, m;
    Integer[][] memo;
    
    public int maximumScore(int[] nums, int[] multipliers) {
        this.nums = nums;
        this.multipliers = multipliers;
        n = nums.length;
        m = multipliers.length;
        memo = new Integer[m][m];
        
        return dfs(0, 0);
    }
    
    private int dfs(int op, int left) {
        if (op == m) return 0;
        if (memo[op][left] != null) return memo[op][left];
        
        int right = (n - 1) - (op - left);
        
        int pickLeft = nums[left] * multipliers[op] + dfs(op + 1, left + 1);
        int pickRight = nums[right] * multipliers[op] + dfs(op + 1, left);
        
        memo[op][left] = Math.max(pickLeft, pickRight);
        return memo[op][left];
    }
}
```

**Key Insight:** State = (operations done, picks from left). Right derived.

## 15. **Maximum Points You Can Obtain from Cards (Leetcode 1423)**

**Instinct:**  
Pick k cards from either ends. Max sum. Equivalent to min sum of remaining n-k cards.

**Approach:**  
1. Prefix sum
2. Answer = total - min sum of subarray of length n-k

```java
class Solution {
    public int maxScore(int[] cardPoints, int k) {
        int n = cardPoints.length;
        int total = 0;
        for (int p : cardPoints) total += p;
        
        if (k == n) return total;
        
        int windowSize = n - k;
        int windowSum = 0;
        for (int i = 0; i < windowSize; i++) {
            windowSum += cardPoints[i];
        }
        
        int minWindow = windowSum;
        for (int i = windowSize; i < n; i++) {
            windowSum += cardPoints[i] - cardPoints[i - windowSize];
            minWindow = Math.min(minWindow, windowSum);
        }
        
        return total - minWindow;
    }
}
```

**Key Insight:** Sliding window, not DP. Included for completeness.

## 16. **Maximum Score from Grid (Leetcode 2008 variant)**

**Instinct:**  
Pick from rows with constraints. State = (row, picks left).

**Approach:**  
1. Memo[row][picks] = max score
2. Try picking different numbers from current row

```java
class Solution {
    int[][] grid;
    int m, n;
    int[][] memo;
    
    public int maxScoreFromGrid(int[][] grid, int k) {
        this.grid = grid;
        m = grid.length;
        n = grid[0].length;
        memo = new int[m][k + 1];
        for (int i = 0; i < m; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        return dfs(0, k);
    }
    
    private int dfs(int row, int picks) {
        if (row == m || picks == 0) return 0;
        if (memo[row][picks] != -1) return memo[row][picks];
        
        int max = 0;
        for (int take = 0; take <= Math.min(n, picks); take++) {
            int sum = getMaxSumInRow(row, take);
            max = Math.max(max, sum + dfs(row + 1, picks - take));
        }
        
        memo[row][picks] = max;
        return max;
    }
    
    private int getMaxSumInRow(int row, int take) {
        int[] sorted = Arrays.copyOf(grid[row], n);
        Arrays.sort(sorted);
        int sum = 0;
        for (int i = n - 1; i >= n - take; i--) {
            sum += sorted[i];
        }
        return sum;
    }
}
```

**Key Insight:** Partition picks across rows. Each row take top elements.

## 17. **Alice and Bob Play (Picking Stones Variation)**

**Instinct:**  
Pick from ends, Alice starts. State = (i, j, turn).

**Approach:**  
1. Memo[i][j][turn] = max score for current player
2. Turn 0 = Alice, 1 = Bob
3. Alice maximizes, Bob minimizes

```java
class Solution {
    int[] stones;
    int[][][] memo;
    
    public int stoneGameWithTurn(int[] stones) {
        this.stones = stones;
        int n = stones.length;
        memo = new int[n][n][2];
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        
        return dfs(0, n - 1, 0);
    }
    
    private int dfs(int i, int j, int turn) {
        if (i > j) return 0;
        if (memo[i][j][turn] != -1) return memo[i][j][turn];
        
        if (turn == 0) { // Alice - maximize
            int pickLeft = stones[i] + dfs(i + 1, j, 1);
            int pickRight = stones[j] + dfs(i, j - 1, 1);
            memo[i][j][turn] = Math.max(pickLeft, pickRight);
        } else { // Bob - minimize (or maximize his own)
            int pickLeft = dfs(i + 1, j, 0);
            int pickRight = dfs(i, j - 1, 0);
            memo[i][j][turn] = Math.min(pickLeft, pickRight);
        }
        
        return memo[i][j][turn];
    }
}
```

**Key Insight:** Turn-based DP. Alice maximizes, Bob minimizes.

## 18. **Number of Ways to Win (Counting Winning Moves)**

**Instinct:**  
Count number of ways current player can win. State = (mask, sum).

**Approach:**  
1. Memo[mask][sum] = number of winning ways
2. Try all moves
3. If move wins directly, add 1
4. Else if opponent loses, add opponent's losing count

```java
class Solution {
    int maxInt;
    int target;
    int[][] memo;
    
    public int countWaysToWin(int maxChoosableInteger, int desiredTotal) {
        if (desiredTotal <= 0) return 1;
        if (maxChoosableInteger * (maxChoosableInteger + 1) / 2 < desiredTotal) return 0;
        
        this.maxInt = maxChoosableInteger;
        this.target = desiredTotal;
        memo = new int[1 << maxInt][target + 1];
        for (int i = 0; i < (1 << maxInt); i++) {
            Arrays.fill(memo[i], -1);
        }
        
        return dfs(0, 0);
    }
    
    private int dfs(int mask, int sum) {
        if (sum >= target) return 0;
        if (memo[mask][sum] != -1) return memo[mask][sum];
        
        int ways = 0;
        for (int i = 1; i <= maxInt; i++) {
            int bit = 1 << (i - 1);
            if ((mask & bit) != 0) continue;
            
            if (sum + i >= target) {
                ways++;
            } else {
                int opponentWays = dfs(mask | bit, sum + i);
                ways += (1 << (maxInt - Integer.bitCount(mask) - 1)) - opponentWays;
            }
        }
        
        memo[mask][sum] = ways;
        return ways;
    }
}
```

**Key Insight:** Count winning moves = total possible - losing moves.

## 19. **Optimal Strategy for a Game (Pick from ends)**

**Instinct:**  
Classic optimal strategy game. State = (i, j).

**Approach:**  
1. Memo[i][j] = max score for current player
2. Pick left or right
3. Score = pick + min(opponent's best from remaining)

```java
class Solution {
    int[] arr;
    int[][] memo;
    
    public int optimalStrategy(int[] arr) {
        this.arr = arr;
        int n = arr.length;
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        return dfs(0, n - 1);
    }
    
    private int dfs(int i, int j) {
        if (i > j) return 0;
        if (i == j) return arr[i];
        if (memo[i][j] != -1) return memo[i][j];
        
        int pickLeft = arr[i] + Math.min(dfs(i + 2, j), dfs(i + 1, j - 1));
        int pickRight = arr[j] + Math.min(dfs(i, j - 2), dfs(i + 1, j - 1));
        
        memo[i][j] = Math.max(pickLeft, pickRight);
        return memo[i][j];
    }
}
```

**Key Insight:** After picking, opponent will choose optimally to minimize your score.

## 20. **Miserable Game (Minimize Score)**

**Instinct:**  
Current player wants to minimize final score. Min-max DP with minimization.

**Approach:**  
1. Memo[i][j] = min score current player can force
2. Pick left or right
3. Score = pick + max(opponent's best) because opponent wants to maximize

```java
class Solution {
    int[] arr;
    int[][] memo;
    
    public int miserableGame(int[] arr) {
        this.arr = arr;
        int n = arr.length;
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        return dfs(0, n - 1);
    }
    
    private int dfs(int i, int j) {
        if (i > j) return 0;
        if (i == j) return arr[i];
        if (memo[i][j] != -1) return memo[i][j];
        
        int pickLeft = arr[i] + Math.max(dfs(i + 2, j), dfs(i + 1, j - 1));
        int pickRight = arr[j] + Math.max(dfs(i, j - 2), dfs(i + 1, j - 1));
        
        memo[i][j] = Math.min(pickLeft, pickRight);
        return memo[i][j];
    }
}
```

**Key Insight:** Minimizer picks, then opponent maximizes.

## 🎯 Key Takeaways for Game Theory DP Pattern

### 1. **Core Game Theory DP Structure:**

**For Maximizing Difference (Standard):**
```java
dfs(i, j) {
    if (i > j) return 0;
    if (i == j) return nums[i];
    
    int pickLeft = nums[i] - dfs(i+1, j);
    int pickRight = nums[j] - dfs(i, j-1);
    
    return Math.max(pickLeft, pickRight);
}
```

**For Maximizing Score (with opponent minimizing):**
```java
dfs(i, j) {
    if (i > j) return 0;
    if (i == j) return nums[i];
    
    int pickLeft = nums[i] + Math.min(dfs(i+2, j), dfs(i+1, j-1));
    int pickRight = nums[j] + Math.min(dfs(i, j-2), dfs(i+1, j-1));
    
    return Math.max(pickLeft, pickRight);
}
```

### 2. **Common Game States:**

| Game Type | State | Transition |
|-----------|-------|------------|
| Ends Pick | `(i, j)` | pick left or right |
| Take 1-K stones | `(i)` | take 1 to K stones |
| Take with M | `(i, M)` | take 1 to 2M stones |
| Bitmask | `(mask, sum)` | try unused numbers |
| Turn-based | `(i, j, turn)` | alternate turns |

### 3. **When to use Game Theory DP:**
- Two players alternate turns
- Both play optimally
- Need to determine winner or optimal score
- Moves affect remaining game state
- Zero-sum game

### 4. **Key Formulas:**

| Operation | Formula |
|-----------|---------|
| Difference (ends) | `max(nums[i] - dp[i+1][j], nums[j] - dp[i][j-1])` |
| Score (ends) | `max(nums[i] + min(dp[i+2][j], dp[i+1][j-1]), nums[j] + min(dp[i][j-2], dp[i+1][j-1]))` |
| Can Win | `if (any move leads to opponent losing) return true` |
| Nim Game | `n % (maxTake+1) != 0` |

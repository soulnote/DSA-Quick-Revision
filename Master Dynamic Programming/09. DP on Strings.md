
# 📝 DP on Strings Pattern 

## String DP ke 10 Main Variants:

| Variant | Description | Approach |
|---------|-------------|----------|
| **Edit Distance** | Insert/delete/replace operations | 2D DP on indices |
| **Longest Common Subsequence (LCS)** | Longest common subsequence | Match/mismatch on chars |
| **Longest Common Substring** | Contiguous common part | Break on mismatch |
| **Shortest Common Supersequence** | Shortest string containing both | m + n - LCS |
| **Distinct Subsequences** | Count of s2 in s1 as subsequence | Take/skip counting |
| **Wildcard Matching** | Pattern with * and ? | DP with pattern matching |
| **Regular Expression Matching** | Pattern with . and * | Handle * on prev char |
| **Interleaving Strings** | s3 formed by interleaving s1,s2 | 2D DP on indices |
| **Minimum Window Subsequence** | Shortest substring containing subsequence | Two pointers + DP |
| **Palindromic Substrings** | Count palindromic substrings | Expand around center |

## 1. **Edit Distance (Leetcode 72)**

**Instinct:**  
Word1 ko Word2 me convert karne ke liye minimum operations (insert, delete, replace). Har character match ho ya nahi, teen options hain - insert, delete, replace.

**Approach:**  
1. Memo[i][j] = min operations for word1[i..] and word2[j..]
2. Base: i == n → remaining chars in word2 insert karo (m-j)
3. Base: j == m → remaining chars in word1 delete karo (n-i)
4. If chars match → dfs(i+1, j+1)
5. Else → 1 + min(insert, delete, replace)

```java
class Solution {
    String w1, w2;
    int[][] memo;
    
    public int minDistance(String word1, String word2) {
        this.w1 = word1;
        this.w2 = word2;
        int n = w1.length(), m = w2.length();
        memo = new int[n][m];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        return dfs(0, 0);
    }
    
    private int dfs(int i, int j) {
        if (i == w1.length()) return w2.length() - j; // insert all remaining
        if (j == w2.length()) return w1.length() - i; // delete all remaining
        if (memo[i][j] != -1) return memo[i][j];
        
        int res;
        if (w1.charAt(i) == w2.charAt(j)) {
            res = dfs(i + 1, j + 1);
        } else {
            int insert = dfs(i, j + 1);
            int delete = dfs(i + 1, j);
            int replace = dfs(i + 1, j + 1);
            res = 1 + Math.min(insert, Math.min(delete, replace));
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** Three operations - insert, delete, replace. Base cases handle remaining chars.

## 2. **Longest Common Subsequence (Leetcode 1143)**

**Instinct:**  
Do strings ka longest common subsequence (non-contiguous). Order preserve karte hue common characters nikaalne hain.

**Approach:**  
1. Memo[i][j] = LCS length for text1[i..] and text2[j..]
2. Base: i == n or j == m → 0
3. If chars match → 1 + dfs(i+1, j+1)
4. Else → max(dfs(i+1, j), dfs(i, j+1))

```java
class Solution {
    String s1, s2;
    int[][] memo;
    
    public int longestCommonSubsequence(String text1, String text2) {
        this.s1 = text1;
        this.s2 = text2;
        int n = s1.length(), m = s2.length();
        memo = new int[n][m];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        return dfs(0, 0);
    }
    
    private int dfs(int i, int j) {
        if (i == s1.length() || j == s2.length()) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int res;
        if (s1.charAt(i) == s2.charAt(j)) {
            res = 1 + dfs(i + 1, j + 1);
        } else {
            res = Math.max(dfs(i + 1, j), dfs(i, j + 1));
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** Classic LCS - match diagonal, mismatch max(up, left).

## 3. **Longest Common Substring (Contiguous)**

**Instinct:**  
Do strings ka longest common substring (contiguous). Agar mismatch ho toh chain toot jaati hai.

**Approach:**  
1. Memo[i][j] = length of common substring ending at i in s1 and j in s2
2. If chars match → 1 + dfs(i-1, j-1)
3. Else → 0
4. Track max during recursion

```java
class Solution {
    String s1, s2;
    int[][] memo;
    int maxLen = 0;
    
    public int longestCommonSubstring(String text1, String text2) {
        this.s1 = text1;
        this.s2 = text2;
        int n = s1.length(), m = s2.length();
        memo = new int[n][m];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                maxLen = Math.max(maxLen, dfs(i, j));
            }
        }
        return maxLen;
    }
    
    private int dfs(int i, int j) {
        if (i < 0 || j < 0) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int res = 0;
        if (s1.charAt(i) == s2.charAt(j)) {
            res = 1 + dfs(i - 1, j - 1);
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** Substring = contiguous. Mismatch pe chain break (return 0).

## 4. **Shortest Common Supersequence (Leetcode 1092)**

**Instinct:**  
Do strings ko merge karna hai such that both are subsequences. Shortest length = m + n - LCS. Print bhi karna hai.

**Approach:**  
1. First find LCS using DP
2. Build supersequence by traversing backwards
3. Agar chars match → include once
4. Agar nahi match → include char from string with larger remaining

```java
class Solution {
    String s1, s2;
    int[][] memo;
    
    public String shortestCommonSupersequence(String str1, String str2) {
        this.s1 = str1;
        this.s2 = str2;
        int n = s1.length(), m = s2.length();
        memo = new int[n][m];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        dfs(0, 0);
        
        StringBuilder sb = new StringBuilder();
        int i = 0, j = 0;
        while (i < n && j < m) {
            if (s1.charAt(i) == s2.charAt(j)) {
                sb.append(s1.charAt(i));
                i++;
                j++;
            } else {
                int down = (i + 1 < n) ? memo[i + 1][j] : 0;
                int right = (j + 1 < m) ? memo[i][j + 1] : 0;
                if (down >= right) {
                    sb.append(s1.charAt(i));
                    i++;
                } else {
                    sb.append(s2.charAt(j));
                    j++;
                }
            }
        }
        
        while (i < n) sb.append(s1.charAt(i++));
        while (j < m) sb.append(s2.charAt(j++));
        
        return sb.toString();
    }
    
    private int dfs(int i, int j) {
        if (i == s1.length() || j == s2.length()) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int res;
        if (s1.charAt(i) == s2.charAt(j)) {
            res = 1 + dfs(i + 1, j + 1);
        } else {
            res = Math.max(dfs(i + 1, j), dfs(i, j + 1));
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** SCS length = m + n - LCS. Reconstruction uses DP table.

## 5. **Distinct Subsequences (Leetcode 115)**

**Instinct:**  
Count of times string t appears as subsequence in string s. Har char ya toh match karo (if equal) ya skip karo.

**Approach:**  
1. Memo[i][j] = ways to form t[j..] from s[i..]
2. If j == t.length() → 1 (found)
3. If i == s.length() → 0 (not found)
4. If s[i] == t[j] → take + skip
5. Else → skip

```java
class Solution {
    String s, t;
    int[][] memo;
    
    public int numDistinct(String s, String t) {
        this.s = s;
        this.t = t;
        int n = s.length(), m = t.length();
        memo = new int[n][m];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        return dfs(0, 0);
    }
    
    private int dfs(int i, int j) {
        if (j == t.length()) return 1;
        if (i == s.length()) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int res = dfs(i + 1, j); // skip
        if (s.charAt(i) == t.charAt(j)) {
            res += dfs(i + 1, j + 1); // take
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** Count subsequences = take/skip DP. Take only when chars match.

## 6. **Wildcard Matching (Leetcode 44)**

**Instinct:**  
Pattern with '?' (any single char) and '*' (any sequence). DP on indices of string and pattern.

**Approach:**  
1. Memo[i][j] = can pattern[j..] match string[i..]
2. If j == pattern.length() → i == string.length()
3. If i == string.length() → check if remaining pattern are all '*'
4. If pattern[j] == '*' → match zero or more chars
5. If pattern[j] == '?' or match char → move both

```java
class Solution {
    String s, p;
    int[][] memo;
    
    public boolean isMatch(String s, String p) {
        this.s = s;
        this.p = p;
        int n = s.length(), m = p.length();
        memo = new int[n + 1][m + 1];
        for (int i = 0; i <= n; i++) {
            Arrays.fill(memo[i], -1);
        }
        return dfs(0, 0);
    }
    
    private boolean dfs(int i, int j) {
        if (j == p.length()) return i == s.length();
        if (memo[i][j] != -1) return memo[i][j] == 1;
        
        boolean res;
        if (p.charAt(j) == '*') {
            res = dfs(i, j + 1) || (i < s.length() && dfs(i + 1, j));
        } else {
            boolean match = i < s.length() && 
                (p.charAt(j) == '?' || p.charAt(j) == s.charAt(i));
            res = match && dfs(i + 1, j + 1);
        }
        
        memo[i][j] = res ? 1 : 0;
        return res;
    }
}
```

**Key Insight:** '*' matches zero or more chars. '?' matches exactly one char.

## 7. **Regular Expression Matching (Leetcode 10)**

**Instinct:**  
Pattern with '.' (any char) and '*' (zero or more of previous char). More complex than wildcard.

**Approach:**  
1. Memo[i][j] = can pattern[j..] match string[i..]
2. If j == p.length() → i == s.length()
3. Check if next char is '*'
4. If yes → match zero or more of current char
5. If no → match single char (with '.' support)

```java
class Solution {
    String s, p;
    int[][] memo;
    
    public boolean isMatch(String s, String p) {
        this.s = s;
        this.p = p;
        int n = s.length(), m = p.length();
        memo = new int[n + 1][m + 1];
        for (int i = 0; i <= n; i++) {
            Arrays.fill(memo[i], -1);
        }
        return dfs(0, 0);
    }
    
    private boolean dfs(int i, int j) {
        if (j == p.length()) return i == s.length();
        if (memo[i][j] != -1) return memo[i][j] == 1;
        
        boolean firstMatch = i < s.length() && 
            (p.charAt(j) == '.' || p.charAt(j) == s.charAt(i));
        
        boolean res;
        if (j + 1 < p.length() && p.charAt(j + 1) == '*') {
            res = dfs(i, j + 2) || (firstMatch && dfs(i + 1, j));
        } else {
            res = firstMatch && dfs(i + 1, j + 1);
        }
        
        memo[i][j] = res ? 1 : 0;
        return res;
    }
}
```

**Key Insight:** Regex '*' applies to previous character. Need to check next char for '*'.

## 8. **Interleaving Strings (Leetcode 97)**

**Instinct:**  
Check if s3 is interleaving of s1 and s2 (order preserved, but interleaved). 2D DP on indices of s1 and s2.

**Approach:**  
1. Memo[i][j] = can we form s3[i+j..] from s1[i..] and s2[j..]
2. If i+j == s3.length() → true
3. Try taking from s1 if s1[i] == s3[i+j]
4. Try taking from s2 if s2[j] == s3[i+j]

```java
class Solution {
    String s1, s2, s3;
    int[][] memo;
    
    public boolean isInterleave(String s1, String s2, String s3) {
        this.s1 = s1;
        this.s2 = s2;
        this.s3 = s3;
        if (s1.length() + s2.length() != s3.length()) return false;
        
        int n = s1.length(), m = s2.length();
        memo = new int[n + 1][m + 1];
        for (int i = 0; i <= n; i++) {
            Arrays.fill(memo[i], -1);
        }
        return dfs(0, 0);
    }
    
    private boolean dfs(int i, int j) {
        if (i + j == s3.length()) return true;
        if (memo[i][j] != -1) return memo[i][j] == 1;
        
        boolean res = false;
        if (i < s1.length() && s1.charAt(i) == s3.charAt(i + j)) {
            res = res || dfs(i + 1, j);
        }
        if (j < s2.length() && s2.charAt(j) == s3.charAt(i + j)) {
            res = res || dfs(i, j + 1);
        }
        
        memo[i][j] = res ? 1 : 0;
        return res;
    }
}
```

**Key Insight:** Interleaving = pick from s1 or s2 at each step. 2D DP on indices.

## 9. **Minimum Window Subsequence (Leetcode 727)**

**Instinct:**  
S1 ka shortest substring which contains S2 as subsequence. Two pointers + DP on positions.

**Approach:**  
1. Precompute next[i][c] = next index of char c in S1 from position i
2. For each start in S1, find end using next array
3. Track minimum window

```java
class Solution {
    public String minWindow(String s1, String s2) {
        int n = s1.length(), m = s2.length();
        
        // Precompute next occurrence
        int[][] next = new int[n + 1][26];
        Arrays.fill(next[n], -1);
        for (int i = n - 1; i >= 0; i--) {
            for (int c = 0; c < 26; c++) {
                next[i][c] = next[i + 1][c];
            }
            next[i][s1.charAt(i) - 'a'] = i;
        }
        
        int minLen = Integer.MAX_VALUE;
        int startIdx = -1;
        
        for (int start = 0; start < n; start++) {
            if (s1.charAt(start) != s2.charAt(0)) continue;
            
            int pos = start;
            boolean found = true;
            for (int j = 1; j < m; j++) {
                pos = next[pos + 1][s2.charAt(j) - 'a'];
                if (pos == -1) {
                    found = false;
                    break;
                }
            }
            
            if (found && pos - start + 1 < minLen) {
                minLen = pos - start + 1;
                startIdx = start;
            }
        }
        
        return startIdx == -1 ? "" : s1.substring(startIdx, startIdx + minLen);
    }
}
```

**Key Insight:** Precompute next array for O(1) jumps. Greedy matching from each start.

## 10. **Palindromic Substrings (Leetcode 647)**

**Instinct:**  
String me kitne palindromic substrings hain (contiguous). Expand around center approach.

**Approach:**  
1. Expand around center for odd and even length palindromes
2. Count each valid palindrome

```java
class Solution {
    String s;
    
    public int countSubstrings(String s) {
        this.s = s;
        int n = s.length();
        int count = 0;
        
        for (int center = 0; center < n; center++) {
            count += expand(center, center);     // odd length
            count += expand(center, center + 1); // even length
        }
        return count;
    }
    
    private int expand(int left, int right) {
        int count = 0;
        while (left >= 0 && right < s.length() && s.charAt(left) == s.charAt(right)) {
            count++;
            left--;
            right++;
        }
        return count;
    }
}
```

**Key Insight:** Expand around center O(n²). Each palindrome counted once.

## 11. **Longest Palindromic Substring (Leetcode 5)**

**Instinct:**  
String ka longest palindromic substring. Expand around center or DP.

**Approach:**  
1. Expand around center for odd and even
2. Track max length and start index

```java
class Solution {
    String s;
    int maxLen = 0;
    int start = 0;
    
    public String longestPalindrome(String s) {
        this.s = s;
        int n = s.length();
        
        for (int center = 0; center < n; center++) {
            expand(center, center);     // odd
            expand(center, center + 1); // even
        }
        return s.substring(start, start + maxLen);
    }
    
    private void expand(int left, int right) {
        while (left >= 0 && right < s.length() && s.charAt(left) == s.charAt(right)) {
            if (right - left + 1 > maxLen) {
                maxLen = right - left + 1;
                start = left;
            }
            left--;
            right++;
        }
    }
}
```

**Key Insight:** Expand around center O(n²). Track max length.

## 12. **Minimum Insertions to Make String Palindrome (Leetcode 1312)**

**Instinct:**  
String ko palindrome banane ke liye minimum insertions. Insertions = n - LPS length.

**Approach:**  
1. Find LPS length using LCS(s, reverse(s))
2. Answer = n - LPS

```java
class Solution {
    String s;
    int[][] memo;
    
    public int minInsertions(String s) {
        this.s = s;
        int n = s.length();
        memo = new int[n][n];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        int lps = dfs(0, n - 1);
        return n - lps;
    }
    
    private int dfs(int i, int j) {
        if (i > j) return 0;
        if (i == j) return 1;
        if (memo[i][j] != -1) return memo[i][j];
        
        int res;
        if (s.charAt(i) == s.charAt(j)) {
            res = 2 + dfs(i + 1, j - 1);
        } else {
            res = Math.max(dfs(i + 1, j), dfs(i, j - 1));
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** Minimum insertions = n - LPS. LPS = longest palindromic subsequence.

## 15. **Minimum ASCII Delete Sum for Two Strings (Leetcode 712)**

**Instinct:**  
Delete karna hai, but cost = ASCII value. Min cost to make strings equal. Weighted LCS.

**Approach:**  
1. Memo[i][j] = max ASCII sum of common subsequence
2. If match → sum + dfs(i+1, j+1)
3. Else → max(dfs(i+1, j), dfs(i, j+1))
4. Answer = totalSum1 + totalSum2 - 2 * maxCommonSum

```java
class Solution {
    String s1, s2;
    Integer[][] memo;
    
    public int minimumDeleteSum(String s1, String s2) {
        this.s1 = s1;
        this.s2 = s2;
        int n = s1.length(), m = s2.length();
        memo = new Integer[n][m];
        
        int totalSum = 0;
        for (char c : s1.toCharArray()) totalSum += c;
        for (char c : s2.toCharArray()) totalSum += c;
        
        int commonSum = dfs(0, 0);
        return totalSum - 2 * commonSum;
    }
    
    private int dfs(int i, int j) {
        if (i == s1.length() || j == s2.length()) return 0;
        if (memo[i][j] != null) return memo[i][j];
        
        int res;
        if (s1.charAt(i) == s2.charAt(j)) {
            res = s1.charAt(i) + dfs(i + 1, j + 1);
        } else {
            res = Math.max(dfs(i + 1, j), dfs(i, j + 1));
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** Weighted LCS - maximize ASCII sum instead of length.

## 16. **Number of Ways to Form a Target String Given a Dictionary (Leetcode 1639)**

**Instinct:**  
Dictionary words se target string form karni hai. Har position pe koi bhi word se char le sakte ho. Frequency count + DP.

**Approach:**  
1. frequency[col][char] = kitne words me col pe char hai
2. Memo[i][j] = ways to form target from index i using column j

```java
class Solution {
    int MOD = 1_000_000_007;
    String target;
    int[][] freq;
    Integer[][] memo;
    
    public int numWays(String[] words, String target) {
        this.target = target;
        int m = words[0].length();
        int n = target.length();
        if (m < n) return 0;
        
        freq = new int[m][26];
        for (String word : words) {
            for (int j = 0; j < m; j++) {
                freq[j][word.charAt(j) - 'a']++;
            }
        }
        
        memo = new Integer[n][m];
        return dfs(0, 0);
    }
    
    private int dfs(int targetIdx, int col) {
        if (targetIdx == target.length()) return 1;
        if (col >= freq.length) return 0;
        if (memo[targetIdx][col] != null) return memo[targetIdx][col];
        
        int ch = target.charAt(targetIdx) - 'a';
        
        // skip current column
        long ways = dfs(targetIdx, col + 1);
        
        // use current column
        if (freq[col][ch] > 0) {
            ways += (long) freq[col][ch] * dfs(targetIdx + 1, col + 1);
        }
        
        memo[targetIdx][col] = (int) (ways % MOD);
        return memo[targetIdx][col];
    }
}
```

**Key Insight:** Column-wise DP. Each column ka frequency multiply hota hai.

## 17. **String Compression II (Leetcode 1531)**

**Instinct:**  
String compress karna hai with at most k deletions. Minimum compressed length. DP on index with remaining deletions.

**Approach:**  
1. Memo[i][k] = min compressed length from index i with k deletions left
2. Try deleting current char or keeping it
3. When keeping, count consecutive same chars

```java
class Solution {
    String s;
    int n;
    int[][] memo;
    
    public int getLengthOfOptimalCompression(String s, int k) {
        this.s = s;
        this.n = s.length();
        memo = new int[n][k + 1];
        for (int i = 0; i < n; i++) {
            Arrays.fill(memo[i], -1);
        }
        return dfs(0, k);
    }
    
    private int dfs(int i, int k) {
        if (k < 0) return Integer.MAX_VALUE / 2;
        if (i == n || n - i <= k) return 0;
        if (memo[i][k] != -1) return memo[i][k];
        
        // delete current char
        int res = dfs(i + 1, k - 1);
        
        // keep current char
        int count = 0;
        int same = 0;
        for (int j = i; j < n && same <= k; j++) {
            if (s.charAt(j) == s.charAt(i)) {
                count++;
            } else {
                same++;
                if (same > k) break;
            }
            
            int len = 1 + (count >= 100 ? 3 : count >= 10 ? 2 : count >= 2 ? 1 : 0);
            res = Math.min(res, len + dfs(j + 1, k - same));
        }
        
        memo[i][k] = res;
        return res;
    }
}
```

**Key Insight:** Run-length encoding. Keep consecutive same chars, delete others.

## 18. **Minimum Number of Operations to Make String Sorted (Leetcode 1830)**

**Instinct:**  
String ko sorted order me laane ke liye minimum operations. Count of smaller permutations.

**Approach:**  
1. Count frequency of characters
2. Use factorial and inverse factorial
3. For each position, count how many smaller chars can come here

```java
class Solution {
    int MOD = 1_000_000_007;
    
    public int makeStringSorted(String s) {
        int n = s.length();
        long[] fact = new long[n + 1];
        long[] invFact = new long[n + 1];
        fact[0] = 1;
        for (int i = 1; i <= n; i++) {
            fact[i] = fact[i - 1] * i % MOD;
        }
        invFact[n] = pow(fact[n], MOD - 2);
        for (int i = n - 1; i >= 0; i--) {
            invFact[i] = invFact[i + 1] * (i + 1) % MOD;
        }
        
        int[] freq = new int[26];
        for (char c : s.toCharArray()) {
            freq[c - 'a']++;
        }
        
        long ans = 0;
        for (int i = 0; i < n; i++) {
            int cur = s.charAt(i) - 'a';
            for (int c = 0; c < cur; c++) {
                if (freq[c] == 0) continue;
                freq[c]--;
                long ways = fact[n - i - 1];
                for (int f : freq) {
                    ways = ways * invFact[f] % MOD;
                }
                ans = (ans + ways) % MOD;
                freq[c]++;
            }
            freq[cur]--;
        }
        return (int) ans;
    }
    
    private long pow(long a, long b) {
        long res = 1;
        while (b > 0) {
            if ((b & 1) == 1) res = res * a % MOD;
            a = a * a % MOD;
            b >>= 1;
        }
        return res;
    }
}
```

**Key Insight:** Lexicographically smaller permutations count. Use combinatorics.

## 19. **Number of Substrings With Only 1s (Leetcode 1513)**

**Instinct:**  
String me substrings count jisme sirf '1' hai. Count consecutive 1s.

**Approach:**  
1. Count consecutive 1s
2. For n consecutive 1s, number of substrings = n*(n+1)/2

```java
class Solution {
    int MOD = 1_000_000_007;
    
    public int numSub(String s) {
        long ans = 0;
        int count = 0;
        
        for (char c : s.toCharArray()) {
            if (c == '1') {
                count++;
                ans = (ans + count) % MOD;
            } else {
                count = 0;
            }
        }
        
        return (int) ans;
    }
}
```

**Key Insight:** Substrings of consecutive 1s = sum of 1+2+...+n.

## 20. **Count Substrings That Differ by One Character (Leetcode 1638)**

**Instinct:**  
Do strings me substrings count jinka exactly one character difference ho. DP on indices with mismatch count.

**Approach:**  
1. Memo[i][j][diff] = count from i,j with diff mismatches allowed
2. diff can be 0 or 1
3. If chars match → add both diff states
4. If chars differ and diff == 1 → add with diff-1

```java
class Solution {
    String s, t;
    int n, m;
    int[][][] memo;
    
    public int countSubstrings(String s, String t) {
        this.s = s;
        this.t = t;
        n = s.length();
        m = t.length();
        memo = new int[n][m][2];
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                Arrays.fill(memo[i][j], -1);
            }
        }
        
        int total = 0;
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                total += dfs(i, j, 1);
            }
        }
        return total;
    }
    
    private int dfs(int i, int j, int diff) {
        if (i >= n || j >= m) return 0;
        if (memo[i][j][diff] != -1) return memo[i][j][diff];
        
        int res = 0;
        if (s.charAt(i) == t.charAt(j)) {
            res = dfs(i + 1, j + 1, diff);
            if (diff == 1) {
                res += 1; // this substring itself
            }
        } else if (diff == 1) {
            res = 1 + dfs(i + 1, j + 1, 0);
        }
        
        memo[i][j][diff] = res;
        return res;
    }
}
```

**Key Insight:** Track number of mismatches allowed (0 or 1). Extend while condition holds.

## 📝 Key Formulas to Remember

| Operation | Formula |
|-----------|---------|
| Edit Distance | `min(insert, delete, replace) + 1` |
| LCS | `1 + dp[i+1][j+1]` if match, else `max(dp[i+1][j], dp[i][j+1])` |
| Distinct Subsequences | `dp[i][j] = dp[i+1][j] + (match ? dp[i+1][j+1] : 0)` |
| Wildcard '*' | `dp[i][j] = dp[i][j+1] OR dp[i+1][j]` |
| Regex '*' | `dp[i][j] = dp[i][j+2] OR (match AND dp[i+1][j])` |
| Min Insertions to Palindrome | `n - LPS` |
| LPS | `LCS(s, reverse(s))` |


## 1. **Word Break II** (Memoization)

**Instinct:**  
Har index se end tak sentences generate karne hain. Agar current word dictionary me hai, toh aage ke index se sentences laake join karo.

**Approach:**  
- Memo[i] = list of sentences from i to end
- Base case: i == n → empty string list

```java
class Solution {
    public List<String> wordBreak(String s, List<String> wordDict) {
        Set<String> wordSet = new HashSet<>(wordDict);
        Map<Integer, List<String>> memo = new HashMap<>();
        return dfs(s, 0, wordSet, memo);
    }
    
    private List<String> dfs(String s, int start, Set<String> wordSet, Map<Integer, List<String>> memo) {
        if (memo.containsKey(start)) return memo.get(start);
        List<String> res = new ArrayList<>();
        
        if (start == s.length()) {
            res.add("");
            return res;
        }
        
        for (int end = start + 1; end <= s.length(); end++) {
            String word = s.substring(start, end);
            if (wordSet.contains(word)) {
                List<String> suffixes = dfs(s, end, wordSet, memo);
                for (String suffix : suffixes) {
                    res.add(word + (suffix.isEmpty() ? "" : " " + suffix));
                }
            }
        }
        memo.put(start, res);
        return res;
    }
}
```

## 2. **Burst Balloons** (Memoization)

**Instinct:**  
Last balloon ko sojo jo burst hoga. Usse pehle left part aur right part independent solve ho jaayenge.

**Approach:**  
- dp[i][j] = max coins from i to j (inclusive)
- Try karo har k ko last balloon

```java
class Solution {
    public int maxCoins(int[] nums) {
        int n = nums.length;
        int[] arr = new int[n + 2];
        arr[0] = arr[n + 1] = 1;
        for (int i = 1; i <= n; i++) arr[i] = nums[i - 1];
        
        Integer[][] memo = new Integer[n + 2][n + 2];
        return solve(arr, 1, n, memo);
    }
    
    private int solve(int[] arr, int i, int j, Integer[][] memo) {
        if (i > j) return 0;
        if (memo[i][j] != null) return memo[i][j];
        
        int max = 0;
        for (int k = i; k <= j; k++) {
            int left = solve(arr, i, k - 1, memo);
            int right = solve(arr, k + 1, j, memo);
            int val = arr[i - 1] * arr[k] * arr[j + 1];
            max = Math.max(max, left + right + val);
        }
        memo[i][j] = max;
        return max;
    }
}
```

## 3. **Russian Doll Envelopes** (Memoization)

**Instinct:**  
Pehle width sort karo, phir height pe LIS (Longest Increasing Subsequence) jaisa problem. Memoization me har envelope ko pick karke aage dekho.

**Approach:**  
- Width ascending, agar width same toh height descending sort karo
- Memo[i] = max envelopes starting from i

```java
class Solution {
    public int maxEnvelopes(int[][] envelopes) {
        Arrays.sort(envelopes, (a, b) -> {
            if (a[0] == b[0]) return b[1] - a[1];
            return a[0] - b[0];
        });
        
        int n = envelopes.length;
        int[] memo = new int[n];
        Arrays.fill(memo, -1);
        
        int max = 0;
        for (int i = 0; i < n; i++) {
            max = Math.max(max, dfs(envelopes, i, memo));
        }
        return max;
    }
    
    private int dfs(int[][] envelopes, int i, int[] memo) {
        if (memo[i] != -1) return memo[i];
        
        int maxLen = 1;
        for (int j = i + 1; j < envelopes.length; j++) {
            if (envelopes[j][0] > envelopes[i][0] && envelopes[j][1] > envelopes[i][1]) {
                maxLen = Math.max(maxLen, 1 + dfs(envelopes, j, memo));
            }
        }
        memo[i] = maxLen;
        return maxLen;
    }
}
```

## 4. **Maximum Profit in Job Scheduling** (Memoization)

**Instinct:**  
Jobs ko end time ke hisaab se sort karo. Har job ko ya toh lo (aur agla non-conflicting job dhundho) ya mat lo.

**Approach:**  
- Sort by end time
- Binary search se next non-conflicting job find karo
- Memo[i] = max profit from i to end

```java
class Solution {
    public int jobScheduling(int[] startTime, int[] endTime, int[] profit) {
        int n = startTime.length;
        Job[] jobs = new Job[n];
        for (int i = 0; i < n; i++) {
            jobs[i] = new Job(startTime[i], endTime[i], profit[i]);
        }
        Arrays.sort(jobs, (a, b) -> a.end - b.end);
        
        Integer[] memo = new Integer[n];
        return dfs(jobs, 0, memo);
    }
    
    private int dfs(Job[] jobs, int i, Integer[] memo) {
        if (i >= jobs.length) return 0;
        if (memo[i] != null) return memo[i];
        
        // take current job
        int nextIndex = findNext(jobs, i);
        int take = jobs[i].profit + dfs(jobs, nextIndex, memo);
        
        // skip current job
        int skip = dfs(jobs, i + 1, memo);
        
        memo[i] = Math.max(take, skip);
        return memo[i];
    }
    
    private int findNext(Job[] jobs, int i) {
        int lo = i + 1, hi = jobs.length;
        while (lo < hi) {
            int mid = lo + (hi - lo) / 2;
            if (jobs[mid].start >= jobs[i].end) hi = mid;
            else lo = mid + 1;
        }
        return lo;
    }
    
    class Job {
        int start, end, profit;
        Job(int s, int e, int p) { start = s; end = e; profit = p; }
    }
}
```

## 5. **Stone Game II** (Memoization)

**Instinct:**  
Alice aur Bob dono optimal khelte hain. Alice maximum laane ki koshish karegi, Bob usse kam dene ki. Turn ke hisaab se plus/minus karo.

**Approach:**  
- Memo[i][m] = max difference from index i with M = m
- Alice picks 1 se 2m stones, fir turn Bob ka

```java
class Solution {
    public int stoneGameII(int[] piles) {
        int n = piles.length;
        int[] suffixSum = new int[n + 1];
        for (int i = n - 1; i >= 0; i--) {
            suffixSum[i] = suffixSum[i + 1] + piles[i];
        }
        
        Integer[][] memo = new Integer[n][n + 1];
        return dfs(piles, 0, 1, memo, suffixSum);
    }
    
    private int dfs(int[] piles, int i, int m, Integer[][] memo, int[] suffixSum) {
        if (i >= piles.length) return 0;
        if (memo[i][m] != null) return memo[i][m];
        
        int maxStones = 0;
        for (int x = 1; x <= 2 * m; x++) {
            if (i + x > piles.length) break;
            int nextM = Math.max(m, x);
            int opponent = dfs(piles, i + x, nextM, memo, suffixSum);
            int current = suffixSum[i] - opponent;
            maxStones = Math.max(maxStones, current);
        }
        memo[i][m] = maxStones;
        return maxStones;
    }
}
```

## 6. **Predict the Winner** (Memoization)

**Instinct:**  
Do players, ek front se lega toh doosra remaining se. Difference maximize karo current player ke liye.

**Approach:**  
- Memo[i][j] = max difference from i to j for current player
- Agar i == j → nums[i]
- Warna max(nums[i] - solve(i+1, j), nums[j] - solve(i, j-1))

```java
class Solution {
    public boolean predictTheWinner(int[] nums) {
        int n = nums.length;
        Integer[][] memo = new Integer[n][n];
        int diff = dfs(nums, 0, n - 1, memo);
        return diff >= 0;
    }
    
    private int dfs(int[] nums, int i, int j, Integer[][] memo) {
        if (i > j) return 0;
        if (memo[i][j] != null) return memo[i][j];
        
        int takeLeft = nums[i] - dfs(nums, i + 1, j, memo);
        int takeRight = nums[j] - dfs(nums, i, j - 1, memo);
        
        memo[i][j] = Math.max(takeLeft, takeRight);
        return memo[i][j];
    }
}
```

## 7. **Minimum Cost Tree From Leaf Values** (Memoization)

**Instinct:**  
Binary tree me har internal node = product of two leaves. Partition array into left aur right subtree.

**Approach:**  
- Memo[i][j] = min cost from i to j
- Try karo har partition point k
- left max * right max + left cost + right cost

```java
class Solution {
    public int mctFromLeafValues(int[] arr) {
        int n = arr.length;
        Integer[][] memo = new Integer[n][n];
        return dfs(arr, 0, n - 1, memo);
    }
    
    private int dfs(int[] arr, int i, int j, Integer[][] memo) {
        if (i >= j) return 0;
        if (memo[i][j] != null) return memo[i][j];
        
        int minCost = Integer.MAX_VALUE;
        for (int k = i; k < j; k++) {
            int leftMax = getMax(arr, i, k);
            int rightMax = getMax(arr, k + 1, j);
            int cost = leftMax * rightMax + dfs(arr, i, k, memo) + dfs(arr, k + 1, j, memo);
            minCost = Math.min(minCost, cost);
        }
        memo[i][j] = minCost;
        return minCost;
    }
    
    private int getMax(int[] arr, int i, int j) {
        int max = arr[i];
        for (int k = i + 1; k <= j; k++) max = Math.max(max, arr[k]);
        return max;
    }
}
```

## 8. **Unique Binary Search Trees II** (Memoization)

**Instinct:**  
Har root ke liye left subtree possibilities aur right subtree possibilities combine karo.

**Approach:**  
- Memo[i][j] = list of trees from i to j
- Root k ke liye left = i to k-1, right = k+1 to j

```java
class Solution {
    public List<TreeNode> generateTrees(int n) {
        return dfs(1, n);
    }
    
    private List<TreeNode> dfs(int start, int end) {
        List<TreeNode> res = new ArrayList<>();
        if (start > end) {
            res.add(null);
            return res;
        }
        
        for (int rootVal = start; rootVal <= end; rootVal++) {
            List<TreeNode> leftTrees = dfs(start, rootVal - 1);
            List<TreeNode> rightTrees = dfs(rootVal + 1, end);
            
            for (TreeNode left : leftTrees) {
                for (TreeNode right : rightTrees) {
                    TreeNode root = new TreeNode(rootVal);
                    root.left = left;
                    root.right = right;
                    res.add(root);
                }
            }
        }
        return res;
    }
}
```

## 9. **Minimum Difficulty of a Job Schedule** (Memoization)

**Instinct:**  
d days me jobs ko partition karna hai. Har day ka max job difficulty, sum minimize karna hai.

**Approach:**  
- Memo[i][d] = min difficulty from index i with d days left
- Try karo har possible partition

```java
class Solution {
    public int minDifficulty(int[] jobDifficulty, int d) {
        int n = jobDifficulty.length;
        if (n < d) return -1;
        
        Integer[][] memo = new Integer[n][d + 1];
        return dfs(jobDifficulty, 0, d, memo);
    }
    
    private int dfs(int[] jobs, int i, int daysLeft, Integer[][] memo) {
        if (i == jobs.length && daysLeft == 0) return 0;
        if (i == jobs.length || daysLeft == 0) return Integer.MAX_VALUE / 2;
        if (memo[i][daysLeft] != null) return memo[i][daysLeft];
        
        int maxSoFar = 0;
        int minDiff = Integer.MAX_VALUE / 2;
        
        for (int j = i; j <= jobs.length - daysLeft; j++) {
            maxSoFar = Math.max(maxSoFar, jobs[j]);
            int rest = dfs(jobs, j + 1, daysLeft - 1, memo);
            minDiff = Math.min(minDiff, maxSoFar + rest);
        }
        
        memo[i][daysLeft] = minDiff;
        return minDiff;
    }
}
```

## 10. **Tallest Billboard** (Memoization)

**Instinct:**  
Do rods ka height difference maintain karo. Har rod ko left, right, ya discard karo.

**Approach:**  
- Memo[i][diff] = max common height with difference = diff from index i
- Try karo add to left, add to right, ya skip

```java
class Solution {
    public int tallestBillboard(int[] rods) {
        Integer[][] memo = new Integer[rods.length][10001];
        return dfs(rods, 0, 0, memo);
    }
    
    private int dfs(int[] rods, int i, int diff, Integer[][] memo) {
        if (i == rods.length) return diff == 0 ? 0 : Integer.MIN_VALUE / 2;
        if (memo[i][diff + 5000] != null) return memo[i][diff + 5000];
        
        // skip
        int skip = dfs(rods, i + 1, diff, memo);
        
        // add to left (diff increases)
        int left = rods[i] + dfs(rods, i + 1, diff + rods[i], memo);
        
        // add to right (diff decreases)
        int right = dfs(rods, i + 1, Math.abs(diff - rods[i]), memo);
        if (diff >= rods[i]) right += rods[i];
        
        int res = Math.max(skip, Math.max(left, right));
        memo[i][diff + 5000] = res;
        return res;
    }
}
```

## 11. **Number of Ways to Stay in the Same Place After Some Steps** (Memoization)

**Instinct:**  
Step index par ho, either left, right, ya stay. Memoization se overlapping subproblems solve karo.

**Approach:**  
- Memo[step][index] = ways to reach index with remaining steps
- Base: steps == 0 → index == 0 ? 1 : 0

```java
class Solution {
    public int numWays(int steps, int arrLen) {
        Integer[][] memo = new Integer[steps + 1][Math.min(arrLen, steps / 2 + 1)];
        return dfs(steps, 0, arrLen, memo);
    }
    
    private int dfs(int steps, int pos, int arrLen, Integer[][] memo) {
        if (steps == 0) return pos == 0 ? 1 : 0;
        if (memo[steps][pos] != null) return memo[steps][pos];
        
        long ways = dfs(steps - 1, pos, arrLen, memo); // stay
        if (pos > 0) ways += dfs(steps - 1, pos - 1, arrLen, memo); // left
        if (pos < arrLen - 1 && pos < steps) ways += dfs(steps - 1, pos + 1, arrLen, memo); // right
        
        memo[steps][pos] = (int)(ways % 1_000_000_007);
        return memo[steps][pos];
    }
}
```

## 12. **Number of Music Playlists** (Memoization)

**Instinct:**  
n unique songs, goal length = goal, har song repeat kar sakte hain but k gap hona chahiye.

**Approach:**  
- Memo[played][used] = ways to fill remaining with used unique songs
- Pick new song ya old song (agar available ho)

```java
class Solution {
    int MOD = 1_000_000_007;
    
    public int numMusicPlaylists(int n, int goal, int k) {
        Long[][] memo = new Long[goal + 1][n + 1];
        return (int)dfs(goal, n, n, k, memo);
    }
    
    private long dfs(int remaining, int used, int n, int k, Long[][] memo) {
        if (remaining == 0) return used == n ? 1 : 0;
        if (memo[remaining][used] != null) return memo[remaining][used];
        
        long res = 0;
        // play new song
        if (used < n) {
            res += (n - used) * dfs(remaining - 1, used + 1, n, k, memo);
        }
        // play old song
        if (used > k) {
            res += (used - k) * dfs(remaining - 1, used, n, k, memo);
        }
        
        memo[remaining][used] = res % MOD;
        return memo[remaining][used];
    }
}
```

## 13. **Count Different Palindromic Subsequences** (Memoization)

**Instinct:**  
Har character 'a' to 'd' ke liye pehla aur last occurrence find karo. Uske beech me recursion.

**Approach:**  
- Memo[i][j] = count from i to j
- Agar s[i] != s[j] → left + right - middle
- Agar same → 2 * inner + special case

```java
class Solution {
    int MOD = 1_000_000_007;
    
    public int countPalindromicSubsequences(String s) {
        int n = s.length();
        Integer[][] memo = new Integer[n][n];
        return dfs(s, 0, n - 1, memo);
    }
    
    private int dfs(String s, int i, int j, Integer[][] memo) {
        if (i > j) return 0;
        if (i == j) return 1;
        if (memo[i][j] != null) return memo[i][j];
        
        long res = 0;
        if (s.charAt(i) != s.charAt(j)) {
            res = dfs(s, i + 1, j, memo) + dfs(s, i, j - 1, memo) - dfs(s, i + 1, j - 1, memo);
        } else {
            int left = i + 1, right = j - 1;
            while (left <= right && s.charAt(left) != s.charAt(i)) left++;
            while (left <= right && s.charAt(right) != s.charAt(i)) right--;
            
            if (left > right) {
                res = 2 * dfs(s, i + 1, j - 1, memo) + 2;
            } else if (left == right) {
                res = 2 * dfs(s, i + 1, j - 1, memo) + 1;
            } else {
                res = 2 * dfs(s, i + 1, j - 1, memo) - dfs(s, left + 1, right - 1, memo);
            }
        }
        
        memo[i][j] = (int)((res + MOD) % MOD);
        return memo[i][j];
    }
}
```

## 14. **K Inverse Pairs Array** (Memoization)

```java
class Solution {

        for (int i = 0; i <= Math.min(k, n - 1); i++) {
```

## 15. **Strange Printer** (Memoization)

**Instinct:**  
Ek turn me same character print kar sakte ho. Agar s[i] == s[j] toh ek turn save hota hai.

**Approach:**  
- Memo[i][j] = min turns for s[i..j]

```java
class Solution {

            }
        }

    }
```

## 16. **Maximum Number of Events That Can Be Attended II** (Memoization)

**Instinct:**  
Events sort by end time. Har event ya toh attend karo ya skip. Next non-conflicting event find karo.

**Approach:**  
- Sort by end time
- Memo[i][k] = max value from i with k events left

```java
class Solution {
    public int maxValue(int[][] events, int k) {
        Arrays.sort(events, (a, b) -> a[1] - b[1]);
        int n = events.length;
        Integer[][] memo = new Integer[n][k + 1];
        return dfs(events, 0, k, memo);

    private int dfs(int[][] events, int i, int k, Integer[][] memo) {
        if (i >= events.length || k == 0) return 0;
        if (memo[i][k] != null) return memo[i][k];
        
        // skip
        int skip = dfs(events, i + 1, k, memo);
        
        // take
        int nextIndex = findNext(events, i);
        int take = events[i][2] + dfs(events, nextIndex, k - 1, memo);
        
        memo[i][k] = Math.max(skip, take);
        return memo[i][k];
    }
    
    private int findNext(int[][] events, int i) {
        int lo = i + 1, hi = events.length;
        while (lo < hi) {
            int mid = lo + (hi - lo) / 2;
            if (events[mid][0] > events[i][1]) hi = mid;
            else lo = mid + 1;
        }
        return lo;
    }
}
```

**Instinct:**

**Approach:**  
- Memo[i][stops] = max fuel reachable at station i with given stops

```java
class Solution {
    public int minRefuelStops(int target, int startFuel, int[][] stations) {
        int n = stations.length;
        Long[][] memo = new Long[n + 1][n + 1];
        int res = dfs(target, startFuel, stations, 0, 0, memo);

            res = Math.min(res, dfs(target, fuel, stations, i + 1, stops, memo));

        // take
        if (fuel >= stations[i][0]) {
            res = Math.min(res, dfs(target, fuel + stations[i][1], stations, i + 1, stops + 1, memo));
        }
        
        memo[i][stops] = (long)res;
        return res;
    }
}
```

## 19. **Super Egg Drop** (Memoization)

**Instinct:**  
K eggs, N floors. Har floor par try karo, agar toot gaya toh neeche, nahi to upar.

**Approach:**  
- Memo[k][n] = min moves
- Binary search se optimize karo

```java
class Solution {
    public int superEggDrop(int k, int n) {
        Integer[][] memo = new Integer[k + 1][n + 1];
        return dfs(k, n, memo);
    }
    
    private int dfs(int k, int n, Integer[][] memo) {
        if (n == 0 || n == 1) return n;
        if (k == 1) return n;
        if (memo[k][n] != null) return memo[k][n];
        
        int lo = 1, hi = n;
        int res = Integer.MAX_VALUE;
        
        while (lo <= hi) {
            int mid = lo + (hi - lo) / 2;
            int broken = dfs(k - 1, mid - 1, memo);
            int notBroken = dfs(k, n - mid, memo);
            
            if (broken > notBroken) {
                hi = mid - 1;
                res = Math.min(res, broken + 1);
            } else {
                lo = mid + 1;
                res = Math.min(res, notBroken + 1);
            }
        }
        
        memo[k][n] = res;
        return res;
    }
}
```

## 20. **Student Attendance Record II** (Memoization)

**Instinct:**  
3 states: absent count (0 or 1) aur consecutive late count (0,1,2). Memoization se count karo.

**Approach:**  
- Memo[n][absent][late] = ways
- Base: n == 0 → 1

```java
class Solution {
    int MOD = 1_000_000_007;
    
    public int checkRecord(int n) {
        Integer[][][] memo = new Integer[n + 1][2][3];
        return dfs(n, 0, 0, memo);
    }
    
    private int dfs(int n, int absent, int late, Integer[][][] memo) {
        if (n == 0) return 1;
        if (memo[n][absent][late] != null) return memo[n][absent][late];
        
        long res = 0;
        // Present
        res += dfs(n - 1, absent, 0, memo);
        
        // Absent
        if (absent == 0) {
            res += dfs(n - 1, 1, 0, memo);
        }
        
        // Late
        if (late < 2) {
            res += dfs(n - 1, absent, late + 1, memo);
        }
        
        memo[n][absent][late] = (int)(res % MOD);
        return memo[n][absent][late];
    }
}
```

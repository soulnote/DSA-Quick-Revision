# 📈 Longest Increasing Subsequence (LIS) Pattern

## LIS ke 7 Main Variants:

| Variant | Description | Approach |
|---------|-------------|----------|
| **Classic LIS** | Strictly increasing sequence | Pick or skip with index tracking |
| **Number of LIS** | Count of all longest increasing subsequences | Track length + count |
| **Maximum Sum Increasing Subsequence** | Sum maximize karna hai, length nahi | Track max sum |
| **Minimum deletions to make sorted** | n - LIS length | Convert to LIS |
| **Longest Bitonic Subsequence** | Increase then decrease | LIS from left + LIS from right |
| **Longest Chain of Pairs** | (a,b) -> (c,d) where b < c | Sort + LIS on second element |
| **Patience Sorting / Russian Doll** | 2D envelopes | Sort one dim, LIS on other |

## 1. **Longest Increasing Subsequence (Leetcode 300)**

**Instinct:**  
Array me se longest strictly increasing subsequence nikaalni hai. Classic LIS - har element ko ya toh current sequence me lo (agar previous se bada hai) ya skip karo. Memoization me track karna hoga **current index** aur **previous index**.

**Approach:**  
1. Memo[i][prev] = LIS length from index i with previous index = prev
2. Base: i == n → 0
3. Two options:
   - Skip current element
   - Take current element (if prev == -1 ya nums[i] > nums[prev])
4. Return max of both

```java
class Solution {
    int[] nums;
    int n;
    Integer[][] memo;
    
    public int lengthOfLIS(int[] nums) {
        this.nums = nums;
        n = nums.length;
        memo = new Integer[n][n + 1]; // prev+1 for -1 handling
        return dfs(0, -1);
    }
    
    private int dfs(int i, int prev) {
        if (i == n) return 0;
        int prevIdx = prev + 1; // shift for -1
        if (memo[i][prevIdx] != null) return memo[i][prevIdx];
        
        // skip current
        int skip = dfs(i + 1, prev);
        
        // take current
        int take = 0;
        if (prev == -1 || nums[i] > nums[prev]) {
            take = 1 + dfs(i + 1, i);
        }
        
        memo[i][prevIdx] = Math.max(skip, take);
        return memo[i][prevIdx];
    }
}
```

**Key Insight:** Classic 0/1 knapsack style LIS. State = (i, prevIndex). O(n²) time, O(n²) space.

## 2. **Number of Longest Increasing Subsequence (Leetcode 673)**

**Instinct:**  
Sirf length nahi, count bhi chahiye of all LIS. Har state ke liye (length, count) store karo. Memoization me **pair** return karna hoga.

**Approach:**  
1. Memo[i] = {maxLength from i, count of sequences with that length}
2. For each i, check all j > i where nums[j] > nums[i]
3. Find best length, sum counts

```java
class Solution {
    int[] nums;
    int n;
    Pair[][] memo;
    
    class Pair {
        int len;
        int count;
        Pair(int l, int c) { len = l; count = c; }
    }
    
    public int findNumberOfLIS(int[] nums) {
        this.nums = nums;
        n = nums.length;
        memo = new Pair[n][n + 1];
        
        Pair result = dfs(0, -1);
        return result.count;
    }
    
    private Pair dfs(int i, int prev) {
        if (i == n) return new Pair(0, 1);
        int prevIdx = prev + 1;
        if (memo[i][prevIdx] != null) return memo[i][prevIdx];
        
        // skip
        Pair skip = dfs(i + 1, prev);
        
        // take
        Pair take = new Pair(0, 0);
        if (prev == -1 || nums[i] > nums[prev]) {
            take = dfs(i + 1, i);
            take = new Pair(take.len + 1, take.count);
        }
        
        // combine
        if (skip.len > take.len) {
            memo[i][prevIdx] = skip;
        } else if (take.len > skip.len) {
            memo[i][prevIdx] = take;
        } else {
            memo[i][prevIdx] = new Pair(skip.len, skip.count + take.count);
        }
        
        return memo[i][prevIdx];
    }
}
```

**Key Insight:** Pair return karo (length, count). Jab lengths equal ho toh counts add karo.

## 3. **Maximum Sum Increasing Subsequence**

**Instinct:**  
Length ki jagah sum maximize karna hai. Same LIS logic, but take condition me sum add karo, length nahi.

**Approach:**  
1. Memo[i][prev] = max sum from i with previous index prev
2. Base: i == n → 0
3. Skip ya take (if increasing)
4. Return max sum

```java
class Solution {
    int[] nums;
    int n;
    Integer[][] memo;
    
    public int maxSumIS(int[] nums) {
        this.nums = nums;
        n = nums.length;
        memo = new Integer[n][n + 1];
        return dfs(0, -1);
    }
    
    private int dfs(int i, int prev) {
        if (i == n) return 0;
        int prevIdx = prev + 1;
        if (memo[i][prevIdx] != null) return memo[i][prevIdx];
        
        int skip = dfs(i + 1, prev);
        int take = 0;
        if (prev == -1 || nums[i] > nums[prev]) {
            take = nums[i] + dfs(i + 1, i);
        }
        
        memo[i][prevIdx] = Math.max(skip, take);
        return memo[i][prevIdx];
    }
}
```

**Key Insight:** Length → Sum. Same structure, different aggregation.

## 4. **Minimum Number of Operations to Make Array Strictly Increasing**

**Instinct:**  
Array ko strictly increasing banane ke liye minimum changes. Equivalent to: n - (length of longest increasing subsequence) but careful - yaha values change kar sakte ho, delete nahi. Actual me ye different hai.

**Approach (LIS based):**  
Agar delete allowed hota toh n - LIS hota. But yaha change allowed hai. Isliye different approach chahiye.

```java
// For delete operations: n - LIS
class Solution {
    public int minOperationsToMakeStrictlyIncreasing(int[] nums) {
        // If we can only delete, answer = n - LIS
        int lisLen = lengthOfLIS(nums);
        return nums.length - lisLen;
    }
}
```

**Key Insight:** Minimum deletions to make sorted = n - LIS.

## 5. **Longest Bitonic Subsequence**

**Instinct:**  
Sequence jo pehle strictly increase kare, phir strictly decrease. Har index ke liye: LIS from left + LIS from right (decreasing). Bitonic length = leftLIS[i] + rightLDS[i] - 1.

**Approach:**  
1. left[i] = LIS ending at i (from left)
2. right[i] = LDS starting at i (from right, or LIS on reverse)
3. Max of left[i] + right[i] - 1

```java
class Solution {
    int[] nums;
    int n;
    
    public int longestBitonicSubsequence(int[] nums) {
        this.nums = nums;
        n = nums.length;
        
        int[] left = new int[n];
        int[] right = new int[n];
        
        // LIS from left
        for (int i = 0; i < n; i++) {
            left[i] = 1;
            for (int j = 0; j < i; j++) {
                if (nums[j] < nums[i]) {
                    left[i] = Math.max(left[i], left[j] + 1);
                }
            }
        }
        
        // LDS from right (LIS on reverse)
        for (int i = n - 1; i >= 0; i--) {
            right[i] = 1;
            for (int j = n - 1; j > i; j--) {
                if (nums[j] < nums[i]) {
                    right[i] = Math.max(right[i], right[j] + 1);
                }
            }
        }
        
        int maxLen = 0;
        for (int i = 0; i < n; i++) {
            maxLen = Math.max(maxLen, left[i] + right[i] - 1);
        }
        return maxLen;
    }
}
```

**Key Insight:** Bitonic = LIS(left) + LDS(right). LDS = LIS on reversed array.

## 6. **Maximum Length of Pair Chain (Leetcode 646)**

**Instinct:**  
Pairs (a,b) diye hain, chain banaani hai where b < c of next pair. Sort by first element, then LIS on second element. Ye **interval scheduling LIS** hai.

**Approach:**  
1. Sort pairs by first element
2. Memo[i][prev] = max chain length from i with previous index prev
3. Take if pairs[i][0] > pairs[prev][1]

```java
class Solution {
    int[][] pairs;
    int n;
    Integer[][] memo;
    
    public int findLongestChain(int[][] pairs) {
        this.pairs = pairs;
        n = pairs.length;
        Arrays.sort(pairs, (a, b) -> a[0] - b[0]);
        memo = new Integer[n][n + 1];
        return dfs(0, -1);
    }
    
    private int dfs(int i, int prev) {
        if (i == n) return 0;
        int prevIdx = prev + 1;
        if (memo[i][prevIdx] != null) return memo[i][prevIdx];
        
        int skip = dfs(i + 1, prev);
        int take = 0;
        if (prev == -1 || pairs[i][0] > pairs[prev][1]) {
            take = 1 + dfs(i + 1, i);
        }
        
        memo[i][prevIdx] = Math.max(skip, take);
        return memo[i][prevIdx];
    }
}
```

**Key Insight:** Sort by first element, then classic LIS on condition `pairs[i][0] > pairs[prev][1]`.

## 7. **Russian Doll Envelopes (Leetcode 354)**

**Instinct:**  
Envelopes (w, h). Ek envelope doosre me tabhi ja sakta hai jab dono dimensions strictly chhote ho. Sort by width ascending, agar width same toh height descending. Phir LIS on height.

**Approach:**  
1. Sort: width ascending, if same width then height descending
2. Apply LIS on heights array
3. Memoization O(n²) accepted for n <= 2000

```java
class Solution {
    int[][] envelopes;
    int n;
    Integer[][] memo;
    
    public int maxEnvelopes(int[][] envelopes) {
        this.envelopes = envelopes;
        n = envelopes.length;
        Arrays.sort(envelopes, (a, b) -> {
            if (a[0] == b[0]) return b[1] - a[1];
            return a[0] - b[0];
        });
        
        memo = new Integer[n][n + 1];
        return dfs(0, -1);
    }
    
    private int dfs(int i, int prev) {
        if (i == n) return 0;
        int prevIdx = prev + 1;
        if (memo[i][prevIdx] != null) return memo[i][prevIdx];
        
        int skip = dfs(i + 1, prev);
        int take = 0;
        if (prev == -1 || (envelopes[i][0] > envelopes[prev][0] && 
                           envelopes[i][1] > envelopes[prev][1])) {
            take = 1 + dfs(i + 1, i);
        }
        
        memo[i][prevIdx] = Math.max(skip, take);
        return memo[i][prevIdx];
    }
}
```

**Key Insight:** 2D LIS = sort one dimension, LIS on other. Same width wale kabhi fit nahi honge.

## 8. **Longest String Chain (Leetcode 1048)**

**Instinct:**  
Strings ki chain jahan next string = previous string + 1 character anywhere. Sort by length, then LIS on condition: isPredecessor(prev, curr).

**Approach:**  
1. Sort strings by length
2. Memo[i][prev] = max chain length
3. Check if prev string is predecessor of current

```java
class Solution {
    String[] words;
    int n;
    Integer[][] memo;
    
    public int longestStrChain(String[] words) {
        this.words = words;
        n = words.length;
        Arrays.sort(words, (a, b) -> a.length() - b.length());
        memo = new Integer[n][n + 1];
        return dfs(0, -1);
    }
    
    private int dfs(int i, int prev) {
        if (i == n) return 0;
        int prevIdx = prev + 1;
        if (memo[i][prevIdx] != null) return memo[i][prevIdx];
        
        int skip = dfs(i + 1, prev);
        int take = 0;
        if (prev == -1 || isPredecessor(words[prev], words[i])) {
            take = 1 + dfs(i + 1, i);
        }
        
        memo[i][prevIdx] = Math.max(skip, take);
        return memo[i][prevIdx];
    }
    
    private boolean isPredecessor(String a, String b) {
        if (b.length() != a.length() + 1) return false;
        int i = 0, j = 0;
        while (i < a.length() && j < b.length()) {
            if (a.charAt(i) == b.charAt(j)) {
                i++;
            }
            j++;
        }
        return i == a.length();
    }
}
```

**Key Insight:** Sort by length. Predecessor check: exactly one character extra, order same.

## 9. **Minimum ASCII Delete Sum for Two Strings (Leetcode 712)**

**Instinct:**  
Do strings ko equal banane ke liye delete karna hai. Minimum sum of ASCII values of deleted characters. Ye **LCS variant** hai - longest common subsequence with weights.

**Approach:**  
1. Memo[i][j] = min delete sum for s1[i..] and s2[j..]
2. If s1[i] == s2[j] → dfs(i+1, j+1)
3. Else → min(s1[i] + dfs(i+1, j), s2[j] + dfs(i, j+1))

```java
class Solution {
    String s1, s2;
    int[][] memo;
    
    public int minimumDeleteSum(String s1, String s2) {
        this.s1 = s1;
        this.s2 = s2;
        memo = new int[s1.length()][s2.length()];
        for (int i = 0; i < s1.length(); i++) {
            Arrays.fill(memo[i], -1);
        }
        return dfs(0, 0);
    }
    
    private int dfs(int i, int j) {
        if (i == s1.length()) {
            int sum = 0;
            for (int k = j; k < s2.length(); k++) sum += s2.charAt(k);
            return sum;
        }
        if (j == s2.length()) {
            int sum = 0;
            for (int k = i; k < s1.length(); k++) sum += s1.charAt(k);
            return sum;
        }
        if (memo[i][j] != -1) return memo[i][j];
        
        int res;
        if (s1.charAt(i) == s2.charAt(j)) {
            res = dfs(i + 1, j + 1);
        } else {
            int deleteS1 = s1.charAt(i) + dfs(i + 1, j);
            int deleteS2 = s2.charAt(j) + dfs(i, j + 1);
            res = Math.min(deleteS1, deleteS2);
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** LCS with weighted deletion. When chars match, no deletion cost.

## 10. **Longest Common Subsequence (Leetcode 1143) - LIS Relation**

**Instinct:**  
LCS ko LIS me convert kar sakte ho agar ek string unique ho. But general LCS 2D DP hai. Yaha LIS pattern se relation samajhna important hai.

**Approach:**  
1. Memo[i][j] = LCS length for text1[i..] and text2[j..]
2. If chars match → 1 + dfs(i+1, j+1)
3. Else → max(dfs(i+1, j), dfs(i, j+1))

```java
class Solution {
    String t1, t2;
    int[][] memo;
    
    public int longestCommonSubsequence(String text1, String text2) {
        this.t1 = text1;
        this.t2 = text2;
        memo = new int[text1.length()][text2.length()];
        for (int i = 0; i < text1.length(); i++) {
            Arrays.fill(memo[i], -1);
        }
        return dfs(0, 0);
    }
    
    private int dfs(int i, int j) {
        if (i == t1.length() || j == t2.length()) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int res;
        if (t1.charAt(i) == t2.charAt(j)) {
            res = 1 + dfs(i + 1, j + 1);
        } else {
            res = Math.max(dfs(i + 1, j), dfs(i, j + 1));
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** LCS 2D DP. LIS pattern me convert tab hota hai jab ek string ke characters unique ho.

## 11. **Maximum Height by Stacking Cuboids (Leetcode 1691)**

**Instinct:**  
Cuboids ko stack karna hai (rotate allowed). Ek cuboid doosre upar tabhi aa sakta hai jab all dimensions <=. Sort all dimensions of each cuboid, then sort cuboids, then LIS.

**Approach:**  
1. Har cuboid ke dimensions ko sort karo (rotate allowed)
2. Sort all cuboids by first dimension, then second, then third
3. LIS on condition: all three dimensions <= next

```java
class Solution {
    int[][] cuboids;
    int n;
    Integer[][] memo;
    
    public int maxHeight(int[][] cuboids) {
        this.cuboids = cuboids;
        n = cuboids.length;
        
        // sort dimensions of each cuboid
        for (int[] c : cuboids) {
            Arrays.sort(c);
        }
        
        // sort cuboids by dimensions
        Arrays.sort(cuboids, (a, b) -> {
            if (a[0] != b[0]) return a[0] - b[0];
            if (a[1] != b[1]) return a[1] - b[1];
            return a[2] - b[2];
        });
        
        memo = new Integer[n][n + 1];
        return dfs(0, -1);
    }
    
    private int dfs(int i, int prev) {
        if (i == n) return 0;
        int prevIdx = prev + 1;
        if (memo[i][prevIdx] != null) return memo[i][prevIdx];
        
        int skip = dfs(i + 1, prev);
        int take = 0;
        if (prev == -1 || (cuboids[i][0] >= cuboids[prev][0] &&
                           cuboids[i][1] >= cuboids[prev][1] &&
                           cuboids[i][2] >= cuboids[prev][2])) {
            take = cuboids[i][2] + dfs(i + 1, i);
        }
        
        memo[i][prevIdx] = Math.max(skip, take);
        return memo[i][prevIdx];
    }
}
```

**Key Insight:** 3D LIS - all dimensions must be <=. Height = third dimension.

## 12. **Longest Increasing Subsequence in 2D Grid (Matrix LIS)**

**Instinct:**  
Matrix me longest path where next cell value > current. 4 directions allowed. Ye graph DFS + memoization hai - har cell se LIS length nikaalo.

**Approach:**  
1. Memo[i][j] = LIS length starting from (i,j)
2. Try all 4 neighbors where value > current
3. 1 + max(neighbors)

```java
class Solution {
    int[][] matrix;
    int m, n;
    int[][] memo;
    int[][] dirs = {{0,1},{1,0},{0,-1},{-1,0}};
    
    public int longestIncreasingPath(int[][] matrix) {
        this.matrix = matrix;
        m = matrix.length;
        n = matrix[0].length;
        memo = new int[m][n];
        
        int maxLen = 0;
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                maxLen = Math.max(maxLen, dfs(i, j));
            }
        }
        return maxLen;
    }
    
    private int dfs(int i, int j) {
        if (memo[i][j] != 0) return memo[i][j];
        
        int max = 1;
        for (int[] dir : dirs) {
            int ni = i + dir[0];
            int nj = j + dir[1];
            if (ni >= 0 && ni < m && nj >= 0 && nj < n && 
                matrix[ni][nj] > matrix[i][j]) {
                max = Math.max(max, 1 + dfs(ni, nj));
            }
        }
        
        memo[i][j] = max;
        return max;
    }
}
```

**Key Insight:** 2D grid me LIS = longest increasing path. 4 directions allowed.

## 13. **Number of Longest Increasing Subsequence in 2D Grid**

**Instinct:**  
Matrix me longest increasing path ki count chahiye. Har cell ke liye (length, count) store karo.

**Approach:**  
1. Memo[i][j] = {maxLen, count} from (i,j)
2. For all neighbors with greater value, find best length
3. Sum counts for best length

```java
class Solution {
    int[][] matrix;
    int m, n;
    Pair[][] memo;
    int[][] dirs = {{0,1},{1,0},{0,-1},{-1,0}};
    
    class Pair {
        int len;
        int count;
        Pair(int l, int c) { len = l; count = c; }
    }
    
    public int countLongestIncreasingPath(int[][] matrix) {
        this.matrix = matrix;
        m = matrix.length;
        n = matrix[0].length;
        memo = new Pair[m][n];
        
        int maxLen = 0;
        int totalCount = 0;
        
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                Pair p = dfs(i, j);
                if (p.len > maxLen) {
                    maxLen = p.len;
                    totalCount = p.count;
                } else if (p.len == maxLen) {
                    totalCount = (totalCount + p.count) % 1_000_000_007;
                }
            }
        }
        return totalCount;
    }
    
    private Pair dfs(int i, int j) {
        if (memo[i][j] != null) return memo[i][j];
        
        int maxLen = 1;
        int totalCount = 1;
        
        for (int[] dir : dirs) {
            int ni = i + dir[0];
            int nj = j + dir[1];
            if (ni >= 0 && ni < m && nj >= 0 && nj < n && 
                matrix[ni][nj] > matrix[i][j]) {
                Pair next = dfs(ni, nj);
                if (next.len + 1 > maxLen) {
                    maxLen = next.len + 1;
                    totalCount = next.count;
                } else if (next.len + 1 == maxLen) {
                    totalCount = (totalCount + next.count) % 1_000_000_007;
                }
            }
        }
        
        memo[i][j] = new Pair(maxLen, totalCount);
        return memo[i][j];
    }
}
```

**Key Insight:** 2D grid me count of LIS - same logic as 1D but with 4 neighbors.

## 14. **Minimum Number of Removals to Make Mountain Array (Leetcode 1671)**

**Instinct:**  
Mountain array = first strictly increasing then strictly decreasing. Minimum removals = n - (longest bitonic subsequence). But yaha strictly mountain chahiye - peak can't be first or last.

**Approach:**  
1. LIS from left ending at each index
2. LDS from right ending at each index
3. Mountain length = left[i] + right[i] - 1 (if both > 1)
4. Answer = n - max mountain length

```java
class Solution {
    public int minimumMountainRemovals(int[] nums) {
        int n = nums.length;
        int[] left = new int[n];
        int[] right = new int[n];
        
        // LIS from left
        Arrays.fill(left, 1);
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < i; j++) {
                if (nums[j] < nums[i]) {
                    left[i] = Math.max(left[i], left[j] + 1);
                }
            }
        }
        
        // LDS from right
        Arrays.fill(right, 1);
        for (int i = n - 1; i >= 0; i--) {
            for (int j = n - 1; j > i; j--) {
                if (nums[j] < nums[i]) {
                    right[i] = Math.max(right[i], right[j] + 1);
                }
            }
        }
        
        int maxMountain = 0;
        for (int i = 0; i < n; i++) {
            if (left[i] > 1 && right[i] > 1) {
                maxMountain = Math.max(maxMountain, left[i] + right[i] - 1);
            }
        }
        
        return n - maxMountain;
    }
}
```

**Key Insight:** Mountain = LIS(left) + LDS(right) - 1. Both sides must be >=2.

## 15. **Maximum Sum Increasing Subsequence (Print it)**

**Instinct:**  
Maximum sum increasing subsequence print karni hai. Memoization me sum ke saath path bhi store karo.

**Approach:**  
1. Memo[i][prev] = {maxSum, nextIndex} 
2. Track which choice gave max sum
3. Reconstruct path

```java
class Solution {
    int[] nums;
    int n;
    Pair[][] memo;
    
    class Pair {
        int sum;
        int nextIdx;
        Pair(int s, int n) { sum = s; nextIdx = n; }
    }
    
    public List<Integer> maxSumISPrint(int[] nums) {
        this.nums = nums;
        n = nums.length;
        memo = new Pair[n][n + 1];
        
        Pair result = dfs(0, -1);
        
        // reconstruct path
        List<Integer> seq = new ArrayList<>();
        int i = 0, prev = -1;
        while (i < n) {
            int prevIdx = prev + 1;
            Pair p = memo[i][prevIdx];
            if (p.nextIdx == i) {
                seq.add(nums[i]);
                prev = i;
                i++;
            } else {
                i++;
            }
        }
        return seq;
    }
    
    private Pair dfs(int i, int prev) {
        if (i == n) return new Pair(0, -1);
        int prevIdx = prev + 1;
        if (memo[i][prevIdx] != null) return memo[i][prevIdx];
        
        Pair skip = dfs(i + 1, prev);
        Pair take = new Pair(0, -1);
        if (prev == -1 || nums[i] > nums[prev]) {
            take = dfs(i + 1, i);
            take = new Pair(nums[i] + take.sum, i);
        }
        
        Pair res;
        if (skip.sum > take.sum) {
            res = skip;
        } else {
            res = take;
        }
        
        memo[i][prevIdx] = res;
        return res;
    }
}
```

**Key Insight:** Store both value and next index for path reconstruction.

## 16. **Longest Well-Performing Interval (Leetcode 1124)**

**Instinct:**  
Hours array me tiring day ( > 8 ) ko +1, normal day ko -1 treat karo. Longest subarray with sum > 0. Ye LIS pattern nahi hai but prefix sum + map hai. Phir bhi LIS se relation hai.

**Approach:**  
1. Convert: tiring = +1, normal = -1
2. Prefix sum array
3. For each i, find earliest j where prefix[j] < prefix[i]
4. Use map to store first occurrence of each prefix

```java
class Solution {
    public int longestWPI(int[] hours) {
        int n = hours.length;
        int[] prefix = new int[n + 1];
        for (int i = 0; i < n; i++) {
            prefix[i + 1] = prefix[i] + (hours[i] > 8 ? 1 : -1);
        }
        
        Map<Integer, Integer> firstOccurrence = new HashMap<>();
        int maxLen = 0;
        
        for (int i = 0; i <= n; i++) {
            if (!firstOccurrence.containsKey(prefix[i])) {
                firstOccurrence.put(prefix[i], i);
            }
            if (prefix[i] > 0) {
                maxLen = Math.max(maxLen, i);
            } else {
                if (firstOccurrence.containsKey(prefix[i] - 1)) {
                    maxLen = Math.max(maxLen, i - firstOccurrence.get(prefix[i] - 1));
                }
            }
        }
        return maxLen;
    }
}
```

**Key Insight:** Longest subarray with positive sum. Prefix sum + first occurrence map.

## 17. **Longest Arithmetic Subsequence (Leetcode 1027)**

**Instinct:**  
Array me longest arithmetic subsequence (difference same). Har index ke liye, difference track karo. Memo[i][diff] = length ending at i with given diff.

**Approach:**  
1. Memo[i][diff] = max length ending at i with difference diff
2. For each j < i, diff = nums[i] - nums[j]
3. length = memo[j][diff] + 1

```java
class Solution {
    public int longestArithSeqLength(int[] nums) {
        int n = nums.length;
        Map<Integer, Integer>[] memo = new Map[n];
        for (int i = 0; i < n; i++) {
            memo[i] = new HashMap<>();
        }
        
        int maxLen = 1;
        
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < i; j++) {
                int diff = nums[i] - nums[j];
                int len = memo[j].getOrDefault(diff, 1) + 1;
                memo[i].put(diff, Math.max(memo[i].getOrDefault(diff, 1), len));
                maxLen = Math.max(maxLen, len);
            }
        }
        return maxLen;
    }
}
```

**Key Insight:** Difference based LIS. Memoization with Map for diff.

## 18. **Longest Arithmetic Subsequence of Given Difference (Leetcode 1218)**

**Instinct:**  
Difference fixed hai. Longest subsequence with that exact difference. Simple DP: dp[num] = 1 + dp[num - diff].

**Approach:**  
1. Map to store longest length ending at each value
2. For each num, length = map.getOrDefault(num - diff, 0) + 1

```java
class Solution {
    public int longestSubsequence(int[] arr, int difference) {
        Map<Integer, Integer> dp = new HashMap<>();
        int maxLen = 0;
        
        for (int num : arr) {
            int len = dp.getOrDefault(num - difference, 0) + 1;
            dp.put(num, len);
            maxLen = Math.max(maxLen, len);
        }
        return maxLen;
    }
}
```

**Key Insight:** Fixed difference = simple map DP. O(n) time.

## 19. **Maximum Length of Repeated Subarray (Leetcode 718) - LCS on arrays**

**Instinct:**  
Do arrays me longest common subarray (contiguous). Ye LCS ka contiguous version hai.

**Approach:**  
1. Memo[i][j] = length of common subarray starting at i in nums1 and j in nums2
2. If nums1[i] == nums2[j] → 1 + dfs(i+1, j+1)
3. Else → 0

```java
class Solution {
    int[] n1, n2;
    int[][] memo;
    
    public int findLength(int[] nums1, int[] nums2) {
        this.n1 = nums1;
        this.n2 = nums2;
        memo = new int[nums1.length][nums2.length];
        for (int i = 0; i < nums1.length; i++) {
            Arrays.fill(memo[i], -1);
        }
        
        int maxLen = 0;
        for (int i = 0; i < nums1.length; i++) {
            for (int j = 0; j < nums2.length; j++) {
                maxLen = Math.max(maxLen, dfs(i, j));
            }
        }
        return maxLen;
    }
    
    private int dfs(int i, int j) {
        if (i >= n1.length || j >= n2.length) return 0;
        if (memo[i][j] != -1) return memo[i][j];
        
        int res = 0;
        if (n1[i] == n2[j]) {
            res = 1 + dfs(i + 1, j + 1);
        }
        
        memo[i][j] = res;
        return res;
    }
}
```

**Key Insight:** Subarray = contiguous. Agar match nahi hota toh chain toot jaati hai (0 return).

## 20. **Longest Turbulent Subarray (Leetcode 978)**

**Instinct:**  
Turbulent = alternate increase/decrease. Longest subarray with this property. Ye LIS pattern nahi hai but DP hai.

**Approach:**  
1. up[i] = length ending at i where last step was increase
2. down[i] = length ending at i where last step was decrease
3. If nums[i] > nums[i-1] → up[i] = down[i-1] + 1
4. If nums[i] < nums[i-1] → down[i] = up[i-1] + 1

```java
class Solution {
    public int maxTurbulenceSize(int[] arr) {
        int n = arr.length;
        if (n == 1) return 1;
        
        int[] up = new int[n];
        int[] down = new int[n];
        Arrays.fill(up, 1);
        Arrays.fill(down, 1);
        
        int maxLen = 1;
        
        for (int i = 1; i < n; i++) {
            if (arr[i] > arr[i - 1]) {
                up[i] = down[i - 1] + 1;
            } else if (arr[i] < arr[i - 1]) {
                down[i] = up[i - 1] + 1;
            }
            maxLen = Math.max(maxLen, Math.max(up[i], down[i]));
        }
        return maxLen;
    }
}
```

**Key Insight:** Two states DP - increasing and decreasing. Turbulence = alternate.

## 🎯 Key Takeaways for LIS Pattern (Memoization)

### 1. **Classic LIS Structure:**
```java
dfs(i, prev) {
    skip = dfs(i+1, prev)
    take = 0
    if (prev == -1 || nums[i] > nums[prev]) {
        take = 1 + dfs(i+1, i)
    }
    return max(skip, take)
}
```

### 2. **State Space:**
- Usually `(i, prevIndex)` - O(n²)
- Prev = -1 ke liye shift by +1

### 3. **Variations:**
| Variation | Change |
|-----------|--------|
| Sum LIS | `take = nums[i] + dfs(i+1, i)` |
| Count LIS | Return Pair(len, count) |
| 2D LIS | Condition on both dimensions |
| Grid LIS | 4 directions instead of sequence |

### 4. **Optimizations:**
- **Patience Sorting** for O(n log n) LIS (greedy + binary search)
- But memoization O(n²) is acceptable for n ≤ 2000
- For n ≤ 10⁵, use patience sorting (iterative)

### 5. **LIS to LCS Conversion:**
- If one array has unique elements, LIS on positions = LCS

### 6. **When to use LIS Pattern:**
- Need longest sequence with some ordering property
- Elements can be non-contiguous
- Comparison between current and previous element
